Force A Buy Order

+ Larger Font | + Smaller Font

The Force a Buy Order page is used to create a Buy Order in tcmIS when there is no Material Request or Stocking Levels set. The Force a Buy Order page can be accessed from the Supply Chain menu by selecting Force a Buy Order. The Force a Buy Order page is launched in a new tab.

Figure 1

Select the Hub and Inventory Group for which the Buy Order should be created from the corresponding drop-down lists. Click the Search button to display all active materials. Note that the quantity on hand and in Purchasing is displayed for each material.

Figure 2

To locate s specific Item on the Force a Buy page select Item ID and is from the Search drop-down lists. Enter the Item ID in the Search text box and click the Search button.

Figure 3

To locate s specific Customer Part Number on the Force a Buy page select Part No and is from the Search drop-down lists. Enter the Part Number in the Search text box and click the Search button.

Figure 4

Check the box labeled Show Min/Max Only to limit the results to only MinMax materials. Click the Search button.

Figure 5

After locating the item, enter the quantity to buy in the Replenish Qty column. Be sure to enter the quantity based on the unit of measure in the Inventory UOM column. Enter the date the material is needed in the Need Date column by clicking in the Need Date text box and selecting a date from the calendar. Enter any notes for the Buyer in the Notes box. Check the box labeled OK for all lines to be ordered. Click the Process button to create buy orders for all lines selected.
