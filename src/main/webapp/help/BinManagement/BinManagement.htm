Bin Management

+ Larger Font | + Smaller Font

The Bin Management page is used to manage bin locations for a warehouse in tcmIS. Bins can be inactivated, activated, or created using this page. The Bin Management page can be accessed from the Supply Chain Menu. Under Stock Room Inventory, select Inventory then select Bin Management. The Bin Management page is launched in a new tab.

Figure 1

Select the Operating Entity and Hub from the corresponding drop-down lists. You may select a room from the drop-down list labeled Room to see bins associated with the selected room. Select All from the Room drop-down list to see all bins in the warehouse regardless of the room. Click the Search button.

Figure 2

To locate a specific bin select contains from the Search drop-down and enter the bin in the text box. Click Search.

Figure 3

To inactivate a bin uncheck the box in the Active column. Check the OK box for all bins to update and click the Update link.

Figure 4

To view inactive bins uncheck the box labeled Show Active Only and click Search. Inactive bins are highlighted in black. To activate a bin check the box in the Active column. Check the OK box for all bins to update and click the Update link.

To change or select the Room select the room from the drop-down box in the Room column. Check the OK box for all bins to update and click the Update link.

Figure 5

Click the Create New Bin link to add a new bin. Select the Room to which the bin belongs from the Room drop-down list. Enter the bin name in the Bin text box. If the bin is not located at the hub uncheck the box labeled At the hub. Click OK to create the new bin.

Figure 6

To create a new room in the hub click the Add New Room button.

Enter the name and a description of the room in the corresponding text boxes. Click OK to create the room.
