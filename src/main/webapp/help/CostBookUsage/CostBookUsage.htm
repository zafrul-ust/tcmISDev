Cost Book Usage

The Cost Book Usage page displays usage details for each material over the program year for counting inventory groups. The Cost Book Usage page can be accessed from the Reports menu by selecting Cost Report. Then select Cost Book Usage. The Cost Book Usage page is launched in a new tab.

Select the company, facility, inventory group. and report start date from the corresponding drop-down lists. Click Search.

Click Create Excel File to export the report to Excel.
