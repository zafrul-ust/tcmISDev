The Catalog page provides access to information on items (materials, chemicals) approved for use in each facility and work area. It is used to create and submit requests (Material Requests) for purchase and delivery of those items.

The Catalog page can be accessed from the Order Management menu by selecting Catalog. It will be shown in its own tab.

Figure 1

The first step in ordering an item is to locate it in the Part Catalog. (The Item Catalog is used for items or parts not yet approved, and for ordering them for evaluation. See Engineering Evaluation.)

  • Select the facility where the material will be used from the Facility list.
  • Select the work area where the material will be used from the Work Area list.
  • Select the Part catalog (Item catalog for evaluations) from the Search Type list.
  • Enter details about the desired material in the Search box. Search criteria can include DoE part number, HGI item ID, packaging, trade name, manufacturer, manufacturer's part number, or material description.

The more specific the information entered, the more the search is narrowed. Multiple words as a logical expression can also be used to refine a search (separate words by and, or, but not; for example, prc AND 1750). To search for all chemicals for a specific location, leave the Search box blank.

Ordering Materials

To order any item, add it to the shopping cart by:

  • locating the item in the part catalog;
  • clicking anywhere on the line to select the item (it will then be highlighted with a different background color); and,
  • click the Add To Cart link above the data grid.

If a material with a comment attached to it is selected and the Add to Cart link is clicked the comment will be displayed to notify you.

  • If the comment warrants attention, click Cancel to stop adding the material to the shopping cart and resolve any issues raised by the comment.
  • If the comment does not require you to do anything differently, click the Continue to add the part to the shopping cart.

Locating and adding items to the shopping cart can be repeated until all needed items are in the shopping cart.

Once the first item has been added to the shopping cart, all further additions to that cart must be for delivery to the same facility but may be for different work areas.

Figure 4

The item is added to the shopping cart and displayed in the data grid at the upper right of the page. There are several actions which you can take here to adjust the item on order.

  • The quantity on order can be replacing the number in the Quantity text box with the quantity needed. The quantity can also be changed later on the Material Request page.
  • The date the material is needed should be provided by clicking in the Date Needed text box and selecting a date from the calendar. If you do not provide the date here you will have to provide it on the Material Request page before you check out.
  • If the need for the item is critical, check the Critical box at the right end of the row to designate this material request as a critical order. (It will then be highlighted with a different background color.)
  • Enter notes by clicking in the Notes column, causing a text box to open, and adding your notes. Click outside of the text box to save the notes.

An item can be removed from the shopping cart by clicking on the corresponding line (it will then be highlighted with a different background color) and clicking the Delete Line link above shopping cart data grid.

Specifications

Specifications associated with a part can be displayed by invoking the first context menu and hovering over the Specification menu item. Any specifications associated with the part are displayed as submenu items.

Select the specification menu item you want to view. The most current revision of the specification is displayed with the specification document shown in PDF format within the page (using the same format as viewing an MSDS; see below). If any other specifications are on record you may view them by selecting one from the Spec Available Specs list in the upper-left corner of the page.

In some cases the specification is ’dimmed’ indicating that it exists but is not stored in tcmIS.

Figure 16

Comments

Comments associated with a part or work area can be displayed and managed by invoking the first context menu. Hover on the Comments item. There are three types of comments that can be added or edited, each with its own features and purposes.

  • Choose Cart Comments to view or add a Cart-specific comment for this facility and work area. The user is notified about this comment every time the part is added to a shopping cart for the facility and work area. This feature can be used to share information about excess chemicals with other users to help encourage efficient use of chemicals. Buyers and HGI personnel will not be shown this comment.
  • Choose Part Work Area Comments to view or add comments specific to this facility and work area. These comments are not shown to other facilities or work areas, nor outside DoE.
  • Choose Part Comments to view or add comments for all users of the Catalog page. Part Comments are printed on the HGI pick list at the warehouse.

The Comment, Part Work Area Comments, and Part Comments pages differ slightly in appearance and operation (see below). Each opens in a new window to view, add, or edit comments.

Part (or Work Area) Directed Charge

Selecting the Edit Part Directed Charge context menu item will open a window to display the Charge Information form. This is used to edit the existing directed charge information for a part (or even a work area). It comes in two forms, as illustrated, to handle both Direct and Indirect charges. Make the needed changes directly on the form.

  • Click the Remove Charge Numbers link to delete the existing charge numbers (if any) from the part record for this work area; or,
  • Enter the percentage values that are appropriate in the right text column. This capability is only available to those who have the appropriate responsibility and authority.
  • If available, use the PO box, which may appear as a text box in which to type, or may appear as a drop-down list from which to select, to change the PO for the part record for this work area.

When done, click on the Submit link in the upper-left corner to save the changes: the new information will be used until any future change is made. If you need simply to delete the existing directed charge, click the Remove Charge Numbers link in the upper-left. The window will close when you click either link.

If directed charges have been entered for both a part and the work area, the part directed charge will be used rather than the work area directed charge.
