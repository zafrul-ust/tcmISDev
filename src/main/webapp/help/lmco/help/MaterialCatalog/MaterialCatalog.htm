Material Catalog

Use the Material Catalog search on the Catalog page to add new materials to a Facility Catalog. (After HMRB review and approval, the material may be assigned as an item included as a part and Use Approval can be given as needed. On the Catalog page begin by doing a search with the Search Type of Material Catalog.

  • You must specify the Facility using the Facility drop-down list.
  • The Work Area is not directly relevant to the addition of a material. Choosing one from the Work Area drop-down list will probably reduce the number of records found in the search.
  • You may restrict your search to materials that are already Active for the Facility by selecting that check box; alternatively, search the Full Material Database by selecting that check box.

After you click the Search button and the data grid of results (if any) is displayed, there will be a New Material link on the top line of the data grid.

(Select a record and an Add Approval Code link will be provided. Clicking this link will cause tcmIS to open a Cat Add Req page where you may select the HMRB tab and then click another Add Approval Code link to proceed. See Add Approval Code for details.)

Click on the New Material link to begin the add new material process.

Figure 1

tcmIS will open a new tab and display a limited version of the Cat Add Req page where you may continue the process to specify the material as well as provide appropriate approvals. Because this page is used to add a new material no material information is provided even if you had selected some specific material on the Catalog page data grid.

  • In some cases the ’new’ material is new only to the facility but is already in the Full Material Database. In this case you may proceed by selecting the Add Material link.
    • You may choose this link even if you are uncertain as to whether the material is in the database at all. If you find it, you may proceed. If not, you will still have the choice to move forward with a New Material link.
  • In other cases the material is completely new and you may proceed by selecting the New Material link. (see below)
Figure 2

Add Material

When you select the Add Material link, tcmIS will open a child window to display the Search Material form. You may use this to attempt to locate the material that you want to add.

  • To search the Full Material Database, select that check box.
  • Otherwise, leave it not selected.

After you click the Search button and the data grid of results (if any) is displayed, there will be a New Material link on the top line of the data grid which you may use if the desired material is not in the search results data grid.

If the material you want is in the data grid, selecting the record will highlight the row and cause a new link to appear at the top of the data grid which you may use to Return Selected MSDS to the Cat Add Req page with a Status of Pending Approval.

Figure 3

New Material

In cases where the material does not exist in the Full Material Database, selecting the New Material link will cause tcmIS to open a child window and display a version of the Edit Material Info form. Using this form you may specify the minimum information required to add a New Material request to the Cat Add Req page.

  • If you have it, provide the Material MSDS.
  • If available, specify whether the material fits any of these categories by selecting the respective check boxes:
    • it is an Aerosol;
    • it Contains Nano Material;
    • it is Radioactive.
  • You are required to provide information for these items:
    • Material Description;
    • Manufacturer [name];
    • Manufacturer’s Trade Name for the material.

When you have provided all the information, click the Submit link in the upper left of the form. The form will be closed and the material will be added to the Cat Add Req page with a Status of Pending Approval. (The Cancel link will discard any provided information, close the form, and redisplay the Cat Add Req page without adding anything.)

Figure 4

When a material (added or new) appears on the Cat Add Req page with a status of Pending Approval, a right-click on the record will display a context menu one or more of these items:

  • Choose the Delete Line item to remove the record from the form.
  • Choose the Upload MSDS Image(s) link to store an MSDS image for the material. (You may also use the Upload MSDS Images(s) link at the top of the data grid to do the same for a selected record. (See below.)
  • Choose the View Component [or Kit] MSDS menu item to see an existing MSDS for the material (if there is one).
Figure 5

Upload an MSDS

Click the Browse button, locate and select the file to upload on your workstation. Click the Send File button. The file will be stored in tcmIS and associated with the material record Cat Add Req. The Upload File form will then be closed.

Figure 6

HMRB: Add Approval Code

The HMRB tab on the Cat Add Req page provides access to view, add, or edit approval codes for the material before it is submitted for HMRB review. This code provides the use context for the material which is used to make an HMRB decision. (See the Cat Add Req - HMRB page.)

Figure 7

When the material information and approvals are complete, click the Submit link in the upper left corner of the page to submit the added or new material as a request.
