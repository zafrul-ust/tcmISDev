Ad Hoc Usage

The Ad Hoc Usage report describes which materials were used at a specific facility. The format of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields.

The Ad Hoc Usage report can be accessed from the Reports menu. Select Usage and then Ad Hoc Usage. The Ad Hoc Usage page is launched in a new tab.

Figure 1

The first step in creating an Ad Hoc Usage report is to choose and specify the chemical or chemicals to be reported.

  • To run the report for a chemical list (for example SARA313 or Ozone Depleting Substances) choose the List option and select the chemical list from the drop-down list.
    • Use the View List button to display a complete listing for the selected list.
    • Neither the drop-down list nor the View List button will respond unless the List option button is selected.
  • To run the report for a single chemical, select the Single Chemical option and enter the CAS number in the CAS Number box.
    • Use the CAS Search form, if needed, to locate the exact number. (This form differs from, but serves the same purpose as, the more common Lookup forms.)
    • The Search button will not respond unless the Single Chemical option button is selected.
  • To run this report for all chemicals, choose the All option.
Figure 2

View items included on a selected chemical list by clicking the View List button. The Chemical List page will be displayed in a new window listing all chemicals that are included on the selected list.

Figure 3

Search for a CAS number using the CAS Search form which is similar to a standard Lookup form.

  • Click the Search button to open the form in a child window.
  • Enter the text for which you want to search in the Search text box on the form.
  • Click the Search button on the form AND tcmIS will display all the CAS items that match the specified text in whole or in part.
  • Locate the CAS item you want in the list and click on its row and tcmIS will place the CAS number in the Selected CAS Number text box.
    • If you realize you have chosen the wrong CAS item, choose another and tcmIS will replace the original with your new selection.
    • You may clear the selection and then edit the search text by first clicking on the Reset button, in order to start over completely.
  • When the correct number is in the Selected CAS Number text box, click the OK button on the form. tcmIS will move the selected CAS number onto the Ad Hoc Usage form.
Figure 4 (CAS Search form) and Figure 4A (Completed CAS Search form)

After specifying the chemical(s) for the report, specify a facility, work area(s), dock(s), and delivery point(s) from the corresponding boxes. Specify the time period to report by selecting the Begin Date and End Date using the date selection calendar.

If desired, the report can be limited using the two Part Number boxes. In the first box you specify whether the search is for an exact match, using is, or a partial match, using contains. Keep in mind that an exact match is exactly that, so typos must be avoided with is. In the second box enter the number to be matched for the Part Number.

The report can also be limited to a specific material category and manufacturer. To report only on chemicals in a specific category, select the category from the Material Category list. Use the two Manufacturer boxes to identify the manufacturer you want. In the first box specify whether the search is for an exact match, using is, or a partial match, using contains. Keep in mind that an exact match is exactly that, so typos must be avoided with is. In the second box enter the name or text to be matched for the Manufacturer.

Figure 5

Select the fields to include in the report from the Base Fields box on the left. You may select a field by clicking on it. To select more than one, hold down the CTRL key while selecting. Add the fields to the Report Fields box on the right by clicking the right-pointing arrow . You can remove fields from the Report Fields box by selecting in the Report Fields box and clicking the left-pointing arrow icon .

You can move an item in the Report Fields box up or down relative to the others by selecting it and clicking the up- or down-pointing arrow icons.

Figure 6

A description of the base fields is available by clicking the Description link above the Base Fields box.

Figure 7
