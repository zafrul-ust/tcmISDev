User Profile

The User Profile page is used to create new users and edit user information and certain permissions in tcmIS. The User Profile page can be accessed from the Administration menu by selecting User Profile. The User Profile page is displayed in a new tab.

You must have administrative permission to add or change user information or permissions in tcmIS. Contact the Technical Center to request administrative permissions.

Figure 1

Your user account is displayed by default. You may update any of the information displayed including first name, last name, middle initial, phone number, fax number, mobile number, email, and printer location. Click the Update Personal Info link to save any changes. You may also reset your password by clicking the Change Password link.

Figure 2

You may look up an existing tcmIS user and his or her personal information by clicking the magnifying glass icon . Enter the first name or last name of the user in the Search box and click Search. Locate the user in the resulting list and click anywhere on the line to select the user. Click the Return Selected User link to place that user's personal information on the User Profile page.

If you are a tcmIS administrator:

  • you may edit the user's personal information as you would your own.
  • you may also create new user accounts. Click the New button. A blank User Profile page is displayed in the existing tab.
    • Use the Company drop-down list to specify the organization for the new user account.
    • The following three fields are required:
      • Provide a unique Logon ID for the new user account in the Logon ID text box.
      • Enter a password in the Password text box.
      • Enter the password again in the Confirm Password text box.
    • Provide as much of the following information as you can:
      • Enter the user's first name, last name, email address, phone number, fax number, and cell phone number in the corresponding text boxes.
      • Select a printer location if this user works at a warehouse and needs to print labels.
        • If the needed printer location does not exist or you are unsure which to select, contact the Technical Center.

After all required personal information is completed, click the Create User link at the top left of the page. Click the Cancel link to abandon creation of this new user.

Startup Pages button

The Startup Pages button gives access to control which pages are launched automatically when tcmIS is started for the selected user. Click the Startup Pages button in the upper panel to open the Startup Pages page in a child window.

  • To add a page to the user's startup display select the check box in the Startup column for the page.
  • To remove a page from the user's startup display, clear the check box for the page.
  • As an option, you may specify the order of the pages (from left to right) by entering numbers, starting with 1, in the Startup Order column.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close the page.

Figure 6

Companies button

The Companies button gives access to control which companies the current user may access as well as administrative privileges for each company. Click the Companies button in the lower panel to open the Companies page in a child window.

  • Select the check box in the Access column for each company that the user needs to view.
  • Select the check box in the Admin column for each company where the user is to have administrative privileges.
    • Company Admin privileges include the functionality to create new users for the company.
    • You must have Grant Admin permission for a company to give others administrative permissions for that company.
  • To keep a user account active but eliminate that account's access to a particular company, select the check box in the Locked column for that company.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close this page.

Pages button

The Pages button gives access to control which tcmIS pages the current user may access as well as administrative privileges for each page. Click the Pages button in the lower panel to open the Pages page in a child window.

Select the company for page permissions from the Company drop-down list; then click the Search button. All active pages for that company will be displayed.

  • Select the check box in the Admin column for each page where the user is to have administrative privileges.
    • You must have Grant Admin permission for a company to give others administrative permissions for pages.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this page.

Facilities button

The Facilities button gives access to control the current user’s functionality for each facility of a company. Functionality includes administrative privileges and operational permissions, all of which are granted (or denied) separately for each facility. Click the Facilities button in the lower panel to open the Facilities Permissions for page in a child window.

Select a facility from the Facility drop-down list, or if available, choose to display My Facilities.

  • Clear the Show current permission only check box to display all company facilities regardless of current user settings.
  • Click the Search button to display permissions for the current user.

For each facility (in rows) the columns specify functional capabilities. Select or clear the check boxes as needed.

  • A selected check box grants the current user account the functionality named in a column title.
    • You must have Grant Admin permission for a facility to give others Administrator permission.
  • A cleared check box denies the column’s functionality to the user account.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this page.

Figure 9 and 10

On the Facility Permissions for page, click the Edit List button to add or remove facilities from the selected user's profile. The new Facilities Permissions page, listing all the company Facilities, will be displayed in a new window.

  • Select the check box in the Access column for each facility that the user needs to view in tcmIS.
  • Select one default facility by Selecting the check box in the Default column for the desired facility. (You will receive an error message window if you click more than one box in the Default column.)
  • If the user is an administrator for a facility, select the check box in the Admin column.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close this page.

Work Areas button

The Work Areas button is used to manage the current user’s work area-level access within each facility. Click the Work Areas button in the lower panel to open the Facilities Work Area Permissions for page in a child window.

Select a facility from the Facility drop-down list, or (for some users) choose to display My Facilities, to view work areas by facility and grant or edit permissions.

  • Clear the Show current permission only check box to display all company facility work areas regardless of current user settings.
  • Click the Search button to display permissions for the current user.

For each facility work area (in rows) the columns specify functional capabilities. Select or clear the check boxes as needed. (By default, permissions are granted or denied for All Work Areas in a facility and you will often only see one ’All Work Areas’ row for each facility)

  • A selected check box grants the current user account the functionality named in a column title.
    • Select the check box in the Access column for each work area the user needs to view in tcmIS. Clear those check boxes the user does not need.
    • Select the check box in the Display Application column for work areas for which the user is to have permission to use the Java Application. (Java Application is not discussed in this help system.)
    • Select the check box in the Create MR column for all work areas for which the user can enter Material Requests in tcmIS.
  • A cleared check box denies the column’s functionality to the user account.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this page.

Docks button

The Docks button is used to manage the current user’s Dropship Receiving privileges associated with each Dock. Click the Docks button in the lower panel to open the Facility Dock Permissions page in a child window.

Select a facility from the Facility drop-down list, or (for some users) choose to display My Facilities, to view docks by facility and grant or edit permissions.

  • Clear the Show current permission only check box to display all facility docks regardless of current user settings.
  • Click the Search button to display permissions for the current user.

Select the check box in the Dropship Receiving column for each Dock where the user should be allowed to view open orders and perform dropship receiving. Clear the check box for each facility where the user account is not supposed to have that permission. Where a user may not be granted privilege the box will be inactive.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this page.

Figures 14 and 15

Financial Approver button

The Financial Approver list is used to manage approval authority, spending limits, and approval limits for creating MR’s. A user not listed can create an MR but not authorize any expense; an MR that exceeds a user's authorized order limit requires financial approval from a user with an adequate approval limit. Only some facilities use the Financial Approver list. The financial approval feature can be turned on or off for a facility by the Technical Center.

Click the Financial Approver button in the lower panel to open the Financial Approver page in a child window.

Select a facility from the Facility drop-down list, or (for some users) choose to display My Facilities. Click the Search button to display the Financial Approver page.

  • To add a user to the financial approver list click the Add link--
    • Identify the new user on the Financial Approver detail form that is opened. Search for the user by clicking the magnifying glass icon . (See Lookup forms.)
    • Enter the Users Order Limit in the displayed currency.
    • If the user is also a financial approver, enter the User Approval Limit in the displayed currency.
    • Select the designated approver from the Next Level drop-down list.
    • Click the OK button to add the user to the financial approver list.
  • To edit a user on the financial approver list select the name on the Approver list, then click the Edit link. Here, too, the Financial Approver detail form will be displayed in a child window. Make any needed change to the Users Order Limit, the User Approval Limit, and the designated approver from the Next Level box. Click the OK button to confirm the changes to the user record.
  • To remove a user from the financial approver list select the name on the Approver list and click the Delete link.
Figures 16 and 17
  • To add an Alternate Approver to the financial approver list select the user for whom the Alternate will act by clicking the user's name, then click the Alt Approver link.
    • The Alternate Approver page will be displayed in a new window including any existing Alternate Approvers.
    • Click on the New Alt Approver button. The Name Search form will be opened in a new window.
    • Locate and select the user who will be designated the new Alternate Approver and click the Return Selected User link to place the name in the Alternate Approver page.
  • Click on the Close button to complete the addition.

Alternate approvers receive email notification from tcmIS and have the same approval authority as the approver.

Order User Group button

User groups are a way to associate sets of users who have particular responsibilities, or shared interests, for specific materials. User groups are specific to a single facility but may have members from other facilities in some situations. The same User Group name (e.g., Acids) can therefore be used at more than one facility and even have members in common. For example, Tom Jones might be a member of the Acids user group at both Facility A and Facility B. User Groups can be used to limit who can order specific categories of material.

Click the Order User Group button in the lower panel to open the User Group Permissions page in a child window.

Select a facility on the Facility drop-down list, or (for some users) choose to display My Facilities.

  • Leave the default This User's Access radio button selected. (See below on working with the All Users’ Access radio button selected.)
  • Click the Search button to display the User Group Permissions.

Select (or leave selected) the checkbox in the Member column for each Group where the user should be a member. Clear the check box for each group where the user account is not supposed to be a member. Where a user may not be granted membership the box will be inactive.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this page.

Figure 20

Select the All User’s Access option and click the Search button to display an alternative User Group Permissions form. Here you may Add or Delete a user account from a group, or delete a group altogether. There is no Update link as any changes are made immediately.

  • Click on the plus sign (+) icon for any group to expand the list which displays all the user accounts assigned to that group.
    • Select a group and click the Add User To Group link to add the current user to the selected group.
    • Select a user within a group list and click the Delete User From Group link to remove the current user from that group list.
  • Click on the minus sign (-) icon to hide (collapse) all user accounts assigned to that group.
  • Regardless of whether a group is expanded or collapsed, selecting the group and clicking the Delete Group link will delete the selected group from tcmIS.
Figure 21

On either form of the User Group Permissions and regardless of whether any data grid has been displayed, click on the New User Group button to create a new User Group. tcmIS will display the New User Group form in a child window. Enter a descriptive name in the Description text box. Click the New User Group button to add the group to tcmIS. Click the Cancel button to discard the group you started. (After a Group is created it is still necessary to add members to it before it can be used.)

Click the Cancel button to discard the group you started.

Report User Group button
