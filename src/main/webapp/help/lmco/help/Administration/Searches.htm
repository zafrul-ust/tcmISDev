Search functions

Frequently in tcmIS when looking for information you want to narrow your search to locate more specifically (and quickly) the information you want. This is often done using a specialized search mechanism.

There are several sorts of searches in tcmIS; they usually fall into one of four types:

  • Lookup forms - specialized forms for looking up data, usually when you are trying to fill out another form. These are almost always invoked by clicking on a small magnifying glass icon .
  • Type ahead searches have replaced some lookup forms. As text is entered into a text input box, tcmIS presents a drop-down list that displays any valid values that match that text.
  • Single box text searches - a single text entry box, usually preceded by the word ’Search’, used to match specific text (letters and/or numbers).
  • Three-box text searchs - usually labeled as ’Search By’, this search allows you to indicate the field you want searched, the text for the search, and the physical location of the text in the field.

Search [By]

A Search or Search By frequently uses three input boxes (sometimes only one):

  • In the first box select the basis for the search: usually you will select the type of information (data field) such as MR-Line, Part Number, Part Desc[ription], or Item, but the possibilities will vary depending on the page or report. Choose the field you want from the drop-down list.
  • In the second box choose the type of search. Sometimes there are only two (contains and is) rather than four.
    • contains, for a partial match: see using the ’contains’ selection;
    • starts with, for an exact match of the beginning: see using the ’starts with’ selection;
    • ends with, for an exact match of the ending: see using the ’ends with’ selection; or
    • is, for an exact match: see using the ’is’ selection. Keep in mind that an exact match is exactly that, so typos must particularly be avoided with is.
  • In the third box enter the number or text you want fully or partially to match in the search.

Once you have selected all the criteria for the search, click the Search button. tcmIS will do the search and display the matching records in a grid below the search form, or print a message that there is no matching data.

Search contains

A partial match (contains) looks for the text you enter anywhere in the field you selected. The record will be displayed if your text is a match at the beginning, at the end, or anywhere within the field value. In general a Contains search will return the largest possible number of records.

Figure 2

Search starts with

Starts with is a combination of a partial match and an exact match. The partial is the 'beginning' of the field value and the exact is that the beginning must be identical to the text you enter. If you enter three characters the search will find all those records where the field you selected begins with those three characters. If you were to select twenty characters, the search would find only records where the field selected begins with those twenty characters.

Figure 3

Search ends with

Ends with is another combination of partial match and exact match. In this case the matching records are those that end with whatever text you enter in the field you select.

Figure 4

Search exact match

An exact match (is) search will find only those records that are character-for-character identical to the text you enter in the field you select.

  • Even an unseen space at the end or the beginning will be enough to cause the search to 'miss' the record you want.
  • Entering a text that is shorter than the value in the field you want will always result in no records being found.
  • Entering a text that doesn't exist in any record (for the specified field) will result in no records being found.

Single search input box

In some cases a page or report may be set up to search a set of data fields, seeking a ’contains’ match in any of the fields in the set, for whatever text or digits are entered in the input box. (The exact set of fields will vary depending on the purpose of the page or report.)

In a few cases the single search input box uses the ’is’ search and will not return a record if there is only a partial match. Normally this is indicated in the help section for the form or page.

Lookup forms

A Lookup form is used for finding particular data when filling out another form. Lookup forms are almost always invoked by clicking on a small magnifying glass icon . tcmIS produces a child window to display the form, with a single Search input text box. Enter the search text and click the Search button.

tcmIS will display the records that it finds, in the lower portion of the Lookup form child window. Click on the row that has the record you want. (It will then be highlighted by a contrasting background color.) In the upper portion of the form a link will appear that facilitates using the selected record for the primary form. Click on this link: the child window will close and the data you had chosen will be inserted in to the primary form.

Type ahead search

A Type ahead search is used for finding particular data when filling out a form. As the desired text is entered into the text box, tcmIS presents a drop-down list of valid values that match the entered text. As more text is entered the possible matching values decrease. When the desired value is visible, click it on the drop-down list and it will be placed into the text box.

This technique replaces lookup forms in some situations.
