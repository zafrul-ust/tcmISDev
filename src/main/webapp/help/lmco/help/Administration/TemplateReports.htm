Template Report functions

There are four to six buttons, a pair of option buttons, and a text box across the bottom of the page which are used to manage (clear, save, recall, manage publication, and run) a report.

Figure 1

Running a report

  • There are two ways to run, or generate, a report: interactively or batch.
    • An Interactive report is produced on your own computer. Select the Interactive option button. A small window (illustration below) will open with the options to Save the report on your workstation or to Open the report in your own copy of Microsoft Excel.
    • A Batch report is produced on a tcmIS server. Select the Batch option button and enter a name in the Report Name box. An e-mail will be sent to you when the report is ready.
    • When all report parameters have been specified and you have chosen where to produce the report, click the Generate Report button to run the report. It will be produced in Microsoft Excel format.
Figure 2

Saving a report

  • The Save Template button is used to save a report as a template for later use. Click that button and tcmIS will open a child window to display one of two small forms to enter a name and description for the template (report) or update an existing template.
  • After providing (or editing) the name and description, click the Save link to finish saving the template and/or changes.
  • Once a template has been Saved the Publish Template button is added to the bottom of the report form
Figures 3 and 3A

Opening a template (saved report)

  • To use previously saved report settings (a "template"), retrieve it by clicking the Open Template button. Select the template you want from the list. Click the Open button to retrieve the template.
  • Click the Delete button to permanently delete a selected template.
  • When a saved template (report) is opened, tcmIS will display it in a new tab. If it had not previously been shown, the Unpublish Template button will now be available (in addition to the previously added Publish Template button).

Once a saved report (template) has been opened you may make any changes you like and run the report or save the changed report with a new template name. For information on modifying and running a report, see the help on the report on which the template is based: Cost Report, Ad Hoc Material Matrix, or Ad Hoc Usage.

Figure 4

Publishing a template

Publishing Templates is permissions based. Contact your Facility Admin to request the needed permission if the publish buttons don't appear.

Click on the Publish Template button and tcmIS will open a child window to display the Publish Template form. This form is used to specify to whom access (the meaning, here, of ’Publish’) is granted to open and run the report.

  • It is possible to ’mix and match’ access by entering user access for more than a single category, e.g., (individual) Users plus one or more User Groups. But if the Company option is selected it is not necessary and not possible to add individuals or groups.
  • To grant permission to one or more particular users, click the magnifying glass icon next to the User text box to open the User Lookup form. See Lookup forms for further information.
  • Specify one or more user groups by selecting the User Group check box and selecting the user group name of anyone to whom you want to grant access. Some organizations do not use User Groups; in this case the User Group selection will not be shown.
  • You may specify the entire company by selecting the Company check box.
  • Click the Publish Template link to save the list of those with access.

Unpublishing a template

Click on the Unpublish Template button to open a child window which will allow you to remove access from a user account. Removing access is done using the Publish Template form, which has a slightly different appearance for unpublishing.

  • It is possible to remove your own access to a published template. Simply select the check box for Remove this template from my view.
  • To remove access for another user account, select the check box for User and select the user name or names you want removed from access.

When you have specified all those you want removed, click on the Unpublish Template link in the upper left corner of the Publish Template form. When the unpublish process is complete, tcmIS will display a small message window stating success.
