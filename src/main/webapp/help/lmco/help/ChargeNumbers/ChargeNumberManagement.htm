Charge Number Management

The Charge Number Management page is used to view and manage (add, activate or inactivate) the association of WBS Elements with specific Approvers in tcmIS. This is used to identify approvers responsible for specific charge numbers.

The Charge Number Management page can be accessed from the Administration menu. The Charge Number Management page is launched in a new tab.

For information on information about valid charge number combinations (e.g., active WBS, Cost Centers and GL’s, etc.), see the help for Charge Numbers Set Up.

Figure 1

Select the facility for the charge number(s) from the Facility list.

In some cases you may also be able to choose the accounting system Account Sys list. In other cases there is only one possibility and only one value in the box.

Additionally, you may narrow your search by specifying all or part of the WBS Element. This is a “contains” search: tcmIS will match on records that include the exact text anywhere within the WBS Element.

Click the Search button. The WBS Elements and matched Approvers are displayed in the data grid.

Sort the list of WBS Elements or Approvers, ascending or descending, by clicking in the header of the data grid for the item you want. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Figure 2: Charge Number Management search parameter panel

A WBS Element-Approver combination may be active or inactive. If it is inactive it will not be used to route orders.

On the Charge Number Management page data grid the center column is the Active column. If the check box is selected for the line, the WBS Element-Approver combination on that line is active. If the box is cleared, the combination is inactive.

Figure 3

In order to add a WBS Element-Approver combination to tcmIS you must click the Add link on the upper-left of the data grid. tcmIS will add a new, blank row to the data grid where you enter the values for the WBS Element and the Approver Name in their respective columns. (The active check box is selected by default.)

  • Both the WBS Element text box and the Approver Name text box are Type Ahead Searches to allow easy location of valid Elements and Names. See second figure below.

No new combinations made on the Charge Number Management page are saved until the Update link is clicked.

Click the Create Excel button on the Catalog Add Tracking page to export results in Excel format.

In order to edit or delete a WBS Element-Approver combination in tcmIS you must right-click the record you want to edit or delete. tcmIS will display a small (context) menu with two menu items: Update Approvers and Remove Line.

  • Select the Update Approvers menu item to edit the existing combination. tcmIS will redisplay the record and make it possible to change the Approver. Just as in Adding a record, the Approver text box also serves as a Type Ahead Searches to locate a valid Approver Name.
  • Select the Remove Line menu item and tcmIS will immediately remove the WBS Element-Approver Name combination permanently.
