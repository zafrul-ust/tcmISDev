Work Area Stock Management

The Work Area Stock Mgmt page is used in tcmIS to view and manage parts in work areas: activate and inactivate parts; set stocking levels; and add parts to work areas. The Work Area Stock Mgmt page can be accessed from the Supply Chain menu: choose Work Area Inventory and select the Work Area Stock Mgmt menu item. The Work Area Stock Mgmt page is launched in a new tab.

Figure 1

Select the facility where the work areas are located from the Facility list.

The way in which you specify the particular work area(s) for the search affects the actions you can take on the records.

  • You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list.
    • If you leave the Work Area Group at the default value of All, the Work Area list will include all the work areas for the facility.
    • If you choose one or more specific Work Area Groups, only work areas from that group or groups will be shown in the Work Area list.
  • You may specify specific work area(s) from the Work Area list.
    • If you leave the Work Area list set at the default value of All, there are some actions you cannot take:
      • Update any of the records;
      • Add Nonmanaged Materials to any work area;
      • Add [a] Part to any work area; or,
      • Set [a] Part Level for any item.
    • You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box.
      • None of the limitations listed above will apply.
  • You may choose to narrow your search by indicating records for a specific count type or types
    • Select a single Count Type, or select multiple Count Types by holding down the CTRL key while clicking each wanted count type in the Count Type box.
  • Select the box labeled Show Inactive Parts to INCLUDE parts that have been inactivated.
  • You may provide something to at least partially identify the material you want using the Search by boxes. (Your search may be specified by Part Number or Item ID.) See Search functions for more information.

Click the Search button.

Click the Create Excel button to export the data in Excel format.

There are four links at the top left of the data grid panel:

  • Show Legend will open a small window on top of the current window to display a key to the meaning of the background color for individual records. Click the small X in the upper-right corner of this superimposed window to close it.
  • Update is used after any changes are made to finalize the changes in tcmIS. Click this link after any change or wait and click it after multiple changes have been made.
  • Click the Add Nonmanaged Mat[eria]l link to begin the process to add this kind of part. See Add Non-managed Materials for details.
  • Click the Add Part link to begin the process to add a Haas managed part. See Add Part for details.

There are several things that can be edited directly on the Work Area Stock Management page.

  • The Count Type column identifies the method used for inventory replenishment for the item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to record the expiration date and, in the case of expired material, beep loudly to notify the operator to remove the expired material.
    • KanBan a variant on MinMax where the reorder point is 1 bin (or fewer) and the maximum is 2 bins. Each bin contains a part-specific quantity which can vary from 1 to any number.
    • Not Counted indicates that this item is not included in the scan process count.
  • When the box in the Active column is selected, the part is available within tcmIS. Clear the box to deactivate an item. Select the box to activate an inactive item.
  • Default Part Type can be added or edited by selecting the desired type from the drop-down list for the record.
  • Select from the list in the Default Part Type column to specify whether a part is Aircraft or Non-aircraft by default.
  • Several columns contain information that is reserved for future enhancement and cannot presently be edited:
    • Default Permit
    • Default Application Method
    • Default Substrate
  • Use the list in the Temperature column to indicate storage temperature parameters for the item.

Click the Update link above the data grid to save any changes.

Selecting the Add Part menu item will cause the Add Part page to open in a child window. Also, in a situation where the search finds NO records that match your Search criteria, the display grid panel will display a message stating 'No Data Found' as well as an Add Part link.

In either case, click on the Add Part link to open a child window which displays the Add Part form.

  • If working on more than one Work Area, select work area for the Add Part from the Work Area drop-down list.
  • If appropriate, select the Catalog from the Catalog drop-down list. (There may be only a single option.)
  • Enter the correct Part Number in the Part Number text box. Click on the magnifying glass icon to open the Part Number lookup form. (See Lookup Forms for more details.)
  • (None of these are required numbers but provide them if you can.)
    • Enter the minimum quantity to maintain in the work area in the Reorder Point text box.
    • Enter the maximum quantity to maintain in the work area in the Stocking Level text box.
    • Enter the quantity for a reorder for the work area in the Reorder Quantity text box.
    • Enter the bin quantity for the work area, the work-area-and-part-specific quantity that makes up 1 bin, in the KanBan Reorder Qty text box.
    • If known, enter the number of days which it takes to procure and ship the material in the Lead Time In Days text box.
  • The Count Type column identifies which base is used for inventory replenishment for an item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to recognize expired material and beep loudly to notify the operator to remove the expired material.
    • KanBan indicates that replenishment is done based on bin count rather than item count.
    • Not Counted indicates that this item is not included in the scan process count.
  • Select the temperature parameters from the Tier II Storage Temperature list.

Click the Add button to add the part to the Work Area. Use the Work Area Labels page to print a new Work Area Baseline report and Bin Labels.

Selecting the Set Part Level context menu item will cause the Part Level page to open for the item on the selected row. There you may enter or edit:

  • The minimum quantity to maintain in the work area using the Reorder Point text box.
  • The maximum quantity to maintain in the work area using the Stocking Level text box.
  • The reorder quantity to restock the work area using the Reorder Quantity text box.
  • The KanBan reorder quantity, the work-area-and-part-specific quantity that makes up 1 bin, to restock the work area in the KanBan Reorder Qty text box.
  • The number of days which it takes to procure and ship the material in the Lead Time In Days text box (if known).
  • Any comments in the Remarks text box.

Click the Update link at the upper-left of the data grid to save any changes.
