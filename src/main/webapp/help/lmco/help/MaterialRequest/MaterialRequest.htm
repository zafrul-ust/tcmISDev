Material Request (Orders)

The Material Request (MR) page allows you to view, process, and manage an order. It is launched in its own tab. The Material Request page is always specific to one single Material Request and supports different actions based on the status of the MR.

The Material Request page is used in the ’Shopping Cart’ process and in Order Tracking activity.

  • From the Catalog page, to create, view, or work on a Material Request, place items into the Shopping Cart. When all the needed items are in the order, click on the Check Out link.

  • From the Order Tracking page, to view, or work on a Material Request (order) for an item, right click the item line and select the View/Approve MR menu item.

MR page actions

A variety of actions are possible on the Material Request page depending on the status of the MR as well as the user’s authority or responsibility. These include:

  • Edit details on the unsubmitted MR by providing or modifying information. See MR optional information and MR required information.
  • Save the material request at draft status if you do not have all of the required information by clicking the Save link. Draft orders can be retrieved and completed through the Order Tracking page.
  • Click the Delete link to delete the entire material request if the MR has not been submitted.
  • Click the Delete Line link to remove a selected line from a material request if the MR has not been submitted.
  • Click the Return to Cart link to return to the Catalog page if more parts need to be added to the order. This link does NOT remove the selected item FROM the Cart: it displays the tab with the Catalog page so that additional items may be added to the Cart.
  • Click the Print This Page link to print a copy of the material request.
  • Click the Request Cancel link to request an MR be cancelled. See Cancel an Order for details on when, if, and how an order can be cancelled.
  • Authorizations
    • Click the Approve link to approve the order. See Approve the MR
    • Click the Reject link to disapprove the order. See Reject the MR
    • Click the Release link to release the order. See Reject the MR
  • View the MR Allocation Report by clicking the MR Allocation link.

Material Request page optional information

There is an opportunity on the Material Request page to provide additional delivery and user information to indicate the point-of-use for the order. When provided, this information applies to all line items on the request and will appear on the delivery ticket shipped with the material. These fields are flexible and remarks can be tailored to assist the end user once the item is delivered.

  • You may enter the end user's name in the End User Name text box.
  • You may enter the end user's department in the Department text box.
  • You may enter contact information in the Contact Info text box.

Material Request page required information

Charge numbers (lower left) and delivery information (lower right) must be provided for each line (item) in the Material Request. Select each line (item) on the order in turn and complete the required information. As each Line is selected, its line number is placed into the heading on the Line Item Detail section. (But, see the Shortcut, below.)

  • Under Line Item Detail select the Scrap/Obsolete check box if this order should be filled from material in inventory that has been declared scrap or obsolete.
  • Under Delivery Instructions any need by date was carried over from the shopping cart, but it may also be updated on this page.
  • There are two types of need by (’Delivery by’) dates. Your selection will inform HGI exactly when to deliver the material.
    • Deliver By deliveries are those that are needed on or before a specified date. Enter the date in the text box to the right of the ’Delivery by’ box using the calendar window that will open when you click into the box.
    • Schedule deliveries are those that are needed exactly on a specified date or set of dates. Click the Schedule button that will appear the right of the ’Delivery by’ box (see below).
  • Select the dock from the Dock list.
  • Select the work area from the Deliver To list.

Shortcut

As stated, each line (item) must be assigned charge numbers and delivery information. But there is a Copy button provided to speed the process. Once you have entered the required information for any one item, you may then click on the Copy button and the identical information will be added to all remaining items in every BLANK field. Items which already have charge numbers and/or delivery information will not be affected. Once you press the Copy button an Undo button will be provided in case you want to remove all copied information. After any Copy you may still manually update any field for any item.

Submit an order

The final step in placing an order is to submit the order. Before submitting the order, you should review the order for accuracy and completeness.

Click the Submit link to complete the process of placing the order.

If the order is for a part that is an OOR whose item is currently in stock, tcmIS will prompt the requestor to allow the OOR shelf life requirement to be waived to allow the stocked item to be shipped immediately. You are informed of the minimum remaining shelf life for the delivered items from stock. Often this material is currently owned by your company and may be a lost value if no one accepts the material. If this transaction is authorized, tcmIS will attempt to fill the request immediately. There are some occasions where shipping an OOR from stock could threaten the ability to ship the stocked item to the Min/Max requirement. In these cases, the order will revert to normal OOR processing.

When a Material Request is submitted, tcmIS will open a child window and generate an MR Allocation Report. This is a detailed listing of all the items on the submitted MR.

Cancel an Order (Cancel a Material Request, Cancel an MR)

Sometimes an order can be cancelled; sometimes it cannot. It depends on the type of order and the status of the order, therefore the initial cancellation is treated as a request until tcmIS, or HGI personnel, determine that all the criteria are met for cancellation.

  • The requestor of an order can request a cancellation of the order. Locate the order (MR) on the Order Tracking page and click the View/Approve MR context menu item. See the Catalog page for details.
  • If the requestor is out of the office, or no longer with the company, the facility Administrator can request the order be cancelled.
  • Anyone in the company, with appropriate authority, can request that the HGI CSR cancel an order.

The type and status that determine whether an order can actually be cancelled are:

  • If the MR Type is MM and allocated to stock, HGI will cancel the order. A Restocking Fee may be associated with the cancellation of an MR.
  • If the MR Type is OOR and the PO status is Open, the HGI buyer will contact the seller to request the order be cancelled.
    • If the seller agrees, HGI will cancel the order.
    • If the seller declines, the customer must take the material.
  • If the MR Type is OOR and the PO has not been sent to the seller, HGI will cancel the order.
  • If the MR Type is SCH the HGI buyer will get in touch with the seller to negotiate the end of the scheduled deliveries as close to the desired time as possible. The buyer will notify the requestor of the final delivery date.
