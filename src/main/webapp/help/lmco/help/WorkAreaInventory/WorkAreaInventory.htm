Work Area Inventory

The tcmIS Work Area Inventory page is used to view information regarding on hand inventory in cabinets. It can be accessed from either of two menu selections.

  • The Work Area Inventory page can be accessed from the Supply Chain menu. Under Work Area Inventory select the Work Area Inventory menu item.
  • The Work Area Inventory page can be accessed from the Reports menu. Under Inventory select the Work Area Inventory menu item.

Using either menu selection, the Work Area Inventory page is launched in a new tab.

Figure 1

Select the facility which services the work area from the Facility list. Select the work area from the Work Area list. The Work Area box will be populated with all active work areas for the selected facility. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

Select the check box labeled Show Only Quantity > 0 to see only work area items that currently have stock.

Click the Search button. All active items in the selected work areas will be shown. Details of each Receipt ID in the work area are displayed including manufacturer lot number, expiration date, last count date, last count quantity, quantity delivered since the last count, total quantity, unit cost, and stocking levels.

Figure 2

By default, the search results (inventory items) are sorted and displayed by Work Area. You may also sort by Part Number or Item ID. Select the desired sort option from the Sort By list.

You may also sort by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order.

Figure 3

You may narrow the range of your inventory search by using the three Search boxes.

  • In the first box choose whether to base the search on Part Number, Item Desc[ription], or Item ID.
  • In the second box specify whether the search is for an exact match, using is, or a partial match, using contains.
  • In the third box enter the number or description you want fully or partially to match in the search.

Click the Search button.

You may search for work area inventory by item description. Select Item Desc[ription] from the first Search box and enter a description of the product. Click the Search button.

You may display all work area inventory of an Item ID by selecting Item ID from the first Search box and entering the item number. Click the Search button.

You can use the Work Area Inventory page to locate materials in the work areas which are expiring soon. Enter the number of days in the Expires In and To boxes as shown below. Click Search.

You may enter a minimum number of days before expiration, or a maximum, or both.

Figure 7

Right-click on any line and select the Show Inventory Detail menu item to display details on inventory of the item at the warehouse that serves the work area. The Inventory Details page will be displayed in a new window.

Figure 8

The Inventory Details page shows details on the inventory at the warehouse including quantity, status, manufacturer lot number, and expiration date.

This is particularly useful when the data in the cell does not fit the default size of the cell: it allows you to see the entire contents of the cell without having to adjust the width of the column.
