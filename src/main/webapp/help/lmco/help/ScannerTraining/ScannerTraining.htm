Scanning Overview

Scanning process overview

See, also, Scanning Tips.

  • Barcode scanners are used to inventory the chemicals.
  • Each Work Area and Bin is identified with a unique barcode label.
  • A list of work area and part IDs are loaded into barcode scanners from tcmIS.
  • After counting, the barcode scanner is used to upload the inventory data to tcmIS.
  • tcmIS calculates replenishment requirements.
  • tcmIS creates material requests as required.

Work Area ID Label

Each work area is identified by a work area ID Label. The label is typically located near the work area baseline document.

Starting TCMScan

To begin, click Start in the upper left corner on the Windows Mobile Today touch-sensitive screen.

  • If TCMScan is listed on the Start menu select it.
  • If TCMScan is not displayed on the Start menu select the Programs menu item and then select the TCMScan icon from the Programs screen.
Figure 6: Start menu and Program screen

Logging into the Scanner

Note: The scanner must be in the cradle to log in.

Enter your tcmIS login ID in the User ID box.

Enter your tcmIS password in the Password box.

Select your company from the Company list. (Most users have only one choice here.)

Press the Login button on the screen to complete your log in.

When you are successful you will see the Main Page as illustrated below. In addition to the three function buttons (see below) notice the information and links at the bottom of the Main page:

  • Information
    • The currently logged in user name.
    • The date and time the work area data was last loaded.
  • Links
    • Click the Menu link to display the main menu
    • Click the Load button to display the Work Area loading screen.
    • Click the Scan button to display the Work Area scanning screen.
    • Click the Submit button to display the Send Scans screen.

Clicking the Exit button on the Log in page also cancels your log in and exits the TCMScan application.

Login Error Messages

If you receive an error, please read through the following troubleshooting suggestions.

Username or password is wrong.

  • Select Retry.
  • Re-enter your user data and try again.

Make sure that the

  • Scanner is in the cradle.
  • Cradle has power.
  • Cables are connected between cradle and PC.

Contact the Helpdesk if the problem continues.

Loading Work Areas

Note: The scanner must be in the cradle to load work area information.

On the Main page (Figure 7, right image), click the Load button.

The facilities list will display all of the facilities that the user has permission to scan. Most users will see only one facility.

Select your facility.

  • Select the check box, or
  • Use the up and down arrow keys. Press the space key when your facility is highlighted.

Click the Load button and the work area information will be updated from tcmIS.

Selecting the Cancel button will return to the main screen without loading work areas.

Figure 9

Starting the Work Area Scan

On the Main page (Figure 7, right image), click the Scan button. The scanner will be readied to scan a work area.

Scan the work area label, using the scanner as shown, or enter the work area ID using the scanner keypad.

  • If the work area ID is entered on the keypad you must press enter on the keypad or the Next button on the screen to continue.
Figure 10

Work Area Error Messages

If you receive an error, please read through the following troubleshooting suggestions.

Invalid Label.

  • Caused when the label scanned is not a work area ID label.
  • Rescan the work area label to continue.

No data for this work area has been loaded.

  • Caused when the wrong facility was chosen when performing the Load Work Areas step.
    • Exit the scanning screen, and perform the Load Work Areas step again this time choosing the correct facility.
  • If this error continues after loading the correct facility contact the Helpdesk.

You must scan a work area before scanning a part.

  • Caused when the scanner is expecting a work area ID label but a part ID label is scanned instead.
  • Scan or rescan the work area label to continue.

Entering a Part ID

After you enter the work area and the Next button the scanning screen is presented. The primary purpose is to enter the Part ID information for each item.

Scan the part label, using the scanner as shown, or enter the ID using the scanner keypad.

  • Click the Next button to advance to the quantity prompt.
  • Click the Back button to return to the work area prompt.

The lower portion of the screen displays information about the scan process and two links:

  • The current work area ID.
  • The ID of the last Part scanned.
  • The quantity of the last Part scanned.
  • Click the Exit button to return to the main screen.
  • Click the 'keyboard' icon to display the TCMScan Help pages.
Figure 11

Part ID Error Messages

If you receive an error, please read through the following troubleshooting suggestions.

Invalid Label.

  • Caused when the label scanned is not a part ID label.
  • Rescan the part label to continue.

This part has been inactivated.

  • Caused when the part has been inactivated in tcmIS.
  • Do not count this part, it may have material that is being used up but it cannot be replenished because of its inactive status.

This part has not been configured for this work area.

  • Caused when a part has been moved to a different work area without being relabeled.
  • The first 8 characters of the part ID represent the work area ID.
  • Return the part to its proper work area, or
  • Verify the tcmIS record is updated with the proper work area ID and replace the part ID label.

This part has not been configured.

  • Caused when the work area ID scanned does not match any of the data loaded on the scanner.
  • If the part was configured after the scanner was loaded last, reload the scanner and try again.

Entering the part quantity

Count the items and enter the quantity using the keypad.

Note in the illustration: 3 items; 3 key on scanner keypad; value of 3 entered into the Quantity text box.

Press the Enter key on the keypad or the Next button on the screen.

When the work area count is complete, click on the Exit link at the bottom of the screen to return to the Main page before proceeding to the next work area or submitting scans to tcmIS.

Submitting Scans to tcmIS

Note: The scanner must be in the cradle to submit scan data to tcmIS.

On the Main page (Figure 7, right image), click the Submit button. The scanner will be readied to send scans.

Click the Send Scans button to send the inventory information to tcmIS.

The Cancel button will return the screen to the Main page.

Figure 13

Scan Manager

After the scanning, inventory, and uploading is complete, empty the scanner in preparation for the next set of scans.

On the Main page main menu (Figure 7A), click the Scan Mgr menu item to display the Scan Manager page. All scan sessions stored on the scanner will be displayed, grouped by user, on the scan manager screen.

Choose one or more scan sessions by selecting the corresponding check boxes. Scans can also be selected by using the up and down arrow keys to highlight the desired scan and then pressing the space key.

  • Pressing the Delete button on the screen will deleted any selected scans.
  • Pressing the Delete All button on the screen will delete all scans.
  • Pressing the Done button will display the Main Screen.
