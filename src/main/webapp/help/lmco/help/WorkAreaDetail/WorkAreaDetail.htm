Work Area Detail

The Work Area Detail page is used to view the authorized contents for each work area, including the reorder point (RP) and stocking level (SL). The Work Area Detail page can be accessed from the Reports menu. Under Inventory select the Work Area Detail menu item. The Work Area Detail page is launched in a new tab.

Figure 1

Select the facility where the work areas are located from the Facility list. You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list. The Work Area box will be populated with all active work areas for the facility or the work areas for the chosen work area group.

Select the specific work area(s) from the Work Area list. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

Click the Search button. The authorized contents of the selected work areas will be shown, sorted by work area.

Figure 2

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 3

You may right click anywhere on the data to get a Print menu item which will allow you to produce a copy of the report on any printer accessible from your workstation.

Figure 4

Click the Create Excel button on the page to export the data in Excel format.
