Ad Hoc Inventory

The Ad Hoc Inventory report is on inventory in cabinets during a specific time; the reported items can be based on any of one or more chemical lists, or the entire inventory. The details of this report may be specified by the user and can include any of the fields included in the list of Base Fields.

The Ad Hoc Inventory report can be accessed from the Reports menu. Select Inventory and then Ad Hoc Inventory. The Ad Hoc Inventory page is launched in a new tab.

Figure 1

Using the respective boxes in the Search Fields section of the page, specify the place and dates on which you want a report.

  • Leave Facility as My Facilities or choose one.
  • Leave Area at the default (it varies by facility) or choose one.
  • Leave Building as All or choose one.
  • Leave Room as All or choose one.
    • Select a Begin Date and an End Date by clicking into the text box. A Date Selector (calendar) form will be shown in a child window.
    • You may choose a beginning and/or ending date by clicking on it. To make the report open ended, either may be set to ’None’ using the None button in the lower left corner.
    • The Date Selector calendar allows you to browse one month at a time, moving forward or backward with the 'double chevron' (<< or >>) links in the upper-left and upper-right. Alternatively, you may use the Month and Year selection lists.

Figure 2a

The Chemical List multi-line selection box is used to choose the list or lists to be used for the report. The Base Fields multi-line selection box is used to choose the fields to be to be displayed on the report.

  • Select any item by clicking on it: it will be highlighted in a contrasting background color. To select more than one item, hold down the keyboard Ctrl key when clicking.
  • Add the items to the Report List box or the Report Fields box on the right by clicking the right-pointing arrow .
  • Remove items from either box by selecting them and clicking the left-pointing arrow icon .
  • Move an item in either box up or down relative to the others by selecting it and clicking the up- or down-pointing arrow icons. This is particularly helpful to get the fields on the report in the order you want.
Figure 3

A description of each base field is available by clicking the Description link above the Base Fields box. It will open in a child window which you may close when you are done with it.

Figure 4

To clear all the report parameters on the current page, click the Clear Template button. This is the same as closing the page and reopening it.
