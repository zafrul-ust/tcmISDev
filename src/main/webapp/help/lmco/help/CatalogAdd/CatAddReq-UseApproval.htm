Catalog Add Request - Use Approval tab

The Use Approval tab on the Cat Add Req (Catalog Add Request) page is used to view and manage use permissions on parts (part number records). While the Use Approval tab is visible at all times, it is not accessible until and unless at least one Approval Code has been added to the HMRB tab.

Five individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL

  • QPL - the default tab (shown when the page is opened), used to view and manage products on or being requested for the QPL.
  • HMRB - used to view and manage Hazardous Material Review Form information for the products on the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.
  • A request for a Use Approval on an existing part cannot be combined with any other request.
  • The Use Approvals for a part apply equally to all the products and materials on the QPL for the part.

Use Approval is accessed after opening the Cat Add Req page by clicking on the Use Approval tab. The Use Approval tab lists the Work Areas for which the part number is approved (or disapproved).

Figure 1

To add approval or disapproval for one or more work areas, click the Add Work Area link at the upper left of the tab area. The Add/Edit Use Approval form will open in a new window. On the form:

  • Specify the work area, or set of work areas, for which you want to set approval status in the Work Area list.
  • Specify the inventory group, or set of groups, in the Inventory Group list.
  • Set the status in the Request Status list. The default status is 'Approved' but you may choose Unapproved to disallow use of a material if needed.
  • Enter a description of how the material is used in the Use and Process Desc[ription] text box. This is a required field.
  • Select a user group if the material should be ordered only by specific users or leave it at Approved to all users in the User Group list.
  • Enter the estimated annual usage in the Estimated Annual Usage box.
  • To limit usage of the material, enter any use limits in the Restriction 1 and Restriction 2 boxes as (quantity) per (number) (days or months).
  • Click the Submit link in the upper left of the form to initiate the approval status request.
Figure 2

The form will be closed and the Use Approval will be added to the Cat Add Req - Use Approval tab as part of the current request.

Repeat this process as many times as needed to add the necessary use approval records.

  • Use Approval is assigned by specifying an Approval Code for a part.
  • Every work areas that “belongs to” an Approval Code will be granted that code’s use approval.
  • Every Approval Code must have at least one work area assigned.

The newly added use approval line can be edited or deleted before the QPL modification is finalized. Right click on the new line to access a context menu. Select the Edit menu item to reopen the Add/Edit Use Approval and make and submit changes. To remove the line completely, select the Delete Line menu item.

You may also edit by clicking into the Request Status box and/or the Inventory Group box and selecting another option.
