Catalog Add Request - HMRB tab

The Cat Add Req (Catalog Add Request) page with its HMRB tab is used to view, edit, and add approval codes (needed for the Hazardous Material Review Board) for materials and items.

Five individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL
  •  
  • HMRB
  •  
  • Use Approval
  •  
  • Spec
  •  
  • Flowdown
  • QPL - the default tab (shown when the page is opened), used to view and manage products on or being requested for the QPL.
  • HMRB - used to view and manage Hazardous Material Review Form information for the products on the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.

HMRB is accessed from the Cat Add Req page by clicking on the HMRB tab.

To add an approval code click on the Add Approval Code link. tcmIS will open a child window and display the Add/Edit Approval Code form. (This form will expand significantly as soon as the first selection is made.)

  • Select the intended purpose of the material from the Usage Category list, either Production or Non-Production.
  • Select the usage subcategory from the Usage Subcategory list.
  • Select the material category from the Material Category list.
    • Most categories will cause a set of additional selection and text boxes to be made available in the lower panel.
  • Select the material subcategory from the Material Subcategory list.
  • You may specify, if appropriate, a start date, end date, or date range with the Begin Date and End Date: boxes: clicking into either box will open a small calendar form in a child window where you may select a date. (Choose the None button to close the calendar form without choosing any date.)
  • A Purchasing specification is required: select the appropriate check box to indicate which.
    • Haas Purchased
    • P-Card
    • SAP
    • Haas Purchased
    • P-Card
    • SAP
  • Select either Yes or No from the drop-down list in answer to the question [do you anticipate the] Maximum amount on hand > 54 gallons?
  • If pertinent, select the check box for either Import or Export.
  • Provide any name, business, address, e-mail, or telephone information in the Point of Contact text box.
  • In the lower pane, labeled ‘Process Information’, additional check boxes, drop-down lists, and text boxes may appear, depending on the Usage and Material selections made. Provide information on the Approval Code using these. See Figure 2A for an example.

When all the information is provided, click the Save New and Return link in the upper-left of the form to add the calculated approval code to the HMRB tab on the Cat Add Req page. The form and the child window will close.
