Catalog Add Request - QPL tab

The QPL tab on the Cat Add Req (Catalog Add Request) page is used to request and manage additions or changes to the QPL (qualified products list) in tcmIS. The QPL tab specifics are shown when you first open the Cat Add Req page or when you click on the QPL tab when the Cat Add Req page is already open.

Five individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL
  •  
  • HMRB
  •  
  • Use Approval
  •  
  • Spec
  •  
  • Flowdown
  • QPL - the default tab (shown when the page is opened), used to view and manage products on or being requested for the QPL.
  • HMRB - used to view and manage Hazardous Material Review Form information for the products on the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.

The behavior and appearance of the Cat Add Req - QPL tab page will differ somewhat based on which of the following links was clicked. All of these are described below.

  • The links above the data grid on the Catalog page: Modify QPL | New Part from: Beginning, Part, Item, Item (modify Pkg) | New Work Area Approval;
  • The context menu items on the Catalog Add Tracking page: View/Approve Cat Add (including Approval Detail and Approval Rule results) or Resubmit.
Figure 1

Modify QPL

The Modify QPL process is used change the QPL assigned to a part number. There are two links just below the QPL tab that are used in the Modify QPL process, and one context menu item.

  • The Add Item link is used to begin a request for a new addition to the QPL.
  • The Show Obsolete Items link is used to display any items that have been on the QPL but are no longer.
  • A Fadeout From QPL menu item is used request inactivation of and approved product from the QPL.

When the Add Item link is clicked tcmIS will open a child window to display the Search Global Item form. (This is fundamentally a Lookup form.)

Enter information about the product in the Search Text box and click the Search button. tcmIS will display all records that match or include the text in the Search Text box. In addition there will be a New Material link at the upper left of the data grid which can be used to begin a request for material that is not yet in the catalog.

Figure 2

The desired material may already be in the catalog. If so, select that record and additional links will be placed into the upper left of the data grid.

  • The Add Item link will place the selected record into the Cat Add Req page, with a status of ’Pending Approval’, and close the Search Global Item form and the child window that displays it.
    • Don't be confused by the fact that the term Add Item is used for the link on both this Search Global Item page and the preceding Cat Add Req page.
  • The New Packaging from: Item link will launch the New Package from Item request process.
  • The New Packaging from: Material link will launch the New Package from Material request process.

Regardless of whether matching material is in the system, the New Material link remains available.

New Packaging

The two New Packaging processes, Item or Material, are fundamentally the same. In fact, in some cases the item is the material, and vice versa. But both are available for those situations where they are different and a choice has to be made.

If the product is found but a different size is desired, highlight the item by clicking anywhere on the line and click the appropriate New Packaging link. The Edit Item Info page will be opened in a new window to allow you to specify details about the new packaging.

Figure 4

Since you are working on an existing item or material and requesting only a different packaging for the QPL, most of the information is already shown on the form.

  • If known, enter the Shipping Weight and select the Units from the list at the top of the form.
  • Enter the number of units in the # Per Comp box at the bottom of the form. (The default is 1 which is already there. Change it if necessary.)
  • Enter the Package size in the Size box and select the unit of measure from the Unit list. The number of units should be the smallest individual Package that can be purchased. For drums, pails, rolls, cans, etc., this is typically 1.
  • In the Package Style text box enter that information. (The Type ahead feature will help locate a valid style.)

If there is more than a single component, please look at the information on Kits.

New Material

If the product or material you want cannot be located with the Search Global Item form, click the New Material link to define a new product. The Edit Item Info form will open in a new window. Complete the fields as described below. Required fields are marked with an asterisk (*).

  • If known, enter the Shipping Weight and select the Units from the list.
  • Enter the name of the product in the Material Description box. Include all necessary information to procure the product including color, type, etc.
  • Enter any information regarding the grade of the material in the Grade box.
  • If the material has a specific size not described in the previous fields, enter it in the Dimension box. For example, a roll of film may have a length, width, and thickness.
  • In the Manufacturer text box enter that name. (The Type ahead feature will help locate a valid manufacturer name.)
  • Enter the product name in the Manufacturer's Trade name box. Be sure you use the name as it appears on the MSDS.
  • If known, enter the Manufacturer’s Part Number into that text box.
  • Enter the number of units in the # Per Comp box at the bottom of the form. (The default is 1 which is already there. Change it if necessary.)
  • Enter the Package size in the Size box and select the unit of measure from the Unit list. The number of units should be the smallest individual Package that can be purchased. For drums, pails, rolls, cans, etc., this is typically 1.
    • Some Units, when selected, will add two additional input boxes to the Edit Item Info form, both of which must be entered.
      • Provide a value in the Net Size input box.
      • Select a Net Unit from the drop-down list.
  • In the Package Style text box enter that information. (The Type ahead feature will help locate a valid style.)

If the product or material has only this one component, click the Submit link at the top left of the page to close the form (and window). The product in the new material configuration will be added as a line on the QPL tab of the Cat Add Req page with a status of 'Pending Approval'.

If there is more than a single component, please look at the information on Kits.

See Finalizing a QPL modification request

Kits

If a product has more than one component (or a kit) right-click on the Component tab and select Add Component. A new tab will be created for the additional component of the kit. This may be repeated for each needed component of the product.

Except for the first component tab, any component tab can be deleted by selecting the Remove Component menu item.

Complete the fields for each component. Those marked with an asterisk (*) are required but supply all the information possible.

When all components have been added and identified, click the Submit link at the top left of the page to close the form (and window). The product, with all components stored in tcmIS, will be added as a line on the QPL tab of the Cat Add Req page with a status of 'Pending Approval'.

Once a new item has been submitted you cannot add additional components to the item although you can edit any of the originally submitted components of the item.

Finalizing a QPL modification request

When you have completed work on all five tabs of the Cat Add Req page and your requested addition is in the Pending Approval status, you are ready to complete the request. If you need to, because the page or screen is not wide enough, adjust the data grid with the bottom 'slider bar' control to see the rightmost columns.

Some of the final information you must provide depends on the Yes or No choices selected for the option buttons on the upper right portion of the page: Production Material, Time and Temp Sensitive, and Room Temp Out Time Required.

If you are finalizing a New Part or a New Part from current part then

  • Select the check box for Incoming Testing if that is required for the part.
  • If the part is Production Material, enter the EMAP ID just below the Production Material option buttons. (The text entry box is not there unless you have selected its Yes button.)
  • Leave the Yes option button selected under the Time and Temp Sensitive label if the product or material is time sensitive or temperature sensitive. Otherwise click on the No option button.

For all Catalog Add Requests

  • Enter your company's MSDS number in the Customer MSDS No box on the product line in the data grid.
  • If the product or material is time sensitive, enter the shelf life information on the product line of the data grid:
    • Click on the Shelf Life Unit text box to cause the list to appear; select the appropriate unit/measure.
    • Unless the Shelf Life is Mfg Recommended or Indefinite, enter the number of units in the Shelf Life text box
  • If the product or material is temperature sensitive, click into the Storage Temp text box to cause the list to appear; select the appropriate temperature range.
Figure 11

To add a suggested supplier, select the Add Suggested Supplier menu item from the context menu. The Add Suggested Supplier form will be opened in a new window.

Figure 12

Enter any information you have regarding the supplier on the form. When done, click the Submit link in the upper-left corner of the form. The information will be stored in tcmIS for this product line and the form/window will be closed.

Repeat these steps for each product added to the QPL.

Figure 13

When done, click the Submit link in the upper-left corner of the Cat Add Req form. The information will be stored in tcmIS.

Alternatively,

  • Click the Save link to save the request as a draft. You may return to and complete it later but it will not be reviewed nor approved until it is submitted.
  • Click the Delete link to discard the request completely. None of the information entered will be saved.

Based on your company's approval policy and process, the appropriate approvers are notified by e-mail to log in to tcmIS, review the request and approve or reject. When all the approvals are made the product is made available for use for the part number. A singe rejection means the request is denied and the product is not added to the catalog.

The status of the request can be tracked with the Catalog Add Tracking page.

View or Upload an MSDS

On an existing product, you may view the MSDS by right-clicking on any column in the line (except the five leftmost: Status, Item, Kit MSDS, Kit Desc, and Approval Code). On the context menu, select the View Component MSDS item. The View MSDS menu item is only available if tcmIS has a copy of the MSDS on file.

On either an existing product, or a new material, where there is no MSDS stored in tcmIS, you may upload a copy of the MSDS to tcmIS. Select the Upload All MSDS(s) for Item link or right-click on any column in the line (again, except the five leftmost) where you want to add an MSDS. On the context menu, select the Upload All MSDS(s) for Item menu item to open the Upload File form.

Figure 14

Click the Browse button, locate and select the file on your computer. Click the Send File button. The file will be stored in tcmIS and associated with the part record Modify QPL request. The Upload File form will then be closed.

Removing a QPL Item

To request inactivation of an approved product from the QPL, right-click on the Status or Item (number) and select the Fadeout from QPL menu item. If approved, this will place the item in draw down status. Any material still in the warehouse will be delivered until stock is depleted. When depleted, the item will become obsolete.

To reverse the fadeout request before it is approved, right-click on the Status or Item (number) and select the Remove Fadeout Request menu item.

Figure 16

To remove a product in pending approval status, right-click on the Status or Item (number) and select the Delete Line menu item.

New Part from:

There are four ’starting points’ for initiating a Catalog Add Request on the Catalog (after a part or item is selected from the data grid) and they differ basically in the amount of information for the new item which is already in tcmIS. All requests begin with certain information:

  • The Requestor is the currently logged in user.
  • The Request Date is the day’s date.
  • The Facility, Work Area, and Catalog are those selected for the Catalog search on the page where the link was clicked.
  • The Request [number] is assigned by tcmIS (in sequential order).

(Some other information may have default values which may be changed in the request process.)

The four starting points build on these values, supplying from the tcmIS database what is already on record for the item or material. Click on whichever link best describes the situation for the new part request.

  • New Part from: Beginning starts with a clean slate. All information for the new part must be provided in the request process although some pieces of information may be looked up while completing the forms.
  • New Part from: Part populates the request forms with information on file for the selected Part and the remaining information must be provided in the request process, again some pieces of information may be look up.
  • New Part from: Item populates the request forms with information on file for the selected Item and the remaining information must be provided in the request process, again some pieces of information may be look up.
  • New Part from: Item (modify Pkg) populates the request forms with information on file for the selected Item and opens an Edit Item Info form to enter the details about the new packaging for the item. (See, also, information on this page for New Packaging.)
Figure 18

Before you can specify the products and materials that are to be associated with the new part, you must specify information about the part itself on the Cat Add Req form.

  • Provide a part number in the Part No box. Be sure the part number you use does not duplicate an existing part number, and that it conforms to your company's standards for a part number.
  • Select the stocking method for the part from the Stocked list.
    • Select MM if the material should be stocked in the warehouse.
    • Select OOR if the material should be Order on Request, or a non-stocked item.
  • If this part is replacing a previously cataloged part, enter that part number in the Replaces Part No box. Use the Replacement Part lookup form by clicking the magnifying glass icon to the right of the Replaces Part No text box. See Lookup forms for details.
  • Enter a description of the material as it should appear in the catalog in the Description box.
  • If appropriate, provide instructions for recertification of an item in stock using the Recert Instructions multi-line text box.
    • Explicitly state the maximum number of recertifications that can be done for the part in the # Recert Allowed text box.
  • If incoming testing is required for the part, select the check box for Incoming Testing to record that.
  • If the part is Production Material, enter the EMAP ID just below the Production Material option buttons. (The text entry box is not there unless you have selected its Yes button.)
  • Using the Time and Temp Sensitive option buttons and the Room Temp Out Time Required option buttons, indicate if the material is time and temperature sensitive and whether room temp out time is required.
Figure 19

At this point in the process, you may save the new part record, or submit it, using the Save or Submit links in the upper-left of the page. But the request is not finished until you

  • Complete a Modify QPL by adding one or more products or materials to the QPL (not required except for New Part from Beginning, but allowed for other situations.)
    • This is required for New Part from: Beginning.
    • this is optional for other New Part from: situations as the existing parts or items will already be in place with a status of Pending Approval.
  • Provide any needed information on the other tabs (some of which may also in place with a status of Pending Approval).
    • Use Approval
    • HMRB
    • Spec
    • Flowdown
  • Finalize the request.

View/Approve Cat Add

Approvers may make their judgment about a Cat Add Request beginning on the Catalog Add Tracking page by right-clicking anywhere on the line of a record for which they have responsibility and selecting the View/Approve Cat Add menu item.

To approve the request the approver should click the Approve/Reject link in the top left of the Cat Add Request page. The Approve Detail form will open on top of the Cat Add Request page.

Figure 22

Select the check box next to each approval role you are taking. (There may be only one role, or there may be more, depending on your responsibilities and authority.) Enter any notes in the Comments text box. Click the Approve button to record your approval or the Reject button to record your rejection.

The request will be sent to the next approver. Once all approvals are complete, the requestor will receive an email documenting that the request has been completed and the product is ready to order or that the request has been rejected.

Figure 23

Approval Detail

When approval has been decided, whether to approve or to reject, the CatAddReq-QPL page will display three new links on the upper-left corner of the page: Approval Detail | Approval Rules: Results, All Tests

The click on the Approval Detail link to open a small window that will be superimposed on the current window to display details about the approval decision(s) for the request. Click the small ‘x’ in the upper-right corner of the superimposed window to close it.

Figure 24

Approval Rules Results

There are two Approval Rules forms. Click on the Results link for the first and click on the All Tests for the second form. In either case tcmIS will open a small child window to display the rule information.

The Results version displays only those rules which were actually applied in the approval/rejection decision. The All Tests version displays both the applied and inapplicable rules. In either case it will be necessary to click on the small plus sign in the square box to expand the details for each rule; in most cases there will be several levels of detail to be expanded.

Resubmit a Cat Add Request

When a request for a Catalog Add has been rejected it is possible to resubmit the request. Right-click on the line on the Catalog Add Tracking page and select the Resubmit menu item. tcmIS will open a new child window and display the Catalog Add request as a new item with a status of Pending Approval.

All the normal functions area available just as they are for any other request with a status of Pending Approval (see above). Make any changes and add any comments that would allow the approver(s) to approve rather than reject the request. When done, click the Submit link in the upper-left corner of the form.
