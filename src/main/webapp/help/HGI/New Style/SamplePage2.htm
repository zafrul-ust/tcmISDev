Catalog Lookup

The Catalog Lookup page is a Read-Only version of the Catalog that allows the user to list those materials specific to one Work Area and to print labels for them. The Catalog Lookup page can be accessed from the Order Management menu by selecting the Catalog Lookup menu item. The Catalog Lookup page is launched in a new tab.

Figure 1

To display the catalog items, or a portion of them, select the facility from the Facility box. You may further specify your lookup by selecting the work area from the Work Area box. If you want to narrow your search even more, type text for a match in the Search box. Click on the Search button.

Figure 2

tcmIS will display the item records for the work area you have selected, showing only those that also match any search criteria you entered. The data grid includes a Print column with a check box for each item.

Figure 3

To print a label for any item, check the box in the first column and then click the Print Labels link at the upper-left of the data grid. tcmIS will print the labels on any printer available to your workstation. You may select only one item or you may select more if you want.

Figure 4

Click the Create Excel button on the page to export the data in Excel format.
