Pass Through Report

The Pass Through Report page displays invoice details for counting inventory groups regarding pass through at cost materials. The Pass Through Report page can be accessed from the Reports menu by selecting the Cost Report menu item. Then select Pass Through Report. The Pass Through Report page is launched in a new tab.

Figure 1

Select the Hub and Inventory Group from the corresponding drop-down boxes. Enter a date in the Used Since box to report all pass through items used since the date entered. Click the blue diamond to select the date from the calendar.

Results may be sorted by

  • Use Date/Customer PO (Use Date and Customer PO)
  • Item ID/Invoice Date (Item ID and Invoice Date)
  • Item ID/Use Date (Item ID and Use Date)
  • Part/Invoice Date (Part Number and Invoice Date)
  • Part/Use Date (Part Number and Use Date)

Selecting the desired sort option from the Sort By drop-down list.

Click the Search button to display all pass through items used.

Click the Create Excel button to export the data in Excel format.
