Cabinet Inventory

The tcmIS Cabinet Inventory page is used to view information regarding on hand inventory. It can be accessed from either of two menu selections.

  • The Cabinet Inventory page can be accessed from the Supply Chain menu. Under Cabinet Inventory select the Cabinet Inventory menu item.
  • The Cabinet Inventory page can be accessed from the Reports menu. Under Inventory select the Cabinet Inventory menu item.

Using either menu selection, the Cabinet Inventory page is launched in a new tab.

Figures 1 and 2

The Work Area Inventory page allows you to tailor your search for inventory.

  • Select the hub which services the cabinet(s) from the Hub drop-down list.
  • Select the facility which services the work area from the Facility drop-down list.
  • Select the work area from the Work Area drop-down list.
  • The Cabinet selection list will be populated with all active cabinets for the selected work area(s). You may select a single cabinet, or select multiple cabinets by holding down the CTRL key while clicking each wanted cabinet, or you may select the default All [cabinets].

You may provide something to at least partially identify the material you want using the Search by boxes. (Your search may be specified by Part Number, Item Desc[ription], or Item ID). See Search functions for more information.

You may also tailor the search to locate inventory which is expiring within a specified number of days. Enter the minimum number of days in the Expires In text box and, if wanted, the maximum number of days in the To ___ days text box.

Select the check box labeled Show Only Quantity > 0 to see only work area items that currently have stock.

By default, the search results (inventory items) are sorted and displayed by Cabinet ID. You may choose instead to sort by Part Number or Item ID: select the desired sort option from the Sort By drop-down list.

Click the Search button. All active items in the selected cabinet(s) are shown. Details of each Receipt ID in the cabinet are displayed including manufacturer lot number, expiration date, last count date, last count quantity, quantity delivered since the last count, total quantity, unit cost, and stocking levels.

Figure 3

After the search is run, you may sort by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Figure 4

There are three columns toward the right end of the data grid that provide information about the inventory quantities in the Cabinet.

  • The Last Count Qty column displays the inventory recorded in the most recent scan.
  • The Delivered Since Last Count displays the quantity added subsequent to the most recent scan.
  • The Total Qty is the sum of the previous two columns and represents the current inventory (some of which may already be consumed and will be accounted for in the next scan.

Notice that the expiration date will only be shown on the Cabinet Inventory page when the Count Type is Receipt ID.

There is also a context menu available to view the inventory quantity in the HGI Hub. Right-click on a line and select the Show Inventory Detail menu item to display details on inventory of the item at the warehouse. The Inventory Details page will be displayed in a new window.

Figure 5

The Inventory Details page shows details on the inventory in the HGI Hub. The expiration date of HGI inventory is always shown on the Inventory details page.

This is particularly useful when the data in the cell does not fit the default size of the cell: it allows you to see the entire contents of the cell without having to adjust the width of the column.
