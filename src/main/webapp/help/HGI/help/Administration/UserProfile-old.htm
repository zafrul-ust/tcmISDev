User Profile

- Click to Print This Page -

The User Profile page is used to create new users and edit user information and certain permissions in tcmIS. The User Profile page can be accessed from the Administration menu by selecting User Profile. The User Profile page is displayed in a new tab.

You must have administrative permission to add or change user information or permissions in tcmIS. Contact the Technical Center to request administrative permissions.

Figure 1

Your user account is displayed by default. You may update any of the information displayed including first name, last name, middle initial, phone number, fax number, mobile number, email, and printer location. Click the Update Personal Info link to save any changes. You may also reset your password by clicking the Change Password link.

Figure 2

You may look up an existing tcmIS user and his or her personal information by clicking the magnifying glass icon . Enter the first name or last name of the user in the Search box and click Search. Locate the user in the resulting list and click anywhere on the line to select the user. Click the Return Selected User link to place that user's personal information on the User Profile page.

Figure 4

If you are a tcmIS administrator:

  • you may edit the user's personal information as you would your own.
  • you may also create new user accounts. Click the New button. A blank User Profile page is displayed in the existing tab.
    • Use the Company list to specify the organization for the new user account.
    • The following three fields are required:
      • Provide a unique Logon ID for the new user account in the Logon ID text box.
      • Enter a password in the Password text box.
      • Enter the password again in the Confirm Password text box.
    • Provide as much of the following information as you can:
      • Enter the user's first name, last name, email address, phone number, fax number, and cell phone number in the corresponding text boxes.
      • Select a printer location if this user works at a warehouse and needs to print labels.
        • If the needed printer location does not exist or you are unsure which to select, contact the Technical Center.

After all required personal information is completed, click the Create User link at the top left of the page. Click the Cancel link to abandon creation of this new user.

Figure 5

Startup Pages button

Click the Startup Pages button on the User Profile page to view and edit the list of pages that launch automatically when tcmIS is started for the selected user. The Startup Pages page will be displayed in a new window.

To add a page to the user's startup display check the box in the Startup column for the page. To remove a page from the user's startup display, uncheck the box for the page. As an option, you may specify the order of the pages (from left to right) by entering numbers, starting with 1, in the Startup Order column.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close the page without saving changes, or after saving the changes you want.

Figure 6

Companies button

View/Edit the list of companies that this user can access, their company Admin privileges, and whether the user can create new users for each of these companies.

Click the Companies button in the lower panel to open the Companies page in a new window.

Check the box in the Access column for each company that the user needs to view. If the account is an administrator for any company, check the box for the company in the Admin column. You must have Grant Admin permission for that company to give others administrative permissions for that company.

In some cases you may wish to keep a user account active but eliminate that account's access to a particular Company. In that case, check the box in the Locked column for that company

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close this screen without saving changes.

Pages button

Click the Pages button in the lower panel to open the Pages page in a new window.

For each page that the user needs to view in tcmIS check the box in the Access column. For each page where the user account needs administrative rights check the box in the Admin column . Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this screen without saving changes.

Figure 8

Facilities button

View/Edit the list of facilities that this user can access corresponding to their companies, their facility Admin privileges, and their operational permissions.

Click the Facilities button on the User Profile page to view and edit the list of functional capabilities available to the user account each of the Facilities where they have permissions. Functional capabilities include administrative privileges and operational permissions and are granted (or denied) separately for each Facility.

Select the specific company on which you wish to manage facilities permissions from the Company drop-down list.

Select a facility from the Facility list, or display all facilities with the My Facilities item, to view functional capabilities by facility. Uncheck the box labeled Show current permission only to display all customer facilities for the selected customer. Click the Search button. The Facilities Permissions page is displayed in a new window. (The illustration, below, is split into left and right sections to fit most browsers.)

For each selected Facility (in rows) the columns name the functional capabilities. A checked box indicates the user account has that functional capability for that facility. An unchecked box means the user account does not have the functional capability.

For each facility, check the box

  • in the Administrator column for which the user should be a facility administrator in tcmIS.
  • in the Cost Report column for which the user should be able to view the Cost Report page.
  • in the Create New Chemical column for which the user should be allowed to enter New Chemical Requests in tcmIS.
  • in the List Approval Admin column for which the user should be a List Approver in tcmIS. (The List Approval function must be set up and turned on separately. Contact the Technical Center for assistance.)
  • in the Releaser column for which the user should be a releaser in tcmIS. (The Releaser function must be set up and turned on separately. Contact the Technical Center for assistance.)
  • in the Waste Manager column for which the user should be a facility waste manager in tcmIS.
  • in the Safety Label column for which the user should be able to print safety labels in tcmIS.
  • in the WA Management column for which the user should be a Managed Work Area Approver in tcmIS. (The Managed Work Area function must be set up and turned on separately. Contact the Technical Center for assistance.)
  • etc., as appropriate.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this screen without saving changes.

Figure 10

Work Areas button

View/Edit the list of work areas that this user can access, whether they can create MRs for these work areas, and if they are an over-limit approver for these work areas.

Click the Work Areas button on the User Profile page to view and edit the list of work areas that the user can access as well as whether the user can create Material Requests and do Scans for these work areas. Select a specific facility from the Facility list, or display work areas for all facilities with the My Facilities item, to view work areas by facility and grant or edit access.

Select the specific company on which you wish to manage work areas permissions from the Company drop-down list.

By default, permissions are granted or denied for All Work Areas in a facility and you will likely only see one row for each facility marked 'All Work Areas'. Uncheck Show current permission only and the individual work areas will also be displayed in the data grid though with their check boxes clear. Select the All Work Areas check box in any column to activate the individual work area check boxes for that column.

Click the Search button. The Work Areas page will be displayed in a new window.

  • Check the box in the Access column for each work area the user needs to view in tcmIS. Uncheck those which the user does need to view in tcmIS.
  • Check the box in the Display Application column for all facilities and work areas for which the user can enter Material Requests in tcmIS.
  • Check the box in the Create MR column for all facilities and work areas for which the user can enter Material Requests in tcmIS.
  • Check the box in the Scan column for all facilities and work areas for which the user has scanning responsibility.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this screen without saving changes.

Docks button

View/Edit the list of docks (ship to addresses) that this user can access for Drop Ship Receiving and whether they have receiving privileges for those docks.

Click the Docks button on the User Profile page to view and edit the docks (ship to addresses) permissions for the user. Check the box in the Dropship Receiving column for each facility where the user should be allowed to view open orders and perform dropship receiving. Uncheck the box for each facility where the user account is not supposed to have that permission. Where a user may not be granted privilege the box will be inactive.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this screen without saving changes.

Financial Approver button

View/Edit the list of Financial Approver that this user is a member.

Click the Financial Approver button on the User Profile page to view and edit the list of Financial Approvers of which the user is a member. Select a facility from the Facility list to view financial approval lists for the facility. Click the Search button. The Financial Approver page will be displayed in a new window.

Select the specific company on which you wish to manage financial approver permissions from the Company drop-down list.

Any user who can create Material Requests for facilities that use the tcmIS financial approval function must be added to the financial approval list. The financial approval feature can be turned on or off for a facility by the Technical Center.

  • To add a user to the financial approver list click the Add link. Identify the new user on the Financial Approver detail form that is opened. Search for the user by clicking the magnifying glass icon . Enter the Users Order Limit in the displayed currency. If the user is also a financial approver, enter the User Approval Limit in the displayed currency. Select the designated approver from the Next Level box. Click the OK button to add the user to the financial approver list.
  • To edit a user on the financial approver list select the name on the Approver list, then click the Edit link. Make any needed change to the Users Order Limit, the User Approval Limit, and the designated approver from the Next Level box. Click the OK button to confirm the changes to the user record.
  • To remove a user from the financial approver list select the name on the Approver list and click the Delete link.

Order User Group button

User Group Permissions

User groups are each specific to a single facility but may have members from other facilities in some situations. The same User Group name (e.g., Acids) can therefore be used at more than one facility and even have members in common. For example, Tom Jones might be a member of the Acids user group at both Facility A and Facility B. User Groups can be used to limit who can order specific categories of material.

To manage or view the groups to which a user belongs, click the User Group button on the User Profile page. Select a specific company from the Company drop-down list and the select a specific facility from the Facility drop-down list. Select This User's Access to view only groups for the currently selected user account. Select All Users' Access to view all active user groups for the selected facility. Click the Search button.

  • If you selected This User's Access tcmIS will display a list of all the groups for the facility. Those groups of which the current user is a member will have the box checked in the Member column. The current user may be removed from a group by unchecking the box. The current user may be added to a group by checking the box (if it is not 'active' the account may not be added.

Click the Create Excel button to export the displayed data in Excel format.

Figure 17
  • If you selected All Users' Access tcmIS will display all the requested user groups. Click on the plus sign (+) icon for any group to display the user accounts assigned to that group. Click on a minus sign (-) icon to close the display of user accounts for a user group.

To add the currently selected user account to a group, select the user group on the User Group Permissions page and click the Add User to Group link. The user is added to the user group.

To remove a user from a user group, first display all members of the group by clicking the plus sign (+) icon in front of the user group name. Select by clicking the user account to be removed and click the Delete User From Group link.

Click Cancel to close this screen without saving changes.

Report User Group button

View/Edit list of Publish Report User Group and it members

The Report User Group is established to manage those who can create, edit, or run reports. Click the Report User Group button on the User Profile page to view and edit the list of Report User Groups.

Select This User's Access to view only groups for the currently selected user account. Select All Users' Access to view all active user groups. Click the Search button.

  • If you selected This User's Access tcmIS will display a list of all the groups.
    • Those groups to which the current user has access will have the box checked in the Access column.
    • Those groups of which the current user is an administrator will have the box checked in the Admin column.
    • Those groups to which the current user may publish templates will have the box checked in the Publish column.
  • If you are an Administrator, the current user's permissions may be changed by checking or unchecking the box in the appropriate column.
  • The current user may be added to a group by checking the box (if a box is not 'active' the account may not be added.)
  • You must click the Update link to save any changes.

Click the Create Excel button to export the displayed data in Excel format.

Figure 20
  • If you selected All Users' Access tcmIS will display all the report user groups. Click on the plus sign (+) icon for any group to display the user accounts assigned to that group. Click on a minus sign (-) icon to close the display of user accounts for a user group.

To add the currently selected user account to a group, select the user group on the User Group Permissions page and click the Add User to Group link. The user is added to the user group.

To remove a user from a user group, first display all members of the group by clicking the plus sign (+) icon in front of the user group name. Select by clicking the user account to be removed and click the Delete User From Group link.

To remove a group altogether, select the group and click the Delete Group link. This is irreversible. If you delete a group in error it will have to be completely recreated.

Click the Cancel button to close this screen.
