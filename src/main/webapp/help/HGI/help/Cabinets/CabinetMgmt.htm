Cabinet Management

The Cabinet Management page is used in tcmIS to activate and inactivate parts in Cabinets, set stocking levels, and add parts to Cabinets. The Cabinet Management page can be accessed from the Supply Chain menu: choose Cabinet Inventory and select the Cabinet Management menu item. The Cabinet Management page is launched in a new tab.

Figure 1

To view cabinet inventory records,

  • Select the hub from the Hub drop-down list.
  • Select the facility from the Facility drop-down list.
  • Select the work area from the Work Area drop-down list or select the default of All.
  • Select the specific cabinet(s) from the Cabinet drop-down list or select the default of All. To select more than one cabinet, hold down the keyboard CTRL key while clicking those wanted.

You may display all cabinet inventory of a customer or you may narrow the search by providing additional description of the item(s) you want using the Search by boxes:

  • In the first box select the basis for the search: Part Number, or Item ID.
  • In the second box choose the type of search:
    • contains, for a partial match;
    • is, for an exact match. Keep in mind that an exact match is exactly that, so typos must particularly be avoided with is.
  • In the third box enter the number you want fully or partially to match in the search.

Click the Search button.

There are two management functions you can perform on the Cabinet Management page directly.

  • Clear the check box in the Active column to de-activate an item. Select the check box to activate an item.
  • The Count Type column identifies which base is used for inventory replenishment for an item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to recognize expired material and beep loudly to notify the operator to remove the expired material.

Click the Update link above the data grid to save any changes.

Click the Create Excel button to export the data in Excel format.

Beginning on the Cabinet Management page you may create a new bin record. Right-click on any of the first three columns (Work Area, Cabinet, or Bin) to open a context menu with a Create New Bin menu item. Clicking it will cause tcmIS to open a child window to display the Cabinet Bin form.

When a search returns no records, along with the No Data, a Create New Bin link is available that opens the same form.

Since you have begun from a search for a specific Cabinet, the Cabinet Bin form starts with this information. To actually create the bin, provide the additional information on the form.

  • Select the appropriate Catalog from the Catalog dro-down list.
  • Provide a name, following any company guidelines or naming conventions, in the Bin Name text box.
  • Enter the correct Part Number in the Part Numbertext box.
  • Enter values in the Reorder Point text box, the Stocking Level text box, and the Lead Time in Days text box.
  • The Count Type column identifies which base is used for inventory replenishment for an item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to recognize expired material and beep loudly to notify the operator to remove the expired material.

Click the Add button to save the information and create the new bin record. Click the Cancel button to discard the record before it is finalized.

Figure 5

Also beginning on the Cabinet Management page you may open a form to set or change bin levels. Right-click on any of the next six columns (Active, Catalog, Part Number, RP/SL, Lead Time In Days, or Item to open a context menu with a Set Bin Level menu item. Clicking it will cause tcmIS to open a child window to display the Bin Level form.

Figure 6: Add Part and Generate Labels menu items

On the Bin Level form you may enter or change values in the Reorder Point text box, the Stocking Level text box, and the Lead Time in Days text box. You may also enter information into the Remarks text box to explain the numbers.

Click the Update link above the data grid to save any changes.

Any previous changes, including Remarks, will be displayed in the Change Histor data grid below the Item data grid.
