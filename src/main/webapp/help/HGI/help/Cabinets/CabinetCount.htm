Cabinet Count

If manual entry of Cabinet inventory is necessary, use the Cabinet Count page. The Cabinet Count page can be accessed from the Supply Chain menu. Under Cabinet Inventory, select Cabinet Count. The Cabinet Count page is launched in a new tab.

Figure 1

To begin the Cabinet Count process, identify the cabinet(s) on which you want to work.

  • Select the hub from the Hub drop-down list.
  • Select the facility from the Facility drop-down list.
  • Select the work area from the Work Area drop-down list or select the default of All.
  • Select the specific cabinet(s) from the Cabinet drop-down list or select the default of All. To select more than one cabinet, hold down the keyboard CTRL key while clicking those wanted.

Click the Search button to cause the requested Cabinet information to be shown in the data grid.

Enter or edit the quantity in the Cabinet for each item as needed. Click the Update button to save the numbers.

Material Requests for inventory replenishment will be created based on the information you have provided but the Cabinet Inventory page will not be updated until the next Cabinet scan is uploaded to tcmIS.

Click the Create Excel button to export the results of the report in Excel format. This file can be saved and printed to assist in manually doing an inventory of cabinets.
