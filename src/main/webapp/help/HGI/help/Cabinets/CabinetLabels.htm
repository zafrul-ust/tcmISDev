Cabinet Labels

The Cabinet Labels page is used to print labels for stock management. The Cabinet Labels page can be accessed from the Supply Chain menu. From the Cabinet Inventory menu item select Cabinet Labels. The Cabinet Labels page is launched in a new tab.

These labels should be printed and applied in the proper Cabinet locations before the first cabinet scan.

Figure 1

To begin the Cabinet Label process, identify the labels you want.

  • Select the hub from the Hub drop-down list.
  • Select the facility from the Facility drop-down list.
  • Select the work area from the Work Area drop-down list or select the default of All Work Areas.

You may choose the order in which the records are displayed using the Sort By drop-down list to select Work Area (the default), Cabinet ID, or Cabinet Desc[ription].

Click the Search button.

Figure 2

tcmIS will display those cabinets that match your search criteria, in the sort order you selected.

Using the option buttons immediately below the Search button specify whether you want

  • labels printed on 3" x 1" label stock (using the required Zebra printer), or
  • labels on 8-1/2" x 11" paper (tcmIS will create a PDF file which you may print on a printer accessible from your workstation).

Select the box in the column labeled Print for each cabinet where labels are to be produced.

  • Click the Generate Base Line button to print the Cabinet Baseline report (Landscape orientation for printing on 8.5 x 11-inch paper). The baseline report should be posted in each Cabinet.
  • Click the Generate Cabinet Labels link to print the Cabinet label. This label is scanned at the beginning of each Cabinet scan.
  • Click the Generate Cabinet & Bin Labels link to print both the Cabinet labels and all the associated bin labels. (See, also, bin labels, below.)
  • Click the Generate All Labels link to print whatever labels are available for the cabinet.
Figure 3

tcmIS will display a notification window while generating the base line report. When the report is run tcmIS will open another window to display the report for printing.
