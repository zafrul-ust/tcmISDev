Cost Report

The Cost Report page can be used to create reports regarding invoicing of chemicals, waste, gasses, and service fees. The Cost Report page is basically a view into HGI customer invoices or customer cost. The Cost Report page can be accessed from the Reports menu by selecting Cost Report. Then selectCost Report. The Cost Report page is launched in a new tab.

From the perspective of HGI, this report is truly a revenue report.

Identifying the report criteria

  • Accept the default Company or choose another (if desired and available) from the Company drop-down list.
  • Choose a Reporting Group from the drop-down list or leave it set at the default My Reporting Groups. This choice will affect subsequent options in this section, including the Accounting System, and will add a text field to allow additional accounting information.
  • Choose a Facility from the drop-down list or leave it set at the default My Facilities. This choice will affect subsequent options in this section.
  • Choose a Work Area from the drop-down list or leave it set at the default My Work Groups.
  • BAE Systems facilities each use only one Accounting System so for a single facility the default here is the correct and only choice. Otherwise leave it set to the default My Accounting Systems or select the one needed for the report.
  • Once the accounting system is identified, a text box becomes available to include a Charge # or a PO to the report criteria.
Figure 2
  • You may click the magnifying glass icon next to the Requester text box. The Requestor form will open in a child window. Use that to identify the person whose requests are to be reported. All reports run will display only cost information about materials requested by the specified individual. Delete the requester's name to remove this search criterion.
  • The Unit of Measure (UOM) in which the report is displayed may either be the Invoice's UOM or the Item's UOM. (Some customers are billed in unit of measures which are different from the HGI Item unit of measure.) You may select the preferred reporting Unit Of Measure from the drop-down list.
  • You may locate cost information associated with specific manufacturers, parts, etc. using the three Search by boxes, using one of these fields:
    • Mfg [manufacturer], enter manufacturer information in the third text box.
    • Part Number, enter customer part number information in the third text box.
    • Part Description, enter customer part description information in the third text box.
    • Item, enter HGI/tcmIS item ID information in the third text box.
    • Item Description, enter HGI/tcmIS part description information in the third text box.
    • Item Type use a 2-letter code in the third text box.
    • Reference, enter tcmIS reference information in the third text box.
    • MR-Line, enter tcmIS material request information in the third text box.
    • Haas PO, enter HGI purchase order information in the third text box.

See Search functions for more information.

  • To locate invoice details for a specific HGI invoice, enter the invoice number in the text box labeled Invoice #.
  • To locate records associated with a specific invoice period, enter the period number in the Invoice Period text box. Invoice Period is an internal HGI accounting detail that almost certainly will not correspond to any particular customer's invoice period.
  • To locate cost information related to materials invoiced during a specific time period, use the two fields labeled Invoice Between...and. Each time you place your mouse cursor in one of these fields, the Date Selector form is displayed. You must choose a date for each field, the beginning and ending dates you want.
    • For a specific date: enter the same date in both fields.
    • For a date range: enter the earliest date in the left field and the latest date in the right field.
    • Choose each date by clicking the date on the calendar. The dates selected will appear on the Cost Report page.
  • Cost information about materials delivered during a certain time period can be found in a similar manner as above but using the two Delivered between...and fields. Select your date or date range using the Date Selector form and they will be placed on the Cost Report page.
  • Materials can be selected for cost reporting purposes by invoice type. In the Invoice Type list keep the default All Types or select one or more of Add Charge, Chemical, Gainshare, Gas, Service, or Waste to limit and arrange the report by these categories. To report on multiple invoice types hold down the CTRL key while selecting the desired invoice types.
  • Materials can also be selected for cost reporting purposes by the material Item Type. Keep the default All Types or select the type(s) descriptions from the Item Type multi-item selection list. To select multiple item types hold down the CTRL key whole selecting.

Identifying the report fields

Once all search criteria are set, choose the report fields to include in the cost report by selecting the check boxes across the bottom half of the page. Each box selects the named field to be placed on the report.
