SQL Report

The SQL Report page is used to run stored SQL database queries against the tcmIS database. It is not necessary to have strong SQL skills in order to run reports on this page. The SQL Report page is accessed from the Reports menu by selecting the SQL Report menu item. The page is launched in a new tab.

Figure 1

Enter a properly formed SQL query in the Query multiline text box. Either select one of the queries in the Saved Queries list box or type it directly. (See SQL Help.) In either case you may edit the query if needed, for instance to change the date range.

  • Click the Run Query button to execute the query. tcmIS will run the query and produce an Excel file with the results.
  • You may save a query for future use. Entering a name in the Query Name box and click the Save Query button.
  • You may delete a query from the Saved Queries list. Select the query and click the Delete saved query button.
Figure 2

Help is available. Click on the Help link. tcmIS will open a child window to display very basic information on proper SQL syntax.

Figure 3

Creating or editing a SQL query requires that you use the exact table and field names for the database. To view the tables available scroll through the Table list box. Select a table and tcmIS will display the fields ('columns') for that table in the lower right corner of the page.
