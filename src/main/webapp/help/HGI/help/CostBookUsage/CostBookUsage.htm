Cost Book Usage

The Cost Book Usage page displays usage details for each material over the program year for counting inventory groups. The Cost Book Usage page can be accessed from the Reports menu by selecting the Cost Report menu item . Then select Cost Book Usage. The Cost Book Usage page is launched in a new tab.

Figure 1

Select the Company, Facility, Inventory Group, and Report Start Date from the corresponding drop-down lists. Click the Search button. tcmIS will generate the report.

Click the Create Excel button to export the report in Excel format.
