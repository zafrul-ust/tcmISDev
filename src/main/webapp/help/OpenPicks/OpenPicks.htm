Open Picks

+ Larger Font | + Smaller Font

The Open Picks page provides information regarding Sales Orders which are pending Picking QC. The Open Picks page can be accessed from the Reports menu by selecting Orders and then Open Picks. The Open Picks page is launched in a new tab.

Figure 1

Select the Operating Entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drops-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

Click Search to see all Sales Orders pending Picking QC.

Figure 2

For each Sales Order. the quantity, value, and number of days pending Picking QC is displayed. You may sort by any column by clicking the column header. Click the column header again to sort in descending order.

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 3

Click the Create Excel File button to export the results into Excel format.
