Quick Item View

+ Larger Font | + Smaller Font

The Quick Item View page is a convenient way to get a complete overview of Distribution items in HGI locations, including HGI owned material, client owned material, and consigned material. The Quick Item View can be accessed in two ways:

  • From the Distribution menu by selecting the Quick Item View menu item. It will be opened in a new tab.
  • From the context menu on any record row shown on a Scratch Pad or a Material request select the Quick Item View menu item. It will be displayed in a child window.

Figure 1A

In the upper panel of the Quick Item View tab page there are there are three criteria and two options for specifying the item for which you want details. (On the child window form since you have started from a specific customer and item all these choices have already been accounted for: the upper panel displays only the Inventory Group and Item Desc[ription] information.)

  • Select the operating entity from the Operating Entity drop-down list.
  • Select the HGI hub from the Hub drop-down list.
  • Select the inventory group from the Inventory Group drop-down list.
  • You must specify a particular part for the search.
    • If you know the exact part number you may use that in the Item text box.
    • Otherwise, click on the magnifying glass icon to use the Item Search form in a child window and locate the item. See below and see tcmIS Search/Seach by for more details.
  • You may use the Customer information to limit the search to details for a single customer:
    • begin typing the name and tcmIS will display a drop-down list of matching customers (see Type ahead search); or
    • click on the magnifying glass icon to use the Customer Lookup form. See Lookup forms for general information and see Quick Customer View for more details.
    • Click on the adjacent Clear button to remove that selection from the search. If no Customer is specified, the search will look for all Customers.

Click the Search button to display all the items that match the criteria.

Figure 2

Unlike most pages in tcmIS, the Quick Customer View page displays four data grids. Click on any of these data grid titles to see more information:

  • MR History (upper left)
  • PO History (upper right)
  • Price Quote History (lower left)
  • Inventory (lower right)

Figure 3

Item Search form

The Item Search (lookup) form includes six different sets of search fields; each be used alone or in combination with one or more of the others. When combinations are used search results will display only those items that meet all criteria.

Each of the six set of searches allows matching on Contains, Is, Starts with, and Ends with. (See tcmIS Search/Seach by for more details.) The fields are

  • Part Number
  • Part Description
  • Specification
  • Customer Part Number
  • Text
  • Synonym

MR History data grid

In the MR History panel use any of the text boxes in the column headers to filter the display and show only those inventory items matching the text you enter. The match will be made when any or all of what you enter matches the data in that column, including when you enter text in more than one column .

  • Inventory Group
  • MR-Line
  • Customer Part
  • Desc[ription]
  • Customer PO

Figure 5

Right click on any record in the MR History panel for a context menu that includes four menu items.

  • The Open menu item will display the MR for the record in a new tab. See the Create MR page help for more details.
  • The MR Line Allocation menu item opens a child window to display the MR Allocation report for only the selected record. See help for the MR Allocation Report for details.
  • The Create New Quote menu item display a new Quote page in its own tab, with much of the information already filled in based on the existing quote. See the Quote page for details.
  • The Create New MR menu item will display a new MR in its own tab, based on the existing MR. See the Create MR page help for more details.

Figure 6

Price Quote History data grid

In the Price Quote History panel use the text box in the Inventory Group column to filter the display to display only those inventory items matching the text you enter.

Right click on any record in the Price Quote History panel for a context menu that includes three menu items.

  • The Open menu item will display the Quote for the record in a new tab. See the Quote page help for more details.
  • The Create New Quote menu item display a new Quote page in its own tab, with much of the information already filled in based on the existing quote. See the Quote page for details.
  • The Create New MR menu item will display a new MR in its own tab, based on the existing MR. See the Create MR page help for more details.

Figure 7

PO History data grid

In the PO History panel use the text box in the Supplier column to filter the display to show only those suppliers matching the text you enter.

Also in the PO History panel, there is a Show Current Sourcing Information link. Click on it to display item sourcing information in a child window.

Right click on any record in the PO History panel for a context menu that includes one menu item.

  • The Open menu item will display the Purchase Order for the record in a new tab. See the Purchase Orders page help for more details.

Inventory data grid

In the Inventory panel use the text box in the Inventory Group column to filter the display to show only those inventory groups matching the text you enter.

Right click on any record in the Inventory panel for a context menu that includes one menu item.

  • The Show Details menu item will open a child window to displays the Item Activity for InvGrp: [inventory group] for the record. See the Item Lookup page help for more details.
