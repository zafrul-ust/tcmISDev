Installation

TcmIS requires that you make Internet Explorer you default browser before installation:

  • From your desktop, click the Start button, and choose Programs. Select Internet Explorer from the Programs menu.
  • From the Tools menu, choose Internet Options.
  • Select the Programs tab at top of the screen.
  • At the bottom of the Programs tab, click the check box labeled Internet Explorer should check to see whether it is the default. Click Apply, and then click OK. (If Apply is inactive, just click OK.)
  • Close Internet Explorer and any other programs that are running.
  • Restart Internet Explorer. A message will appear asking if you want to make Internet Explorer your default browser. Click Yes. Also select Do Not Ask Again. (If you do not receive this message then Internet Explorer was already established as your default browser.)

You are now ready to begin the installation of tcmIS:

  • Open Internet Explorer and enter the URL for your company: (If typing the address, type as shown, with capital letters in address capitalized.)
    • First Energy : https://www.tcmis.com/tcmIS/fec/Register
  • When the page opens:
    • click . If you receive a Security Alert message, select In the Future Do Not show this warning.
    • From the File Download window, click Open.
    • Accept the license agreement by clicking Yes.
    • Select Typical when prompted to choose a setup type. Then click Next.
  • Using Internet Explorer, return to the URL listed above for your company.
    • Add this page to your Favorites. In Internet Explorer, from the Favorites menu, choose Add to Favorites.
    • Enter your username and password.
    • Java Web Start will check for the latest version. Once complete you will see a Java Web Start Security Warning: "This application is requesting unrestricted access to your local machine and network". Click Start.
  • Installation is complete and tcmIS will be launched.
  • To launch the new version of tcmIS in the future, open Internet Explorer and select tcmIS from your list of Favorites.

Locating Material Safety Data Sheets

The MSDS Viewer page is used to locate and display a Material Safety Data Sheet (MSDS). In addition to viewing the MSDS, this page can be used to view information about a material's chemical composition, physical properties, and any chemical lists on which the material appears. The MSDS Viewer page can be found at: https://www.tcmis.com/tcmIS/fec/Msds.

Figure 1

The MSDS Viewer page can locate Material Safety Data Sheets for materials in the First Energy catalog as well as for materials that are not in the First Energy catalog but may be used on site. Select the check box labeled Catalog to search for MSD Sheets that are linked to parts in the catalog. Select the Non-Catalog check box to search for MSD Sheets that have been sent to Haas TCM to be added to the MSDS Library, but not to the First Energy catalog.

Select Full Database to search for MSD Sheetss for materials on file for any Haas TCM client. Searching the full database can be an effective research tool when looking for a new material or substitute materials.

To locate an MSDS, enter a trade name, manufacturer, or part number in the Search box. Use the key words AND, OR, BUT NOT between search words to build more complex queries. For example, "Florida AND oranges BUT NOT navel oranges" would return all Florida oranges that are not navel oranges. Sort the search results by any of the columns displayed by choosing a selection from the Sort drop-down list. Click Search.

Chemicals and Cylinder Gasses

Welcome to Total Chemical Management Information System (tcmIS). tcmIS is a web-based chemical management information system which is used to manage requests for chemicals. tcmIS facilitates inventory tracking and adding new chemicals to the catalog. After logging on by entering a valid logon ID and password, the Chemical screen is displayed.

To access the Chemical and Gas Catalog from elsewhere within tcmIS, click the Chemical button on the toolbar as shown below in Figure 10.

Figure 10

The Chemical and Gas Catalog can also be accessed by selecting Chemical Catalog from the Application menu as shown in Figure 11.

Figure 11

Chemical and Gas Catalog

The Chemical and Gas Catalog contains information on materials approved for use in the specified facility and work area. There are currently two types of catalogs. The Part Catalog identifies all materials and their requirements that have been defined for the specified facility and/or work area as well as any ordering restrictions. The Item Catalog is more general and represents all chemicals defined by manufacturers that have been ordered by any Haas TCM customer.

Figure 12

Part Catalog

All materials that have been ordered or are allowed to be ordered are stored in the Part Catalog. Information about materials can be entered in the Search box. Search criteria can include customer part number, Haas TCM part number, packaging, trade name, manufacturer, manufacturer's part number, or material description. The more specific the information entered, the more the search is narrowed. Multiple words as a logical expression can also be used to refine a search (separate words by AND, OR, NOT; for example, acetone AND cleanroom). To search for all chemicals for a specific location, leave the Search box blank and click the Search button.

Some additional constraints can be added to a search of the Part Catalog. Selecting the Active for selected Facility/Work Area option displays only those chemicals associated with the selected facility and work area. Selecting the Approved Only check box limits the search to only those chemicals approved for the selected facility and work area.

Selecting the All Catalogs option displays chemicals for all catalogs associated with your company. This feature can be useful when searching for a new material. Selecting the Active Only check box limits the search to chemicals that are currently active. A chemical is inactive if it becomes obsolete for the part number, is no longer manufactured, or is not yet approved for the part number.

The Part Catalog is divided into three sections. The columns in blue and white represent the part details and include catalog, customer part number, part description, stocking type (MM or OOR), shelf life, and price. The first two columns in green and white represent the item details pertaining to the part. The rest of the columns give component level details such as packaging, manufacturer, and approval status.

Figure 13

More information can be obtained for a specific chemical by clicking anywhere on the line so the line is highlighted, then right-clicking on the part details to display a menu of additional selections. These selections include information regarding the part inventory, item inventory, reorder point, stocking level, specifications, approved work areas, and work area comments.

Figure 14

To find out more about the inventory of a particular item, select the item and right-click the item details. Select Item Inventory. Typically only Min/Max items will have information in the Item Inventory screen. On hand inventory is displayed on the left side of the Item Inventory screen. Items that are on order are displayed on the right side of the Item Inventory screen.

Figure 15

If there is a specification associated with a part, it can be viewed by right-clicking on the part and selecting Specification. If there are multiple specifications associated with a part, they will all be displayed. Highlight and left-click the specification you would like to view. It will be displayed in Adobe Acrobat Reader.

Figure 16

Comments can be added to a particular part by right-clicking on the part details and selecting Edit Comment.

Figure 17

Similarly, more information about the item can be displayed by right-clicking on the Component Description. You can view the Material Safety Data Sheet (MSDS) of a particular component by right-clicking on the component as shown below and selecting View MSDS.

Figure 18

Item Catalog

If a chemical is not found in the Part Catalog, the Item Catalog can be searched by selecting Item Catalog from the Catalog Type drop-down box. The Item Catalog represents all chemicals that have ever been defined within tcmIS including items that are currently defined for other clients. The item number, component description, packaging, grade, manufacturer, manufacturer part number, and shelf life are displayed for each item in the Item Catalog. Client specific information is not displayed.

Searches of the Item Catalog can be performed in a similar manner to searches of the Part Catalog.

Catalog Additions

If a chemical does not exist in the catalog or it is not approved for use in the specified work area, the catalog addition features of tcmIS must be used to add the item or get approval to use the material. To begin this process, click the Catalog Add tab next to the shopping cart on the Chemical Catalog screen. Four buttons appear in the Catalog Add tab to facilitate the request.

Figure 39

The first step in adding a new chemical to the catalog is to search for it in the Part Catalog. The desired material may already be in the catalog, but not approved for the specified facility and work area. Select the Active for selected Facility/Work Area option. Deselect the Work Area Approved Only check box. Enter a description of the part in the Search box and click Search.

Figure 40

If the desired part is found in the Part Catalog, but it is not approved for the specified facility and work area, select the part by left-clicking it. Then, click the New Work Area Approval button.

If the manufactured item is found in the Part Catalog, but it is defined under a part number with different specifications then those required for the specified work area, select the part and click the New Catalog Part button.

If the desired material is still not found in any of the searches above, search the Item Catalog by selecting Item Catalog from the Catalog Type drop-down box.

Figure 41

If the manufactured item is found in the Item Catalog, but in a different size than required, select the item and click the New Size/Packaging button.

If the manufactured item is found in the Item Catalog in the desired size and packaging, select the item and click the New Catalog Part button.

Finally, if there is no part defined in the Part Catalog or Item Catalog that is similar to the desired material, return to the part catalog and click the New Material button.

By selecting the appropriate starting point, tcmIS will advance to the Cat Add Detail screen, inheriting as much known information as possible regarding the request. For example, Figure 42 shows the initial section of the new catalog add form with information inherited from the New Catalog Part selection. Fields marked with asterisks must be completed in order to submit the catalog addition request.

Figure 42

The Cat Add Detail screen is composed of three sections. All sections must be completed in order to submit the catalog addition request. Section 1 contains information about the manufactured item. Section 1 is divided into three parts: new material information, new size/packing information, and suggested supplier information.

The new material information includes basic information about the manufacturer, material description, grade, manufacturer's trade name, and MSDS. The button can be used to look up manufacturers by manufacturer ID or name. The material description will appear in the catalog as the component description, so it should be a clear and easily understandable description of the product. The manufacturer's trade name should be taken directly from the MSDS. If there is not already an MSDS on file for the part or there is a newer revision of the MSDS than Haas TCM currently has on file, a copy of the new MSDS must be sent to Haas TCM. The MSDS on file can be viewed by clicking the MSDS button.

The new size/packing information includes the manufacturer's part number, the number per part, size, unit, and package style. Depending on what unit is chosen, the dimension and net weight may also be needed. Select the Sample Sizing check box if the new part is in a sample size.

The suggested supplier information should be completed if there is a specific supplier that should be used. This information includes supplier name, contact name, e-mail, phone, and fax.

Some materials come in kits and have multiple parts. Additional parts can be added to the request by right-clicking the Part 1 tab and selecting Add a new Part. A part can be deleted from a request by selecting Delete Part.

Figure 43

Once all required information is completed in Section 1, advance to the next section by clicking the Section 2 of 3 button on the left side of the Cat Add Detail screen. Information in Section 1 will automatically be saved.

Section 2 contains information about the item pertaining to the facility, such as the internal part number and quality requirements.

The catalog that the material needs to be added to should be specified in the Catalog drop-down list. The user part number and part description may be entered in this section. The Part Description will be displayed as the FEC Part Description in the catalog. If this part replaces an old part, the old part number can be entered in the Replaces Part Number text box.

The order type must be specified by selecting either the OOR or MM option. OOR stands for Order on Request. These are materials that are used infrequently and are not maintained in inventory. OORs must be ordered each time the material is needed. MM stands for Min/Max. MM orders are materials used routinely and maintained in inventory by Haas TCM. The estimated quarterly usage rate must also be completed.

Figure 44

The required shelf life and storage temperature of the material must also be specified. If the shelf life and storage temperature recommended by the manufacturer is acceptable, select the Manufacturer's Recommendation option. If a custom shelf life and storage temperature need to be specified, select the User Override option. Choose either Indefinite or days from the Shelf life drop-down list. Select a storage temperature from the at Storage temp drop-down list.

Choosing days from the Shelf life drop-down list displays a text box for entering the number of days. Select either the Date of Receipt or Date of Mfg. option to specify when the shelf life of the material begins.

Figure 45

Quality and packaging flowdowns can be added to the material by clicking the Search button in the Flowdown area. The Flow Down Search screen is displayed. To search for a flowdown, select either Quality or Packaging from the Type drop-down list. Flowdowns can be searched by description or name, depending on which option is selected. To add a flowdown to the catalog addition request, select the flowdown by clicking it, then click the Apply button. This can be repeated for as many flowdowns as desired. Click OK when all flowdowns have been added. To delete a flowdown, select it by clicking it in the flowdown area, then click Delete.

Figure 46

If the new material has a specification, select the Spec'ed Material option in the Specification area. Enter the spec ID, revision number, amendment number, and title. If there is already a spec associated with this material, it can be viewed by clicking the View button in the Specification area.

Figure 47

Once all required information is completed in Section 2, advance to the next section by clicking the Section 3 of 3 button. Information in Section 2 will automatically be saved.

Section 3 describes information on the use and fate of the item by one or more work areas at the facility. As information is entered for a work area, selecting the Add/Modify button will store the information for the work area in the Summary section. For each work area, a description of the use of the material and process description must be entered in the Use and Process description box.

Materials may be pre-approved for specific user groups or for all users by completing the pre-approval information. Pre-approving a material will decrease the number of orders sent to a chemical approver. To pre-approve a material select the user group from the User Group drop-down list. Specify how many units are approved for a specific time period in the Qty __ per __ days boxes.

The fate of the material can also be entered in the Fate table. Enter a percentage for each fate description. The percentage must total 100%.

When all information has been entered for a work area, click the Add/Modify button. This will save the information for the work area. Information for a new work area can then be added.

Figure 48

When all the fields marked with asterisks in each section have been completed, click the Submit button on the left side of the Cat Add Detail screen. This will route the request to the appropriate approvers. Upon final approval or rejection, an e-mail will be sent to the requestor.

If not all the information needed to complete the new chemical add request is known, the request can be saved by clicking Draft. A request saved as a draft can be retrieved from the Catalog Add Tracking screen.

Catalog addition requests can be deleted by clicking the Delete button.

Catalog Addition Tracking

The Catalog Addition Tracking feature of tcmIS allows users to view the status of a catalog addition request. This screen also allows approvers to find catalog addition requests needing their approval. To access the Catalog Add Tracking screen from elsewhere within tcmIS, click the Cat Add button on the toolbar as shown below in Figure 49.

Figure 49

The Catalog Add Tracking screen can also be accessed by selecting Catalog Add Tracking from the Application menu as shown below.

Figure 50

The Catalog Add Tracking screen functions similarly to the Order Tracking screen. Searches can be performed for all facilities and all work areas or for only the facility and work area specified in the corresponding drop-down boxes. Search for catalog addition requests by clicking the Search or Refresh button at the bottom of the screen.

Searches for catalog addition requests can be limited in many ways. Entering a requestor name in the Requestor box limits the search to only those requests made by the specified requestor. Entering a name in the Approver box limits the search to only those requests that have been or need to be approved by the specified approver. Clicking the button will display the Name Search screen.

Text can be entered in the Search Text box that will also limit the search. The catalog addition request number can be entered in the Request ID box to locate a specific request. The search can also be limited to only those requests in a certain status by checking one or more of the Status check boxes.

Figure 51

Once the catalog addition request is found, the status can be determined from the Status column in the request details. The status of a catalog addition request can be one of the following: Draft (New Material), Draft (New Size/Pkg), Draft (New Part No), Draft (New Approval), Pending TCM QC, Pending Approval, Pending Pricing, Pending Catalog, Ready to Order, and Rejected.

The status of the approval process can be viewed for any request by selecting the request and clicking the Approval Detail button. The Approval Detail screen displays the material details along with the status and date of every approval associated with the request.

Figure 52

Approvers should select Show all requests needing my approval to display all catalog addition requests awaiting their approval. The request can be viewed by selecting the request line and clicking the Cat Add Detail button at the bottom of the Catalog Add Tracking screen. To approve the request, the approver should click the Approve button in the Cat Add Detail screen.
