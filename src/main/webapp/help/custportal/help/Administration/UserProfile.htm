User Profile

The User Profile page is used to create new users and edit user information and certain permissions in the HGI Customer Portal. The User Profile page can be accessed from the Administration menu by selecting User Profile. The User Profile page is displayed in a new tab.

You must have administrative permission to add or change user information or permissions in the Portal. Contact the Technical Center to request administrative permissions.

Figure 1

Your user account is displayed by default. You may update any of the information displayed including first name, last name, middle initial, phone number, fax number, mobile number, email, and printer location. Click the Update Personal Info link to save any changes. You may also reset your password by clicking the Change Password link.

Figure 2

You may look up an existing Portal user and his or her personal information by clicking the magnifying glass icon . Enter the first name or last name of the user in the Search box and click Search. Locate the user in the resulting list and click anywhere on the line to select the user. Click the Return Selected User link to place that user's personal information on the User Profile page.

If you are an HGI Customer Portal administrator:

  • you may edit a user's personal information as you would your own.
  • you may also create new user accounts. Click the New button. A blank User Profile page is displayed in the existing tab.
    • Five fields are required:
      • Provide a unique Logon ID for the new user account in the Logon ID text box.
      • Enter a name in the First Name text box.
      • Enter a name in the Last Name text box.
      • Enter the user e-mail address in the Email text box.
      • Click on the magnifying glass icon across from the Default Customer label and use the resulting Lookup form to identify the customer associated with the new profile. (See Lookup form.)
    • Provide as much of the following information as you can:
      • Enter the user's phone number, fax number, and cell phone number in the corresponding text boxes.

After all required information is completed, click the Create User link at the top left of the page. Click the Cancel link to abandon creation of this new user.

The HGI Customer Portal will process the User Profile request and enter it into the database. The page will then be redisplayed, including a tcmIS ID.

Figure 5

Startup Pages button

Click the Startup Pages button on the User Profile page to view and edit the list of pages that launch automatically when the Portal is started for the selected user. The Startup Pages page will be displayed in a new window.

To add a page to the user's startup display select the check box in the Startup column for the page. To remove a page from the user's startup display, unselect the check box for the page. As an option, you may specify the order of the pages (from left to right) by entering numbers, starting with 1, in the Startup Order column.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close the page without saving changes, or after saving the changes you want.

Figure 6

Customer button

View/Edit the list of customers that this user can access and the customer Admin privileges for each of these customers.

Click the Customers button in the lower panel to open the Customers page in a new window.

Select the check box in the Access column for each customer that the user needs to view. If the account is an administrator for any customer, select the check box for the customer in the Admin column. You must have Grant Admin permission for that company to give others administrative permissions for that company.

In some cases you may wish to keep a user account active but eliminate that account's access to a particular Customer. In that case, unselect the check box in the Access column for that customer.

Click the Update link above the data grid to save any changes that were made. Click the Create Excel link to export the displayed data in Excel format. Click the Cancel link to close this screen without saving changes.

Figure 7

Ship To button

View/Edit the list of 'Ship To' locations that this user can access and any page Admin privileges.

Click the Ship To button in the lower panel to open the Ship To page in a new window.

Select the customer for page permissions from the Customer list and click the Search button. All active pages for that customer will be displayed.

For each Ship To location that the user needs to view select the check box in the Access column. For each customer where the user account needs to be see invoices select the box in the Display Invoice column. Click the Update link above the data grid to save any changes that were made. Click the Create Excel button to export the displayed data in Excel format. Click the Cancel button to close this screen without saving changes.
