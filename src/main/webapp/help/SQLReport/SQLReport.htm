The SQL Report page is used to run pre-written database queries against the tcmIS database. It is not necessary to have strong SQL skills in order to run reports on this page. The SQL Report page can be accessed from the Reports menu by selecting SQL Report. The page is displayed in a new tab.

Figure 1

Select a query to run from the Saved Queries list box. The query is displayed in the Query window. You may edit the query if needed. For instance, you may want to change the date range. Click Run Query to execute the query and produce an Excel file containing the results of the query.

You may save queries by entering a name in the Query Name box and clicking Save Query. You may delete a query from the Saved Queries list by highlighting the query and clicking Delete saved query.

Figure 2

You may also write your own SQL queries by typing in the Query box. Click the Help link for information on basic SQL syntax.

Figure 3

You may view the tables available in the Table list box. Click on a table to see the columns available in that table.
