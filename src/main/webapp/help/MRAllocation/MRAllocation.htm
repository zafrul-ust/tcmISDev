MR (Material Request) Allocation Report

+ Larger Font | + Smaller Font

The MR Allocation Report page allows you to view detailed information on items on a submitted MR (Material Request).

The MR Allocation Report is accessible in three different situations:

  • When an MR is submitted from the Material Request page at the end of the ordering process, tcmIS will generate an MR Allocation Report page. This allows you to review the entire MR and may be used to identify any issues with the MR that need attention.
Figure 1
  • On the Order Tracking page you may choose to view the order status of all items on an order by selecting an order line and clicking the MR Allocation menu item. The MR Allocation Report page will be opened in a new window for the entire order.
Figure 2
  • On the Order Tracking page you may choose to view the order status of any single part by selecting the part's line, right-clicking anywhere on the line, and clicking the MR Line Allocation menu item. This allows you view the order status of the selected item.
Figure 3

On the MR Allocation Report page the order status is listed in the Status column. If there are multiple deliveries involved each delivery is listed separately. The order status for each listing will be one of the following:

  • Available: The part is available for delivery.
  • Delivered: The item has been delivered.
  • In Purchasing: A purchase order has not yet been issued.
  • On Order: The part is on order. In this case a purchase order number and the estimated ship date will be listed.
  • Test: The material is still in testing.

Click the Create Excel button on the page to export the data in Excel format.
