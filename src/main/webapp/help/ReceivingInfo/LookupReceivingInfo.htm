Lookup Receiving Info

+ Larger Font | + Smaller Font

The Lookup Receiving Info page is used by some hubs to edit quality notes that print on the receiving checklist. The Lookup Receiving Info page can be accessed by selecting Stocking Room Operations from the Supply Chain menu. Select Receiving then Lookup Receiving Info. The Lookup Receiving Info page is launched in a new tab.

Figure 1

Select the hub from the drop-down box labeled Enter an Item ID or Client Part # to see recorded info for.

Enter the Item ID in the Haas Item ID text box. Click Look Up Info to view all receiving notes for the selected hub and Item ID.

Figure 2

Alternatively, you may enter the customer part number in the Client Part # text box. Click Look Up Info to view all receiving notes for the selected hub and customer part number.

Figure 3

Edit the receiving information and click Update to save changes. Click Delete to delete the note.

Click the Return to Lookup Page link to go back to the Lookup Receiving Info page without making any changes.

Figure 4
