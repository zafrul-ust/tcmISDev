Enter Receiving Info

+ Larger Font | + Smaller Font

The Enter Receiving Info page is used by some hubs to enter quality notes that will print on the receiving checklist. The Enter Receiving Info page can be accessed by selecting Stocking Room Operations from the Supply Chain menu. Select Receiving then Enter Receiving Info. The Enter Receiving Info page is launched in a new tab.

Figure 1

Select the hub from the Hub drop-down list.

Enter the Item ID in the text box labeled Haas Item ID.

Enter the customer part number (exactly as it appears in tcmIS) in the text box labeled Client Part #.

Enter the expected shelf life in the corresponding text box.

Select Y or N if the material requires testing, requires chem lab, is recertifiable, and requires quarantine.

Enter the required storage temperature in the text box labeled Storage Requirement.

Enter any notes in the Notes box.

Click the Enter Info button to save.

Figure 2

Use the Lookup Receiving Info page to edit the information entered.
