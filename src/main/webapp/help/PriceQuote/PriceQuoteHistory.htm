Price Quote History

+ Larger Font | + Smaller Font

The Price Quote History page is used to look up price quotes that have already been entered in tcmIS and edit them. The Price Quote History page can be accessed from the Supply Chain menu by selecting Purchasing. Select Price Quote and then select Price Quote History. The Price Quote History page is launched in a new tab.

Figure 1

Enter an Item ID in the text box labeled Show quote history for Haas Item ID. Click the Resubmit & Sort button to display all quotations for the Item ID entered.

Figure 2

Enter a supplier name in the text box labeled or any item where a quote was from. Click the Resubmit & Sort button to display all quotations for the specified supplier.

Figure 3

Click the Edit link in the Quote 1, Quote 2, or Quote 3 column to edit the existing quotation. Click the New link in the Quote 1, Quote 2, or Quote 3 column to add a new quotation.
