Request Price Quote

+ Larger Font | + Smaller Font

The Request Price Quote page is used to generate price quote requests in tcmIS. The Request Price Quote page can be accessed from the Supply Chain menu by selecting Purchasing. Select Price Quote and then select Request Price Quote. The Request Price Quote page is launched in a new tab.

Figure 1

Enter the Item ID to be quoted in the text box labeled Request a price quote for Item ID.

Select the hub from the drop-down box labeled for hub.

Optionally, you may select a customer facility from the drop-down box labeled for end user's facility (if known),

Select the customer from the for company drop-down list.

Click Request Quote.

Figure 2

The price quote request is created.

Figure 3

Proceed to the Price Quote page, locate the price quote request that was generated, and enter supplier quotations.
