Specification Update

The Specification Update page is used to view, add, and update specification information on items in the catalog.

The Specification Update page can be accessed from the Order Management menu by selecting the Specification Update item. The Specification Update page is launched in a new tab.

Figure 1

Select a catalog from the Catalog list. Use the Sort By list to indicate how you want the report sorted, hierarchically by Catalog, then Part, the Specification, or by Specification, then Catalog, and then Part.

If desired, the report can be limited using the three boxes on the right side of the search form. In the first box choose whether the search is for a Part Number or a Specification. In the second box, choose either an exact match, using is, or a partial match, using contains. In the third box enter the text to be matched. Keep in mind that an exact match is exactly that, so typos must be avoided with is.

Click Search to see the records of specifications.

Click on the Show Legend link to see the color key for the data grid.

Click the Create Excel button on the page to export the data in Excel format.

Figure 2A

In the Specification column an underlined spec number is a link: click on it to open a child window that displays the spec document that is on file.

Figure 3

The three rightmost columns contain check boxes for each listed spec.

  • When selected the boxes in the CoC and CoA columns indicate that the record is for a Certificate of Compliance or a Certificate of Analysis, respectively.
  • The Delete column is used to select a specification record for deletion.

After changing ANY of these you must click on the Update link before the change is made permanent.

Figure 4

There is a context menu available for any specification record by right clicking anywhere on the row.

  • View Part History will open a child window to display a log of all the specs for the selected part.
  • View Spec History will open a child window to display a log of changes to the selected spec for the part in the record.
  • Add Row will insert a new record row for, and immediately following, the selected record.
