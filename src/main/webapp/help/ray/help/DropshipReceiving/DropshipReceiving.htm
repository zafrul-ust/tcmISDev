Dropship Receiving

The Dropship Receiving page is used by HGI customers to record the receipt of an order in tcmIS when the material was shipped directly to the customer's facility and did not pass through an HGI hub. The Dropship Receiving page is accessed from the Order Management menu by selecting the Dropship Receiving menu item. The page is launched in a new tab.

Figure 1

There are a number of selections that define the search:

  • You must use the Company, Facility, and Ship To lists to specify the drop ship information you want.
  • In some situations Ship To allows you to choose either all docks or one single dock.
  • You may provide something to at least partially identify the material you want using the Search by boxes. Your search may be specified by Item Desc[ription], Item ID, MR Number, or Haas PO. See Search functions for more information.
  • You may specify the order in which the items are listed using the Sort list.
    • by date expected and, within date, by supplier when you select Date Exptd/Supplier;
    • by Material Request line number when you select MR-Line;
    • by Purchase Order and, within PO, by line when you select PO/PO Line;
    • by supplier and, within supplier, by date expected when you select Supplier/Date Exptd.
  • By default only orders expected to be delivered in the next 30 days are listed. To see more orders, increase the number of days in the Expected Within box.

Click the Search button to display the records. Click the Create Excel button to export the displayed data in Excel format.

Figure 2

When you have located the shipment you want to 'receive' (by entering details into tcmIS), use the text boxes on the selected line of the Dropship Receiving page to enter the needed details.

  • Although the OK check box is the first field available for action, it is actually the very last thing to do when receiving the order into tcmIS. Fill in all the other information before you select this check box.
  • Begin by entering the manufacturer's lot number from the material in the Lot column.
  • If the supplier ship date is known, enter that date in the Actual Supplier Ship Date column. Place your cursor in the field and the Date Selector form will open to allow you to choose the date. This is not a required field and can be left blank if this information is not provided by the supplier.
  • Enter the actual date the material was delivered to your facility in the DOR column. The date defaults to today's date which you may leave if that is correct. Place your cursor in the field and the Date Selector form will open to allow you to choose another date (but you may not use a date later than today).
  • Enter the expiration date of the material in the Exp Date column. Place your cursor in the field and the Date Selector form will open to allow you to choose the date. If this is not a shelf life sensitive product, enter the word Indefinite by clicking the Indefinite button at the bottom of the calendar form.
  • Enter the quantity delivered in the Qty Rec'd column. Note that it is very important to receive material in terms of the unit of measure displayed in the packaging field.
  • Enter any notes in the Notes text box.
  • Provide the supplier's delivery ticket information in the Delivery Ticket text box.
  • Select the country of origin for the material from the Ship From Country list.
Figure 3

If the material delivered has more than a single lot number or more than a single expiration date, split the shipment into enough lines to record each lot number and expiration date combination on a separate line. Click the Dupl[icate] button to add a split line, repeating until you have all the lines you need for the shipment. Complete all the required fields for each line.

Figure 4

Items are sometimes shipped in different quantities in different deliveries. These items are known as variably sized or MV items. For MV items on the Dropship Receiving page, you must enter both quantity and size in the Packaged Qty x Packaged Size column. If more than one size was delivered, use the Dup Pkg button as needed and enter the Packaged Qty x Packaged Size for each size.

Figure 5

Complete all the data entry for each line received. After all lines are complete, select the check box in the OK column for each completed line.

After all lines are marked OK, click the Receive link at the top left of the data grid to complete the process. Unchecked lines will be left for future action.

Figure 6

When the Receive link is clicked, in addition to updating the tcmIS record, a child window is opened to display the Received Receipts form which summarizes all receipts made.

  • If the customer is equipped with a Zebra printer, and it has been configured to work with tcmIS, container/kit labels can be printed by clicking the Container Labels button. Contact the Technical Center to enable label printing.
  • At HGI Distribution Hubs the Document Labels button is used to print labels that, affixed to receipt documents, automates the storage of scanned, labeled documents into tcmIS.
  • This form itself can be printed for your records by clicking the Print This Page button.

In the Receipt ID column there is a small plus-symbol-in-a-box icon. Click on this icon and tcmIS will open a child window to display the Receipt Documents form.

Figure 7

The Receipt Documents form displays a list of any documents that have been uploaded and stored for this particular Receipt ID (if any. Management of uploaded documents is handled from this form.

  • View a receipt document by clicking on the Name which is a link that will open a new window to display the previously uploaded document.
  • Select the check box in the Delete column to discard an existing receipt document. You must click the Update button to cause tcmIS to make any deletions.
  • Click the Add New Document button to display another form for uploading an additional receipt document.
  • Click the Close button to close the form and the window.
Figure 8

When the Add New Document button is clicked, tcmIS will open another child window and display the Add Receipt Document form. All fields have to be completed:

  • Accept the default Company or choose another (if desired and available) from the Company drop-down list.
  • Enter a document name in the Doc Name text box.
  • Accept the default date (which is always the current date) or choose another from the calendar form that is displayed when you click into the Doc Date text box.
  • Use the Browse button to locate the document to be uploaded for the File text box.
  • Select the type of document from the Doc Type drop-down list: COA, COC, Picture, or Receiving Checklist.
  • Click the Save button to complete the upload process or click the Cancel button to abandon the upload process.
