Ad Hoc Waste

Waste tracking was discontinued at the end of 2010. Some residual information may have been recorded in early 2011 but up-to-date information is no longer available.

The Ad Hoc Waste report describes which materials were identified as waste at a specific facility. The format of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields.

The Ad Hoc Waste report can be accessed from the Reports menu. Select Usage and then Ad Hoc Waste. The Ad Hoc Waste page is launched in a new tab.

Figure 1

The first step in creating an Ad Hoc Waste report is to specify the basic search parameters:

  • Choose the Facility from the list or leave it set at the default All facilities.
  • Choose the Work Area if available from the list or leave it set at the default All Work Areas.
  • Choose the Accumulation Points if available from the list or leave it set at the default All.
  • Choose the Vendor if available from the list or leave it set at the default All.
    • Optionally, if you choose a Vendor, you may use a Profile. Click the Search button to open a Search form. Use that to locate and select a Profile.
  • Click in the Begin Date and End Date boxes to choose those dates from the small calendar window that will open for each of them.
  • Optionally, you may use a Management Option. Click the Search button to open a Search form. Use that to locate and select a Management Option.
Figure 2

Select the fields to include in the report from the Base Fields box on the left. You may select a field by clicking on it. To select more than one, hold down the CTRL key while selecting. Add the fields to the Report Fields box on the right by clicking the right-pointing arrow . You can remove fields from the Report Fields box by selecting in the Report Fields box and clicking the left-pointing arrow icon .

You can move an item in the Report Fields box up or down relative to the others by selecting it and clicking the up- or down-pointing arrow icons.

Figure 3

A description of the base fields is available by clicking the Description link above the Base Fields box. The description will open in a new window. Close the window when you finished with it.

Figure 4

To clear all the report parameters on the current page, click the Clear Template button. This is the same as closing the page and reopening it.
