Ad Hoc Material Matrix

The Ad Hoc Material Matrix report describes which materials were used at a specific facility much like the Ad Hoc Usage report. In addition, the Ad Hoc Material Matrix report lists which materials used at the specified facility appear on selected chemical lists. The content of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields as well as any of the lists in the Chemical Lists box.

The Ad Hoc Material Matrix report can be accessed from the Report menu. Select Reports, Usage, and then Ad Hoc Material Matrix. The Ad Hoc Material Matrix page is launched in a new tab.

Figure 1

There are three areas on the Ad Hoc Material Matrix report page to specify the report you need:

  • Identifying the context (Facility, Work Area, Parts, (optionally, Dates), format)
  • Identifying the specifics (Chemicals, Fields)
  • Selecting the action (Clear template, Open template, Save template, Run Report)

Identifying the context

  • Choose the facility from the Facility list.
  • Choose the work area from the Work Area list.
  • Select the Part using the option buttons.
    • To report on one or more specific parts, select the Part Number option button
      • In the box specify whether the report is for an exact match, using is, or a partial match, using contains.
      • In the text box enter the number fully or partially to match in the report.
    • To report on all parts used during a specific time period, select the All Parts Used option button and enter the begin and end dates in the Begin Date and End Date boxes using the date selection calendar.
    • To report on all approved parts, choose the All Parts Approved option button.
  • There is one final option in this section and it does not deal with context, it provides a formatting option for the report: the List Format drop-down list. It provides three options for presentation of the quantity of a reported material an item. Only a single option may be selected at any time.
    • The Yes option format the report to show only the word ’Yes’ for each material found in each item shown on the report.
    • The xx.x% option will format the presence of each material as a percentage to one decimal place.
    • The xx.x E -y% option will format the presence of each material as a percentage in scientific notation and can be used for reports on materials which are present in extremely small concentrations..

Identifying the specific chemicals and fields

Select the chemical list (or lists) to include in the report from the Chemical Lists box on the left by clicking on it (or them). To select more than one, hold down the CTRL key while selecting. Add the list(s) to the Report Lists box on the right by clicking the right-pointing arrow icon . You can remove lists from the Report List by selecting in the Report List box and clicking the left-pointing arrow icon .

You can move an item in the Report Lists box up or down relative to the others by selecting it and clicking the up- or down-pointing arrow icons.

Figure 3

Select the fields to included in the report from the Base Fields box on the left. You may select a field by clicking on it. To select more than one, hold down the CTRL key while selecting. Add the fields to the Report Fields box on the right by clicking the right-pointing arrow . You can remove fields from the Report Fields box by selecting in the Report Fields box and clicking the left-pointing arrow icon .

You can move an item in the Report Fields box up or down relative to the others by selecting it and clicking the up- or down-pointing arrow icons.

Figure 4

A description of the base fields is available by clicking the Description link above the Base Fields box. The description will open in a new window. Close the window when you finished with it.

Figure 5

To clear all the report parameters on the current page, click the Clear Template button. This is the same as closing the page and reopening it.
