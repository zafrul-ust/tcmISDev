Work Area Usage

The tcmIS Work Area Usage page provides a report listing of all materials used in a work area within a specified time, along with a link to the MSDS information for each material. The Work Area Usage page can be accessed from the Reports menu under the Usage item. Select the Work Area Usage menu item. The Work Area Usage page is launched in a new tab.

Figure 1

Select the facility which services the work area from the Facility list. The Work Area box will be populated with all active work areas for the selected facility. Select the work area from the Work Area list. You may select only single work area.

Indicate the starting and ending date for the report in the Begin and End text boxes. You may also click the blue down-arrow characters below the text boxes to select dates from a calendar form.

Click the Search button.

Figure 2

All active items in the selected work area will be shown.

Figure 3

A right click on any record will display a context menu item to allow you to print the displayed list. Note that only the portion of the list on the screen will be printed.

Figure 4

Clicking on the Material ID link will open a child window and display the MSDS Information for the selected item.

Figure 5

Click the Create Excel button on the page to export the data in Excel format.
