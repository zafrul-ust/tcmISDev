Catalog Add Tracking

The Catalog Add Tracking page enables users to view the status of a Catalog Add request. This page also allows approvers to locate Catalog Addition requests needing their approval.

The Catalog Add Tracking page can be accessed from the Order Management menu by selecting the Catalog Add menu item, then the Catalog Add Tracking menu item. The Catalog Add Tracking page is opened in a new tab.

Figure 1

The Catalog Add Tracking page allows you to tailor your search for requested materials and view their status. You may select all facilities for which you have authority by selecting My Facilities from the Facility box or select one specific facility; likewise, the work area can be all work areas for which you have authority by selecting My Work Areas from the Work Area box or select one specific work area.

  • The Requestor box can be used to limit the search to requests by a specific individual. There is no default Requestor.
    • Leave the Requestor box blank to see requests regardless of who placed the request.
    • Enter the name of the person who placed the request in the Requestor box to see only those requests.
    • You can look up a requestor with the Name Search form (see below) by clicking the magnifying glass icon .
  • Similarly, the Approver box can be used to limit the search to those requests that have to be approved by a specific individual. There is no default Approver (but see below on Show all requests needing my Approval).
    • Leave the Approver box blank to see requests regardless of who is supposed to approve.
    • Enter the name of the person to approve the request in the Approver box to see only those requests.
    • You can look up an approver with the Name Search form (see below) by clicking the magnifying glass icon .
  • The Catalog Add request number can be entered in the Request ID box to locate a specific request.
  • Checking the Show all requests needing my Approval box limits the search to only those requests needing your approval (that is, approval by the logged in User Account; see below).
  • The search can also be limited to only those requests in a certain status by selecting one or more of the Status check boxes at the right.
    • Draft (New Material) selects only those requests still in draft status and for addition of new material.
    • Draft (New Size/Pkg) selects only those requests still in draft status and for addition of an existing material in a new size or new packaging.
    • Draft (New part No) selects only those requests still in draft status and for addition of a new part number.
    • Draft (New Approval) selects only those requests still in draft status and for a new work area approval of an existing material.
    • Pending TCM QC selects only those requests that have been submitted and await completion by the HGI Catalog Team of gathering required data (including the MSDS) and validation of the material with the manufacturer.
    • Pending Approval selects those requests that have been submitted and await approval through your company's internal processes.
    • Pending Pricing selects only those requests that have been submitted and await complete baseline pricing information.
    • Pending Catalog selects only those requests that have been submitted and await final verification by the HGI Catalog Team of quality requirements (specifications) after all other approvals are complete.
    • Ready to Order selects only those requests that have been fully approved and the materials are now available for ordering.
    • Rejected selects only those requests that have been disapproved in the request process.

You may narrow the range of your catalog request search by using the three Search boxes.

  • In the first box choose whether to base the search on Part (Raytheon part number), Part Desc[ription], [HGI] Item ID, or [Catalog Add] Request ID.
  • In the second box specify whether the search is for an exact match, using is, or a partial match, using contains. Keep in mind that an exact match is exactly that, so typos must be avoided with is.
  • In the third box you enter the number or description you want fully or partially to match in the search.

Search for requests by clicking the Search button.

Click the Create Excel button on the Catalog Add Tracking page to export results in Excel format.

The status of a request can be read in the Status column. Further information about the status in the approval process can be viewed by right-clicking anywhere on the request line and selecting the Approval Detail menu item.

You may also display the Approval Detail form from the Cat Add Request page or the Engineering Evaluation form. Click on the Approval Detail link.

Figure 3

tcmIS will display the Approval Detail form on top of the existing page.

The Approval Detail page displays the material details along with the status and date of every approval associated with the request.

Close this form when done by clicking the ’X’ in its upper-right corner.

Figure 4

Approvers should check Show all requests needing my approval before a search to display all Catalog Add requests awaiting their approval. Any QPL items and/or Engineering Evaluation items will be displayed.

For each record in the display grid, as appropriate,

  • the Cat Add Request page can be viewed by right-clicking anywhere on the line and selecting the View/Approve Cat Add menu item. The Cat Add Request page will be opened in a new tab;
  • the Engineering Enhancement page can be viewed by right-clicking anywhere on the line and selecting the View/Approve Eval menu item. The Engineering Evaluation page will be opened in a new tab.
Figure 5: Superimposed view of View/Approve Cat Add and View/Approve Eval

To approve the request, the approver should click the Approve/Reject link in the top left of the Cat Add Request page or the Approve button on the lower left of the Engineering Evaluation form, as appropriate. The Approve Detail form will open on top of the page.

Select the check box next to each approval role you are taking. (There may be only one role, or there may be more, depending on your responsibilities and authority.) Enter any notes in the Comments text box. Click the Approve button to record your approval or the Reject button to record your rejection.

The request will be sent to the next approver. Once all approvals are complete, the requestor will receive an email documenting that the request has been completed and the product is ready to order or that the request has been rejected.
