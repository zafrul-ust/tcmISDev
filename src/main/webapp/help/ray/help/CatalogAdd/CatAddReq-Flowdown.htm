Catalog Add Request - Flowdown tab

The Flowdown tab on the Cat Add Req (Catalog Add Request) page is used to view and manage the quality flowdowns for the products on the QPL.

Four individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL
  •  
  • Use Approval
  •  
  • Spec
  •  
  • Flowdown
  • QPL - the default tab (shown when the page is opened), used to view and manage products on or being requested for the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.

Flowdown is accessed after opening the Cat Add Req page by clicking on the Flowdown tab.

The Flowdown tab lists the flowdowns, if any, associated with the part number. Flowdown simply means a rule, policy, practice, or requirement for an item that applies to all work areas.

Figure 1

To associate a flowdown with a part,

  • Click the Add Flowdowns link on the Flowdowns tab. The Flowdown Search form will be opened in a new window.
  • Click the Search button on the form to display all active flowdowns or enter text in the Search box to narrow your search.
  • When you locate the desired flowdown and click anywhere on the line the Selected Flowdown link will be displayed above the data grid. Click on that link to initiate the approval request.
Figure 2

The form will be closed and the Flowdown will be added to the Cat Add Req - Flowdown tab as part of the current request with a status of Pending Approval.

Repeat this process as many times as needed to add the necessary flowdown records.

Figure 3

To remove a flowdown when it is still pending approval, right-click on the line and select the Delete Line menu item.

Figure 4

To store an electronic copy of a quality flowdown for a part, right-click on the line and select the Upload Flowdown menu item. The Upload File form will open in a new window.

Figure 5

Click the Browse button, locate and select the file on your computer. Click the Send File button. The file will be stored in tcmIS and associated with the part record Modify QPL request. The Upload File form will then be closed.
