User Profile

The User Profile page is used to create new users and edit user information and certain permissions in tcmIS. You must have administrative permission to add or change user information or permissions in tcmIS. Contact the Technical Center to request administrative permissions. The User Profile page can be accessed from the Administration menu by selecting User Profile. The User Profile page is displayed in a new tab.

Figure 1

Your user account is displayed by default. You may update any of the information displayed including first name, last name, middle initial, phone number, fax number, mobile number, email, and printer location. Click the Update Personal Info link to save any changes. You may also reset your password by clicking the Change Password link.

Figure 2

You may look up an existing tcmIS user and his or her personal information by clicking . Enter the first name or last name of the user in the Search box and click Search. Locate the user in the resulting list and click anywhere on the line to select the user. click the Return Selected User link to display that user's personal information on the User Profile page.

Figure 3

If you are a tcmIS administrator you may edit the user's personal information as described above. You may also create new users if you are a tcmIS administrator. Click the New button.

Figure 4

Select a Logon ID for the new user and enter it in the Logon ID text box. Enter the user's first name, last name, email address, phone number, fax number, and cell phone number, if available, in the corresponding text boxes. Enter a password in the Password text box. Enter the password again in the Confirm Password text box.

After all required personal information is completed, click the Create User link at the top left of the page. Click the Cancel link to abandon creation of this new user.

Figure 5

Click the Pages button on the User Profile page to view and edit the list of pages that the selected user can access and their page admin privileges. Check the box in the Access column for all pages that the user needs to view in tcmIS. Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Check the box in the Cost Report column for all facilities for which the user should be able to view on the Cost Report page. Check the box in the Create New Chemical column for all customer facilities for which the user should be allowed to enter New Chemical Requests in tcmIS. Check the box in the Releaser column for all facilities for which the user should be a releaser in tcmIS. The Releaser function must be set up and turned on separately. TcmIS can be configured to require approval of orders based on a chemical list. A chemical list is a list of CAS Numbers and limits. When a material request is placed, tcmIS will compare the chemical constituents in the requested product to any active lists. If there is a match between the list and the requested product, and delivery of the product would result in usage above the limit, the material request will be sent to the designated approver and must be approved before the order is released to Haas TCM.

Prior to turning on the List Approval function is tcmIS, lists, limits, and approvers must be defined by the List Approval Administrator. Contact the Technical Center for List Approval Admin permissions and to activate the List approval function for your facility.

From the User Profile Page, click the List Approval button. Select a facility and click Search to see all active lists for the selected facility. To designate the selected user as an approver check the box in the Approver column for each list for which the user should be approver. Click the Update link.

Click the Add List button to view and add lists for approval.

Figure 23

All possible lists are displayed. To make a list active for the approval process, check the box labeled Active.

At least one limit must be set for each activated list. Limits can be per order, per year, or over a time period. To set a limit per order, enter the quantity and select the unit of measure in the Limit Per Order column. To set a limit per year, enter the quantity and unit of measure in the Limit YTD column. Also specify the yearly starting month. To set a limit over a time period, enter the quantity and unit of measure in the Limit Per Period column. Also specify the number of days in the period. You can use any combination or all of these limits together at one time. Click the Update link to save changes.

Once the list is active and limits have been set, it is now possible to designate list approvers as described above. At least one list approver must be designated for each active list. List Approvers will receive an email from tcmIS when there is an order needing approval.
