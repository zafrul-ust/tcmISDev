Order Tracking

The Order Tracking page allows chemical requestors to determine the status of a Material Request in tcmIS. The Order Tracking page also allows approvers and releasers to view and/or release Material Requests. The Order Tracking page can be accessed from the Order Management menu, Select Order Tracking. The Order Tracking page is launched in a new tab.

Figure 1

The Order Tracking page allows chemical requestors to search for requested materials and view their status. Searches can be performed for the facility and/or work area specified in the corresponding drop-down boxes. Searches can also be performed for all facilities and/or work areas for which the requestor is approved to view by selecting My Facilities and My Work Areas respectively from the corresponding drop-down boxes. Search for requests by clicking the Search button.

Figure 2

Searches on the Order Tracking screen can be limited in many ways. For instance, selecting the Orders needing my Approval check box limits the search to only those orders needing the current user's approval. Selecting the Critical Only check box limits the search to only those orders that are currently marked critical.

The Requestor box limits the search to chemicals requested by a specific individual.Your name is shown by default. Leave the Requestor box blank to see all material requests, regardless of who placed the order. Enter the name of the person requesting the chemical in the Requestor box and click .

Find the desired requestor and highlight the associated line by clicking on it. Clicking the Return Selected User link will place the selected name in the requestor text box on the Order Tracking page.

Figure 3

In addition, chemicals may be searched for by part number, description, Material Request ID, or customer purchase order number. To limit the search using one of these criteria, select the desired field from the Search by drop-down list. Select is or contains from the next drop-down list and enter the appropriate data in the text box.

Figure 4

Searches can also be limited by their delivery status. Select the Open option to limit the search to only requests that have not yet been delivered. This option is selected by default.

Select the Released within __ days option to limit the search to only those requests that were released to Haas TCM in the past. Specify how many days in the past by entering a number in the text box.

Select the Fully Delivered within __ days option to limit the search to only those requests delivered in the past. Specify how many days in the past by entering a number in the text box.

Select the Cancelled within __ days option to limit the search to only those orders that were cancelled or rejected in the past. Specify how many days in the past by entering a number in the text box.

To find requests that are pending financial approval or pending cancellation, select the Pending option.

To find requests with any status, select the Any option.

Draft orders, or material requests that were saved in the Material Request page, can be found by selecting the Draft option. To return to the Material Request and complete the order, right-click on the MR Line and select View/Approve MR.

To learn the status of a part, right-click the anywhere on the line and select MR Line Allocation.

Figure 7
The MR Allocation Report is displayed. The order status is listed in the Status column. The order status will be one of the following:
  • Available: The part is available for delivery.
  • On Order: The part is on order. A purchase order number and the estimated ship date will be listed.
  • In Purchasing: A purchase order has not yet been issued.
  • Test: The material is in testing.

This screen can be exported to Excel by clicking the Create Excel File button.

Figure 16

Click the Show Summary link to return to the Delivery Schedule page.

Approvers and releasers may locate orders needing their approval or release by selecting the check box labeled Orders needing my Approval and click the Search button. Right-click on the Material Request to view and approve. Select View/Approve MR.

Figure 17

For Finance Approval, click the Approve link in the top left to approve the entire Material Request. If you are releasing the MR as a releaser, click the Release link. Click Reject to reject the entire material request.

Figure 18

For Use Approval, each line on the MR that has violated a use approval limit will display Pending in the Status column of the Request Line Items section. Click the Pending link to display a list of authorized approvers.

Figure 19

Individual lines on a Material Request can be approved or rejected. Highlight the line to approve or reject by clicking anywhere on the line in the Request Line Item section. Click Approve Line or Reject Line link for each line on the Material Request.

For List Approval, each line on the MR that has violated an active list threshold will display Pending in the List Status column of the Request Line Items section. Click the Pending link to display details of which list, chemical, and limit was violated by this order.

Figure 20

All active lists are displayed. If the material has constituents on the list, but no limit is violated, the list is automatically approved. Scrolling to the right displays the limits associated with each list. If there is an MR Limit, the sum of the amount of list constituents on the order is displayed followed by the MR limit.

If there is a YTD Limit and the order has constituents on the list, the cumulative sum of the constituents used year-to-date is displayed followed by the YTD limit. If there is a Period Limit and the order has constituents on the list, the cumulative sum of the constituents used since the beginning of the period is displayed followed by the period limit.

Individual lines on a Material Request can be approved or rejected. Highlight the line to approve or reject by clicking anywhere on the line in the Request Line Item sect ion. Click Approve Line or Reject Line link for each line on the Material Request.
