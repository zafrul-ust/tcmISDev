Catalog Additions

The Catalog Addition process is used to request additions or changes to the qualified products list (QPL) in tcmIS and begins on the Catalog page. From the Order Management menu select the Catalog menu item. The Catalog page is launched in a new tab.

Figure 1

In order to use the Catalog Addition process, it is important to understand the relationship between the "parts" you use and the "products" manufacturers provide because they are slightly different concepts. Every part in tcmIS has a unique part number. That number is associated with a QPL (qualified product list) that identifies all the manufacturer products encompassed by the part number. That is, a single part number may represent one, two, or more distinct 'products.' These products may differ as to packaging, usage, and even naming, but all are approved by your company for the QPL for your part number.

The Catalog Addition process allows you to request changes to those parts for which you have responsibility, as well as to request changes to some aspects of the products themselves on the QPL. Once you make a request, your company internal processes, as well as the HGI processes, see to it that the changes meet company requirements and are acceptable across the company. When that occurs, the request is approved; if there is a problem the request is rejected.

Even though the process begins with the Catalog page, most of the process is handled on additional pages. The basic pages for the Catalog Addition process are the Cat Add Req (Catalog Addition Request) and the Catalog Add Tracking (Catalog Addition Tracking) pages but there are others.

There are four types of Catalog Addition requests each with its own link. (Only the New Part link is displayed until you select an item line in the data grid.)

  • Modify QPL - request an addition or change to the list of products (qualified products list) assigned to a part number.
  • New Part - request a new part number when the material or product for the part is not yet in tcmIS.
  • New Part from current Part - request a new part number based on a part number which already exists in tcmIS.
  • New Work Area Approval - request an approval or restriction on use of a part for a specific work area or set of work areas.

Using the Catalog page, select the Facility and Work Area for which you want to make changes and use the search to locate the part number which you want to modify. Click the part number line to select it. Once you choose any part by selecting its line, the Modify QPL, New Part from current Part, and New Work Area Approval links become available.

Click whichever of the four links (above the data grid) that best fits your situation to begin the Catalog Addition process. The Cat Add Req page will be opened in a new tab.
