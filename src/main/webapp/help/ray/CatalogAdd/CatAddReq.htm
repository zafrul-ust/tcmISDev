Catalog Addition Requests

The Cat Add Req (Catalog Addition Request) page is used to request and manage additions or changes to the qualified products list (QPL) in tcmIS.

  • Open the Cat Add Req page from the Catalog page by clicking on any of these links: Modify QPL, New Part, New Part from Current Part, and New Work Area Approval.
  • Open the Cat Add Req page from the Catalog Add Tracking page (to approve or reject an existing request) by clicking on the View/Approve Cat Add context menu item on any record line.

As you can see there are four individual tabs in the lower portion of the Cat Add Req page which display different sets of information. The tab in use (active) at any time is shown with white lettering on a dark background; the inactive tabs are shown with dark lettering on a light background.

  • QPL - the default tab, shown when the page is first loaded, used to view and manage products on, or being requested for, the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.
