Catalog Addition Requests - QPL tab

The QPL tab on the Cat Add Req (Catalog Addition Request) page is used to request and manage additions or changes to the qualified products list (QPL) in tcmIS.

QPL is shown by default upon opening the Cat Add Req page or by clicking on the QPL tab when the page is open.

As you can see there are four individual tabs in the lower portion of the Cat Add Req page which display different sets of information. The tab in use (active) at any time is shown with white lettering on a dark background; the inactive tabs are shown with dark lettering on a light background.

  • QPL - the default tab shown when the page is first loaded, used to view and manage products on or being requested for the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.
Figure 1

When the Cat Add Req page is opened it will be 'set up' to handle the type of Catalog Addition that you selected on the Catalog page, the link that you selected. All four possibilities are listed here.

  • Modify QPL - request an item be added to, or removed from, the list of products (qualified products list) assigned to a part number.
  • New Part - request a new part number when the material or product for the part is not yet in tcmIS.
  • New Part from current Part - request a new part number based on a part number which already exists in tcmIS.
  • New Work Area Approval - request an approval or restriction on use of a part for a specific work area or set of work areas.

Modify QPL

The first type of Catalog Addition is the Modify QPL. There are three ways to modify the QPL.

  • Add Item is used to request a material or product already in tcmIS be added.
  • New Packaging is used to request a different size Package of a material or product already in tcmIS be added.
  • New Material is used to identify and request a new material or product for the QPL.

Add Item

To add an additional product or material to the QPL, on the Cat Add Req page (Figure 1) click the Add Item link immediately below the QPL tab. The Search Global Item page will open in a new window.

Figure 2

Use the Search Global Item page to locate the product you want to add to the QPL. Enter information about the product in the Search Text box and click the Search button. (Only the New Material link is displayed until you select an item line in the data grid.)

Figure 3

If you find the exact product, select it by clicking anywhere on the line. Click the Add Item link at the top left of the data grid to close the form (and window). The product will be added as a line on the QPL tab of the Cat Add Req page with a status of 'Pending Approval'.

Don't be confused by the fact that the term Add Item is used for the link on both this Search Global Item page and the preceding Cat Add Req page.

If you do not find the product you want, you may have to add it as a New Packaging or New Material to modify the QPL.

New Packaging

If the product is found but a different size is desired, highlight the item by clicking anywhere on the line and click the New Packaging link. The Edit Item Info page will be opened in a new window to allow you to specify details about the new packaging.

Figure 5

Since you are working on an existing product or material and requesting only a different packaging for the QPL, most of the information is already shown on the form.

  • If known, enter the Shipping Weight and select the Units from the drop-down list.
  • Enter the number of units in the # Per Comp box. (The default is 1 which is already there. Change it if necessary.)
  • Enter the Package size in the Size box and select the unit of measure from the Unit drop-down list. The number of units should be the smallest individual Package that can be purchased. For drums, pails, rolls, cans, etc., this is typically 1.

Select the package style by clicking the magnifying glass icon to the right of the Package Style text box to open the Search form. It will be opened in a new window.

New Material

If the product or material you want cannot be located with the Search Global Item form, click the New Material link to define a new product. The Edit Item Info form will open in a new window. Complete the fields as described below. Required fields are marked with an asterisk (*).

  • If known, enter the Shipping Weight and select the Units from the drop-down list.
  • Enter the name of the product in the Material Description box. Include all necessary information to procure the product including color, type, etc.
  • Enter any information regarding the grade of the material in the Grade box.
  • If the material has a specific size not described in the previous fields, enter it in the Dimension box. For example, a roll of film may have a length, width, and thickness.

Select the manufacturer by clicking the magnifying glass icon to the right of the Manufacturer text box to open the Manufacturer Search form. It will be opened in a new window.

Figure 9

Enter the name of the product manufacturer in the Search box and click the Search button. When you select a manufacturer by clicking a line the Return Selected Manufacturer link above the data grid will appear. Click that link and the Search form window will close. The selected information will be populated on the Edit Item Info page.

If the manufacturer name is not found in the list, close the Manufacturer Search form and resume work on the Edit Item Info page.

Kits

If a product has more than one component (or a kit) right-click on the Component tab and select Add Component. A new tab will be created for the additional component of the kit. This may be repeated for each needed component of the product.

Except for the first component tab, any component tab can be deleted by selecting the Remove Component menu item.

Complete the fields for each component. Those fields marked with an asterisk (*) are required but supply all the information possible.

When all components have been added and identified, click the Submit link at the top left of the page to close the form (and window). The product, with all components stored in tcmIS, will be added as a line on the QPL tab of the Cat Add Req page with a status of 'Pending Approval'.

Once a new item has been submitted you cannot add additional components to the item although you can edit any of the originally submitted components of the item.

Finalizing a QPL modification request

When you have completed work on all four tabs of the Cat Add Req page and your requested addition is in the Pending Approval status, you are ready to complete the request. If you need to because the page or screen is not wide enough, adjust the data grid with the bottom 'slider bar' control to see the rightmost column.

If you a finalizing a New Part or a New Part from current part then

  • Check the box for Incoming Testing if that is required for the part.
  • Leave the Yes radio button selected under the Time and Temp Sensitive label if the product or material is time sensitive or temperature sensitive. Otherwise click on the No radio button.

For all Catalog Add Requests

  • Enter your company's MSDS number in the Customer MSDS No box on the product line in the data grid.
  • If the product or material is time sensitive, there will be input boxes for shelf life information on the product line of the data grid.
    • Click in the Shelf Life Unit text box to cause the drop-down list to appear; select the appropriate unit/measure.
    • Unless the shelf life is Mfg Recommended or Indefinite, enter the number of units in the Shelf Life text entry box.
  • If the product or material is temperature sensitive, click into the Storage Temp text box to cause the drop-down list to appear; select the appropriate temperature range.
Figure 15

To add a suggested supplier, select the Add Suggested Supplier menu item from the context menu. The Add Suggested Supplier form will be opened in a new window.

Enter any information you have regarding the supplier on the form. When done, click the Submit link in the upper-left corner of the form. The information will be stored in tcmIS for this product line and the form/window will be closed.

Repeat these steps for each product added to the QPL.

Figure 16

When done, click the Submit link in the upper-left corner of the Cap Add Req form. The information will be stored in tcmIS.

Alternatively,

  • Click the Save link to save the request as a draft. You may return to and complete it later but it will not be reviewed nor approved until it is submitted.
  • Click the Delete link to discard the request completely. None of the information entered will be saved.

Based on your company's approval policy and process, the appropriate approvers are notified by e-mail to log in to tcmIS, review the request and approve or reject. When all the approvals are made the product is made available for use for the part number. A singe rejection means the request is denied and the product is not added to the catalog.

The status of the request can be tracked with the Catalog Add Tracking page.

View or Upload an MSDS

On an existing product, you may view the MSDS by right-clicking on the item record (except the values for Status and Item). To open a child window which to view existing MSDS on the context menu select:

  • the View MSDS menu item when working with a single item;
  • the View Component MSDS menu item when working with a Kit

The View MSDS or View Component MSDS menu item is only available if tcmIS has a copy of the MSDS on file. Otherwise, it may be present but is dimmed.

On either an existing product, or a new material, where there is no MSDS stored in tcmIS, you may upload a copy of the MSDS to tcmIS. Right-click anywhere on the item record (except the values for Status and Item) where you want to add an MSDS. To open the Upload File form, on the context menu select:

  • the Upload MSDS menu item when working with a single item;
  • the Upload all MSDS(s) for Item menu item when working with a Kit.

On the form, click the Browse button, locate and select the MSDS file on your computer. Click the Send File button. The file will be stored in tcmIS and associated with the item record Modify QPL request. The Upload File form will then be closed.

Removing a QPL Item

To request inactivation of an approved product from the QPL, right-click on the Status or Item (number) and select the Fadeout from QPL menu item. If approved, this will place the item in draw down status. Any material still in the warehouse will be delivered until stock is depleted. When depleted, the item will become obsolete.

To reverse the fadeout request before it is approved, right-click on the Status or Item (number) and select the Remove Fadeout Request menu item.

Figure 18

To remove a product in pending approval status, right-click on the Status or Item (number) and select the Delete Line menu item.

New Part

The New Part link is used request a new part number for the catalog. A search may show 'No Data' or the needed part is not listed in the data grid.

Click the New Part link on the Catalog page to initiate the process to create a new part number record. The Cat Add Req page will be opened in a new tab to allow you to create the part number request.

Figure 20

Before you can specify the particular products and materials that are to be associated with the new part, you must specify information about the part itself on the Cat Add Req form.

  • Provide a part number in the Part No box. Be sure the part number you use does not duplicate an existing part number, and that it conforms to your company's standards for a part number.
  • Select the stocking method for the part from the Stocked drop-down list.
    • Select MM if the material should be stocked in the warehouse.
    • Select OOR if the material should be Order on Request, or a non-stocked item.
  • If this part is replacing a previously cataloged part, enter that part number in the Replaces Part No box. Search for and select the replaced part number by clicking the magnifying glass icon to the right of the Replaces Part No text box to open the Replaced Part Search form in a new window.
  • Enter a description of the material as it should appear in the catalog in the Description box.
  • If appropriate, provide instructions for recertification of an item in stock using the Recert Instructions multi-line text box.
    • Explicitly state the maximum number of recertifications that can be done for the part in the # Recert Allowed text box.
  • If incoming testing is required for the part, check the box for Incoming Testing to record that.
  • Using the Time and Temp Sensitive radio buttons indicate if the material is time and/or temperature sensitive.

You may save the new part record, or submit it, using the Save or Submit link in the upper-left of the page. But the request is not finished until you add one or more products or materials to the QPL (see Modify QPL), provide any needed information on the other tabs (Use Approval, Spec, Flowdown, HMRF) and Finalize the request.

Figure 21

Use the Replaced Part Search form to locate and select the part number record that you want to replace.

  • With the Search box and Search button, locate the old part number.
  • When you locate the desired part and click anywhere on the line the Return Selected Part link will be displayed above the data grid. Click on that link to place the part number onto the Cat Add Req page.

New Part from current Part

The New Part from current Part link is used request a new part number for the catalog but based on information for an existing part. If, after searching, the needed part is not found in the Catalog but a similar part is, select the similar part and click the New Part from current Part link on the Catalog page to initiate the process to create a new part number record.

The Cat Add Req page will be opened in a new tab to allow you to create the part number request but much of the needed information is copied from the existing part you had selected.

Figure 23

As with a New Part, you must specify information about the part itself on the Cat Add Req form. The difference is that the selected product or material is already added to the QPL tab. In addition, information on the Use Approval, Spec, and Flowdown tabs is included but is in a status of Pending Approval (even though for the existing part they may already be approved.)

  • Provide a part number in the Part No box. Be sure the part number you use does not duplicate an existing part number, and that it conforms to your company's standards for a part number.
  • Select the stocking method for the part from the Stocked drop-down list.
    • Select MM if the material should be stocked in the warehouse.
    • Select OOR if the material should be Order on Request, or a non-stocked item.
  • If this part is replacing a previously cataloged part, enter that part number in the Replaces Part No box. Search for and select the replaced part number by clicking the magnifying glass icon to the right of the Replaces Part No text box to open the Replaced Part Search form. It will be opened in a new window. (See under New Part, Replaced Part Search.)
  • Enter a description of the material as it should appear in the catalog in the Description box.
  • If appropriate, provide instructions for recertification of an item in stock using the Recert Instructions multi-line text box.
    • Explicitly state the maximum number of recertifications that can be done for the part in the # Recert Allowed text box.
  • If incoming testing is required for the part, check the box for Incoming Testing to record that.
  • Using the Time and Temp Sensitive radio buttons indicate if the material is time and/or temperature sensitive.

You may add more items to the QPL as needed.
