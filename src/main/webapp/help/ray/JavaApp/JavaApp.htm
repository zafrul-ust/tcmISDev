Java Application

The Java App page is used to launch the tcmIS Java Application. The tcmIS Java Application allows Raytheon employees add new chemicals to their catalog. It is also used to create and submit Waste Orders. The tcmIS Java Application is the program that Raytheon employees traditionally think of as tcmIS. Much of the functionality previously found only in the Java Application has been transferred to web pages which are accessed from the new tcmIS Application. All features currently available only the Java Application will be migrated to web pages, which will eventually eliminate the need for the Java Application.

The Java Application tab can be launched by clicking Java App in the Main menu. Click Launch GUI.
