Ad Hoc Waste

The Ad Hoc Waste report describes which materials were shipped out as waste from a specific facility.The format of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields. The Ad Hoc Waste report can be accessed from the Report menu. Select Usage and then choose Ad Hoc Waste. The Ad Hoc Waste page is launched in a new tab.

Figure 1

The Ad Hoc Waste report is similar to the Ad Hoc Usage report, but it reports on waste that was shipped from the specified facility and work area instead of chemicals used at the specified facility and work area. The first step in creating an Ad Hoc Waste report is to specify the Facility, Work Area, Accumulation Point, and Vendor for which you would like to report from the corresponding text boxes.

Also select the begin and end dates for which you would like to report by clicking in the corresponding text boxes and selecting the date from the calendar.

Check the box labeled Exclude Hub Waste to exclude waste that was shipped from a Haas TCM hub.

Figure 3

A description of the base fields is available by clicking the Description link above the Base Fields box.

Figure 4

You may save a template of the report for use later. If this report will be used on a regular basis, saving a report template can decrease the amount of time spent configuring the report parameters. To save a template, once all parameters are set as desired, click the Save Template button. Enter a name for your template. Click Save.

Figure 5

After saving a template, it can be retrieved by clicking Open Template. Select the desired template from the drop-down list and click Open. Click Delete to remove the template from your list of saved templates.

Figure 6

When all report parameters have been specified, click Generate Report to produce the report in Microsoft Excel format.

Select the Interactive option to run the report locally on your computer and wait for results. Enter a report name and click the Batch option to run the report on the Haas TCM server. An email will be sent to you when the report is ready. It can be retrieved from the Batch Reports page.

To clear all the report parameters, click Clear Template.
