Consigned Inventory Report

+ Larger Font | + Smaller Font

The Consigned Inventory Report page is used to produce a usage report of material that is supplier owned or consigned for counting inventory groups.This report should be run and provided to the supplier monthly. The Consigned Inventory Report page can be accessed by selecting Inventory from the Reports menu. Select Consigned Inventory Report. The Consigned Inventory Report page is launched in a new tab.

Figure 1

Select the hub and inventory group from the corresponding drop-down boxes.

Enter a date range by clicking in the From and To text boxes and selecting a date from the calendar.

If looking for only usage on a specific consigned material, enter the Item ID in the corresponding text box.

Click Search to see the usage of consigned material including PO number, quantity issued, and unit price. Provide this report to the supplier listed for accurate invoicing to Haas TCM.

Click Create Excel File to export the results of the report into Excel.
