Supplier Return

+ Larger Font | + Smaller Font

Return Merchandise Authorization or Return Material Authorization (RMA) is a transaction whereby the recipient of a product arranges to return goods to the supplier to have the product repaired or replaced or in order to receive a refund or credit for another product from the supplier. The Supplier Return page is used when a material is to be returned to the supplier via an RMA. The supplier should be contacted and arrangements for the return of the material should be completed prior to processing the RMA in tcmIS.

The Supplier Return page can be access from the Supply Chain menu under Stock Room Operations. Select Receiving and then select Supplier Return. The Supplier Return page is launched in a new tab.

Figure 1

Select the Operating Entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drops-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

To process the return, first locate the Receipt ID to be returned. The easiest way to do this is to select Haas PO from the Search drop-down list and enter the Purchase Order number. Click Search.

Figure 2

Alternatively, you may select Item ID from the Search drop-down box and enter the Item ID. Click Search to see all inventory for a particular item.

Figure 3

After locating the Receipt ID to be returned, Enter the quantity returned in the Qty Returned column. Also, enter the date returned by clicking in the Return Date text box and selecting a date from the calendar. Enter the supplier's RMA number. If there is not a supplier RMA number, enter NONE. You may enter comments about the reason for the return by clicking in the Comments column and typing any applicable notes.

Check the OK box and click the Process link to process the return. Note that multiple returns can be performed simultaneously by entering the required data on multiple lines and checking the OK box for each line.

Figure 4

Click Create Excel to export the data on the Supplier Returns page to Excel format.
