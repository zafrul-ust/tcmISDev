RMA Process

+ Larger Font | + Smaller Font

Return Merchandise Authorization or Return Material Authorization (RMA) is a transaction whereby the recipient of a product arranges to return goods to the supplier to have the product repaired or replaced or in order to receive a refund or credit for another product from the supplier. The RMA Process page is used when a material is to be returned to the supplier. The supplier should be contacted and arrangements for the return of the material should be completed prior to processing the RMA in tcmIS.

The RMA Process page can be access from the Supply Chain menu under Stock Room Operations. Select Receiving and then select RMA Process. The RMA Process page is launched in a new tab.

Figure 1

To process the RMA, first locate the Receipt ID to be returned. The easiest way to do this is to enter the Purchase Order number and click Submit Query.

Figure 2

Alternatively, you may enter the Item ID and click Submit Query to see all inventory for a particular item. Note that by default inventory at all hubs is shown.

Figure 3

To display only inventory at a particular hub, select the hub and click Submit Query. If the hub is not listed in the drop-down box, please contact the Technical Center.

After locating the Receipt ID to be returned, select it by clicking the appropriate radio button in column labeled Choose. Then click the Process RMA button.

Figure 4

Enter the quantity of the Receipt ID you are returning. By default, the entire quantity of that Receipt ID in inventory will be displayed, so be sure to change this if you are not returning the entire inventory of this Receipt ID. Enter the date the material was shipped and the RMA Number provided by the supplier, if applicable. If a restocking fee is being charged by the supplier, an additional charge line must be added to the Purchase Order for the restocking fee. Enter the PO and PO Line number on the RMA Process page. Finally, enter a comment and include a reason for return and carrier and tracking information if possible. Click Submit RMA.

Figure 5

You should check the Logistics page after processing the RMA to verify that it no longer shows in inventory. If you process the RMA, but the receipt stays in inventory, it may be that the material does not belong to Haas TCM and the RMA transaction failed.  It is not possible for Haas TCM to claim vendor credit for customer owned material.  To tell if a Receipt ID is owned by Haas TCM, check the Receipt on the Logistics Page. If the receipt is client owned, you will need to contact the Technology Center to have the receipt converted back into Haas TCM owned material and credit the customer.  Then you can process the RMA normally on the new Haas TCM owned Receipt ID.
This is also true for RMA of materials for Drop Ship Locations.
