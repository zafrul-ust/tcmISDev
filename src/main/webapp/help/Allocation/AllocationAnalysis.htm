Allocation Analysis


The Allocation Analysis page provides information about which materials are allocated to Material Requests. This page can be accessed by selecting Allocation Analysis from the Supply Chain menu.

Figure 1

The Allocation Analysis page can also be accessed from the Reports menu by selecting Orders and then Allocation Analysis.

Figure 2

Select the Operating Entity and Hub for which Allocation Analysis will be performed from the corresponding drop-down boxes. You may also select an Inventory Group to display only Material Requests for a specific Inventory Group. Select a Facility from the Facility drop-down list to display only Material Requests for a specific facility. Click Search to display all Material Requests and their current status. If a specific Material Request is not listed, change the number of days in the Needed Within text box to a larger number and click Search.

Figure 3

The results may be sorted by facility, item ID and date needed, MR Line, MR Line and date needed, date needed, on time date, item type, item ID and date needed, or customer. Select the desired sort option from the Sort By drop-down list. The default is to sort by date needed.

To locate a specific Material Request or Item ID, select MR or Item ID from the drop-down box and enter the MR number or Item ID in the text box and click Search.

Figure 4

To locate only Material Requests with a specific lot status, select the lot status from the Lot Status drop-down box. Click Search.

Figure 5

To locate only Material Requests with a specific progress status, select the progress status from the Progress Status drop-down box. Click Search.

Figure 6

For Distribution Hubs you may limit the results displayed by customer by clicking the labeled Customer. For more information on looking up customers please see the Customer Lookup help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerLookup2.htm. After the customer has been populated, click Search. Click the Clear button to remove the customer.

Figure 7

To view any notes associated with an MR right-click on the line and select MR Notes.

Figure 8

For Distribution Material Requests, you may view the MR by right-clicking on the line and selecting Open MR. The MAterial Request is opened in a new tab.

Figure 9

The Allocation Analysis page displays information about the status of a Material Request. If the Alloc Type is No Alloc, no material has been allocated to fill the Material Request. If the Alloc Type is BO, a Buy Order has been generated to fill the Material Request. The Buy Order number appears in the Ref column. If the Alloc Type is PO, a Purchase Order has been issued to the supplier listed in the Supplier column. The Purchase Order number appears in the Ref column. If the Alloc Type is Receipt, the material allocated to the Material Request has been received. The Receipt ID appears in the Ref column. The Status column displays the current lot status of the material. If the lot status is Available, the material is ready to be picked and shipped to the requestor.

Click the Show Legend link to view the meanings of the various colors shown on the Allocation Analysis page.

Figure 10

For Distribution Inventory Groups, you may access the Inventory Allocation page by right-clicking on the MR and selecting Show Allocatable for InvGrp.

Figure 11

Click the Create Excel button to export the data to Microsoft Excel format.
