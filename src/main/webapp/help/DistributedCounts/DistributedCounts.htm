Distributed Counts

+ Larger Font | + Smaller Font

Some facilities must distribute usage to specific areas in the plant after the material is used. This is typically because the final usage location of the product is not known when it is ordered and purchased, but it is known after it is used. The Technical Center will instruct you whether or not to use this page.

The Distributed Counts page can be accessed from the Supply Chain menu by selecting Stock Room Operations. Then choose Counting and Distributed Counts . The Distributed Counts page is launched in anew tab.

Figure 1

If you need to distribute usage, after entering inventory levels on the Item Count page, proceed to the Distributed Counts page. Select the hub from the Hub drop-down list. Click Search.

All items requiring distribution are displayed. The usage quantity is displayed in the column labeled Usage. This usage must be entered in the Distributed Usage column corresponding with the area that consumed the material. The usage may be split between many areas as long as the total distributed quantity equals the usage quantity.

Default distributions can be provided to the Technical Center and can be displayed on this page.

Figure 2

After entering the distribution, check the box in the OK column. Multiple rows can be updated and processed together. Click the Save link above the data grid to save your progress. Click the Process link above the data grid to complete the distribution. Please note that the count is not complete until the distribution has been completed.

Contact the Technical Center to add or remove items from distribution.
