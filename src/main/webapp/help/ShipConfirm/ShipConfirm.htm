Ship Confirmation

+ Larger Font | - Smaller Font

The Ship Confirmation process documents the delivery of materials to the customer. The delivery date recorded in tcmIS is the date the materials leave the Haas facility, not the date the material arrives at the customer facility. This date is used to determine the on time delivery rate of materials to the customer. TcmIS uses the current date as the default date unless an alternate date is provided.

Watch the following movie to learn more about the Ship Confirmation process: https://www.tcmis.com/tcmIS/help/ShipConfirm/ShipConfirmMovie.htm.

Ship Confirmation is completed in tcmIS on the Ship Confirm page. This page can be accessed from the Supply Chain menu. Select Stock Room Operations, Under Delivery, choose Ship Confirm.

Figure 1

Select the Hub for which Ship Confirmation will be performed from the Hub drop-down box. Click Search to display all Material Requests which are ready to be Ship Confirmed.

There are two possible Ship Confirm processes. Some hubs must create shipments prior to completing the ship confirmation process. Other hubs simply confirm each MR Line as its own shipment.

If the Shipment ID column is blank, simply select the MR Lines to confirm by checking the box in the OK column. Click the Confirm Shipment button.

Figure 2

If there is a drop-down box in the Shipment ID column, the first step is to create shipments.

Figure 3

Materials going to the same facility may be grouped on the same shipment if they will be delivered on the same Bill of Lading. Select the OK box on the left for each line to include on a shipment. Click Create Shipment.

Figure 4

The Shipment ID is populated by tcmIS. You may change the Shipment ID by selecting a different shipment from the Shipment ID drop-down box and clicking Update Shipment.

Figure 5

Bill of Ladings can be printed from the Ship Confirm page. Select the picks to print BOLs for by checking the OK box on the left of each pick. Click Print BOL Long or Print BOL Short. You may also print a Consolidated BOL.

A Delivery Ticket or Packing List can be printed by selecting the picks for which to print Delivery Tickets. Check the OK box on the left of each pick. Click Print Delivery Documents.

Box labels can be printed by clicking Print BOX labels. Select the label size and complete the Deliver To, Part No., and No. of Labels fields. Click Generate Labels.

Figure 6

Finally, after all picks to be shipped have been assigned Shipment IDs, click Confirm Shipment.

Figure 7

If the ship confirmation date is different from the current date, enter that date in the Date Delivered text box in mm/dd/yyyy format or select the date from the calendar by clicking the diamond.

Enter the Tracking Number and any Freight Charges that apply. You may also specify the Carrier by Clicking .

Select the picks to be Ship Confirmed by checking the OK box on the left. Click Ship Confirm. After completion of the Ship Confirm process, material is considered delivered in tcmIS and is eligible for invoice to the customer. Inventory is decremented when the Ship Confirm process is completed.

Ship Confirmation of a quantity of zero will put the Material Request back on the Picking page.
