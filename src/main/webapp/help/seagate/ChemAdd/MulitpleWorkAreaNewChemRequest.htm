Adding Multiple Work Areas to New Chem Add Requests

It is now possible to request addition of new chemicals to multiple work areas on a single request in tcmIS. Previously, a separate request was required for each work area.

Initiate the New Chemical Add request as usual and complete Sections 1 and 2.

Figure 1

In Section 3 of the request, choose a Work Area from the Work Area drop-down list and enter Use and Process description. Click the Add/Modify button.

Figure 2

Repeat the process for each additional Work Area. After all work areas where the material should be approved are added, you may submit the request by clicking the Submit button.
