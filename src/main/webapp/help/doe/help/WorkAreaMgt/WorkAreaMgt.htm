Work Area Management

The tcmIS Work Area Management page is used to make needed work area modifications in the use of QPL items. The Work Area Management page can be accessed from the Order Management by selecting the Work Area Management menu item. The page is launched in a new tab.

Figure 1

Select both the facility which services the work area from the Facility list and the work area from the Work Area list. You may work on only a single work area at any one time.

You may further narrow your search for items to manage by:

  • selecting a specific User Group from the list;
  • Selecting the check box for Show Active in Work Area Only to include only items with a status of Active;
  • entering a specific text to match in the Keywords text box; and/or
  • Selecting the check box for Show Only Parts with Limits to include only items with specified limits for the work area.

There is also a Show Over Limit Approvers link on the form to display information on who has authority to approve exceptions to limits.

Click the Search button.

Figure 2

tcmIS will display all the items in the selected work area meeting your criteria. The data grid is also color-keyed to clarify the role of the column values. Click on the Show Legend link to display the color legend.

Click the Create Excel link on the page to export the data in Excel format.

Clicking on the Turn Override Off link initiates the process to change the status of the Work Area from Managed to Not Managed. tcmIS will display a small message window requiring confirmation before making the change. Also, the link itself will be changed to Turn Override On to reverse the change if needed.

  • A managed work area is one where someone in authority sets use limits on (some) materials. An order placed beyond the limit automatically generates notification to the overlimit approver. (Before approval, these show a status of ’Pending Use Approval’ in Order Tracking.

Figure 4a and 4b

The two links Activate All and Deactivate All act on all the records in the data grid. Notice that it is possible for some items to be Active and some Inactive, as indicated by the check box in the Active column. When one of these links is selected, all items are set to the same status.

The process includes a small message window requiring confirmation before making the change. The messages explain briefly the impact of the change.

Figure 5
Figure 5a: Active All and 5b: Deactivate All

Selecting any record both causes it to be highlighted and activates additional information fields in some columns of the record. This allows you to edit these Work Area Manager Override fields.

  • The Limit 1 and Limit 2 columns each contain two editable fields. Those fields are the ’x’ and ’y’ values in the limit rule ’x [quantity] every y days.
  • The Process Description column contains a single large text box where process details may be documented.
  • The Order Quantity column text box allows you to modify the quantity of the item used in a standard order.
  • The Order Qty Type column contains a list from which you identify the nature of a standard order: Default or Fixed.

This also allows you to edit these Automated Ordering Information fields.

  • Customer Ship To Code may be edited in the text box.
  • tcmIS Dock may be edited using the list.
  • Customer Deliver To may be edited in the text box.
  • tcmIS Deliver to may be edited using the list.
  • Barcode Requestor may be edited by clicking on the ellipsis (three periods) to make a selection.

Click the Update link to save the changes made in any of these fields.
