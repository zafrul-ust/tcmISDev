Engineering Evaluation

Items not assigned to the requestor's catalog may be allowed restricted use for evaluation purposes. The approval process is similar, but much shorter than adding a new catalog part. Later, if desired, the item can be converted to a catalog part.

The Engineering Evaluation process may be begun from the Catalog page after a Search is done using the Search Type of Item Catalog.

  • You must specify both the Facility and Work Area from the respective boxes since an evaluation must be for a specific work area.
  • You may restrict your search to items that have already been in the Engineering Evaluation process for your facility. Select the Previous Eval orders for this facility check box.
    • You may further specify that you want items that have been processed at your request (By me) and/or items that have been processed for the current work area (For this work area).

There will be up to four action links on the top line of the data grid.

  • For a completely new item (not in the Item Catalog), click the Define new Item and Place Evaluation Order to initiate the process.
    • If the search returned ’No Data’ this will be the only link available.
    • Similarly, this will be the only link available until a record is selected in the data grid. Three additional links (below) will appear once a record is selected.
  • For a previously evaluated item, select the item by clicking on the record and click the Place Evaluation Order from selected item link.
  • Two of the links are for initiating a new part request. These work in the same way as the identical links available when doing a Part Catalog search. See Cat Add Req page: New Part from Item.
    • New Part from: Item
    • New Part from: Item (modify Pkg)

When either Place Evaluation Order link is selected tcmIS will open a new tab and display a version of the Edit Item Info form used for viewing and managing details on an item. There is a narrow left panel displaying some basic information as well as five buttons and a wide right panel for viewing, editing, or providing details.

  • The Section 1 of 2 button causes Section 1 of the wider right panel to be displayed. This is the default display when the form is first shown.
  • The Section 2 of 2 button causes Section 2 of the wider right panel to be displayed.
  • The Submit button is the last to be used: click it when all needed information for the Evaluation Order has been provided.
  • The Save button will preserve an unsubmitted Evaluation Order for future use even though it is not yet submitted.
  • The Delete button will discard the Evaluation Order.
  • If the Item is a Kit there will be more than one tab at the top of the right panel, labeled Component 1, Component 2, etc. If it is not a kit there will be a single tab, labeled Component 1.

On both sections, any information where the text box label has an asterisk (*) is mandatory. Other information is not mandatory but the more information you can provide the easier the process may be. (For existing Catalog Items the required, and some other information, is provided by tcmIS.) Where called for, accept the default values or change them to the values needed for this item.

Use the manufacturer lookup form by clicking the magnifying glass icon to the right of the Manufacturer text box. See Lookup forms for more details.

  • If the manufacturer name is not found in the list, close the Manufacturer Search form and enter the name manually.

After you have provided all the required information, as well as other information that helps identify the product, you may request formal approval by clicking the Submit button.

  • If you have inadvertently missed providing any required information you will get a small error message window. The process will not continue until you have completed every required field.

When the item has been successfully submitted tcmIS will redisplay the Engineering Evaluation page (Section 1). The Actions section in the lower left corner will no longer display the Submit, Save, or Delete buttons. They are all replaced by a single Approval Detail button.

Figure 4

Clicking on the Approval Detail button will cause a smaller window to be displayed superimposed on the Section 1 display. Details of the status of the request as well as names of approvers, dates of any approval (or rejection) actions, and approver comments are shown.

Close the Approval Detail window by clicking on the ’X’ in the upper-right corner.

Figure 5

You may review the status of the request using the Order Tracking page. Search for the material or the request and right click on the record row when you find it. You may redisplay the Approval Detail window or, by selecting the View/Approve Eval menu item, display the Engineering Evaluation page for the request.
