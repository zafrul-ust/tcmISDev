Catalog Add Request - Spec tab

The Spec tab on the Cat Add Req (Catalog Add Request) page is used to view and manage specifications for the products on the QPL.

Five individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL
  •  
  • Use Approval
  •  
  • Spec
  •  
  • Flowdown
  • QPL - the default tab (shown when the page is opened), used to view and manage products on or being requested for the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.

Spec is accessed after opening the Cat Add Req page by clicking on the Spec tab.

The Spec tab lists the specifications which the part or material meets. Add specs to the part by clicking the Add Specifications link on the Spec tab. The Add/Edit Specification form will open in a new window.

Figure 1

Although only the specification name and title are required, enter as much information as you have for the specification.

  • Enter the specification name in the Spec box.
  • Enter any version number in the Revision box.
  • Enter any amendment information in the Amendment box.
  • Enter the specification title in the Title box.
  • Indicate whether a Certificate of Conformance or Certificate of Analysis is required by checking the COC, COA, or both boxes.

Click the Submit link to add the new spec information to the Catalog Add Request. The Add/Edit specification form will close and the spec information be added to a line on the Spec tab of the request with a Pending Approval status.

Repeat this process as many times as needed to add the required specifications.

Figure 2

To edit a specification while it is Pending Approval, right-click on the line and select the Edit menu item. The Add/Edit Specification form will open in a new window: make needed changes and click the Submit link to update the request.

To remove a specification while it is Pending Approval, right-click on the line and select the Delete Line menu item.

Figure 3

To upload a copy of the specification, right-click on the line and select the Upload Specification menu item. The Upload File form will open in a new window.

Figure 4

Click the Browse button, locate and select the file on your computer. Click the Send File button. The file will be stored in tcmIS and associated with the part record Modify QPL request. The Upload File form will then be closed.

Figure 6

If a copy has been uploaded to tcmIS, you can view the spec by right-clicking on the line and selecting the View Specification menu item. The file will be opened and displayed in a new window.
