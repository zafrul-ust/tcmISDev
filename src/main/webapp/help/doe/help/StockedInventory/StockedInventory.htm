Stocked Inventory

The Stocked Inventory page displays information about on hand and on order inventory stocked for DoE facilities at HGI hubs.

The Stocked Inventory page can be accessed from the Report menu by selecting Inventory, then Stocked Inventory. The Stocked Inventory page is launched in a new tab.

Figure 1

If there are options, select a facility and inventory group from their corresponding drop-down lists to limit the search to items associated with the selected facility and inventory group.

  • To narrow an inventory search further, enter some (or all) of a part number or material description in the Search Text box. (See Single search input box for more information.)
  • An inventory search can also be limited to those chemicals whose shelf life expires within or after a certain time frame by entering a number of days in the Expires Within or Expires After text boxes.
  • On order items that are due to be delivered within a specific number of days can be located by entering a number of days in the On Order Arrives within text box.

Click the Search button to display the records. Click the Create Excel button to export the displayed data in Excel format.

Figure 2

To find out more about the inventory of a particular item right-click anywhere on the row for the part.

  • For inventory details, select the Inventory Details menu item. The Inventory for Part page will open in a new window.
Figure 3

On the Inventory for Part page the On Hand Material (inventory) is displayed and includes the manufacturer's lot number and expiration date. On order items (In Supply Chain) are displayed and include the HGI purchase order number and the estimated date the material will be ready for shipment.

Click the Create Excel link on the page to export the data in Excel format.
