Catalog Report

The Catalog Report page can be used to create reports of the contents of any catalog to which you have access. The Catalog Report page can be accessed from the Reports menu by selecting the Catalog Report menu item. The Catalog Report page is launched in a new tab.

The Catalog Report page is highly similar to the Catalog page but does not provide any means to order materials -- it is just a report.

Figure 1

Select the facility for which the catalog reporting is to be done from the Facility list. Select the Facility Approved Only check box to limit the report to those items in the catalog which are on the QPL (qualified product list) for the selected facility.

Select the Catalog for your catalog report from the Catalog list if that is available and relevant to your report need. Indicate the work area you want: it can be all work areas for which you have authority by selecting My Work Areas from the Work Area list or one specific work area.

Enter details to more narrowly specify the report in the Search box. Search criteria can include DoE part number, HGI item ID, packaging, trade name, manufacturer, manufacturer's part number, or material description. See Single box text searches for details.

Click the Search button to display the records. Click the Create Excel button to export the displayed data in Excel format.

The Catalog report is displayed in a data grid within the same window, below the specification panel. Items are displayed in Part number order.

Figure 2

The first four columns in the data grid provide a context menu that provides access to additional information. Right click on any of them in a row for the menu.

  • The first context menu item is the Part No. for the item in the record. This is not a link: the part number itself is displayed in the item.
  • The second context menu item, Comments, leads to two additional menu items.
    • Choose Part Work Area Comments to view, add, or edit comments on this particular item, specific to this facility and work area. These comments are not shown to other facilities or work areas, nor outside DoE.
    • Choose Part Comments to view, add, or edit comments for all users of the Catalog page or the Catalog Report. Part Comments are printed on the HGI pick list at the warehouse.
    • There is no menu item for Cart Comment because the Catalog Report is not used directly in the Shopping Cart process.
Figure 3

The rightmost columns, beginning with Material, provide a context menu to view an MSDS. Right click on any of them in a row for the menu. Click on the View MSDS menu item to open a child window that displays the MSDS for the item. The View MSDS menu item is only available if tcmIS has a copy of the MSDS on file. (See, also, the MSDS Viewer.)
