Cost Report

+ Larger Font | + Smaller Font

The Cost Report page can be used to create reports regarding invoicing of chemicals, waste, gasses, and service fees. The Cost Report page is basically a view into Haas TCM customer invoices or customer cost. From the perspective of Haas TCM, this report is truly a revenue report. The Cost Report page can be accessed from the Reports menu by selecting Cost Report. Then select Cost Report. The Cost Report page is launched in a new tab.

Figure 1

Select the customer for which cost reporting is to be done from the Company drop-down list. Some customer have been segregated into financial reporting groups. If necessary, select the reporting group or leave it set to My Reporting Groups to capture data for all reporting groups for which you have permission.

Select the facility for which cost reporting is to be done from the Facility drop-down box. Choosing My Facilities will report cost information for all facilities which are currently invoiced through tcmIS for which you have Cost Report permission.

Figure 2

If a specific facility is selected, then you may choose to report on a specific Work Area or Accounting System by selecting from the corresponding drop-down boxes. If an Accounting System is selected, you can enter a charge number or PO Number to report on transactions related to this number.

Figure 3

To locate cost information related to materials requested by a specific individual, click . next to the Requestor text box.

Enter the first or last name of the requestor you are searching for and click Search.

Figure 4

Locate the requestor on the list and click the name to select the user. Click the Return Selected User link.

Figure 5

The requestors name is now populated in the Requestor text box. All reports run will display only cost information about materials requested by the specified individual. Delete the requestors name to remove this search criteria.

Figure 6

The Unit of Measure in which the report is displayed may either be the Invoice's UOM or the Item's UOM. Some customers are billed in unit of measures which are different from the Haas TCM Item unit of measure. You may select the preferred reporting unit of measure from the corresponding drop-down box.

Figure 7

To locate cost information related to a specific manufacturer, select Mfg from the Search By drop down list and enter the manufacturer in the text box. Be sure select "contains" as opposed to "is."

Figure 8

To locate cost information related to a specific customer part number, select Part Number from the Search By drop-down list and enter the part number in the text box.

Figure 9

To locate cost information related to a specific Haas item ID, select Item from the Search By drop-down list and enter the item number in the text box.

Figure 10

You may also enter an Item Description to locate cost information. Select Item Description from the Search By drop-down list, select contains, and enter a description of the material.

Figure 15

To locate cost information related to materials invoiced during a specific time period, in the field labeled Invoice Between. The calendar is displayed. Choose a date for both the beginning and ending date. Choose a specific day by clicking the date on the calendar. The dates selected will appear on the Cost Report page

Figure 16

Cost information about materials delivered during a certain time period can be found in the same manner as above. Click the text boxes labeled Delivered Between.

Materials can be grouped for cost reporting purposes by the material item type. Item types include Bulk Gas Deliveries, Bulk Waste, Cylinder Rental, Material Dispensing, Material Repackaged, Material Samples, Materials, PEI Gainshare, Special Charges, TCM Service Fees, Tank Charges, Waste Disposal, Waste Materials, and others. Select the item types to report on from the Item Type drop-down list. To report on multiple item types hold down CTRL and click the desired item types as shown below. To report on all item types, choose All Types.

Materials can also be grouped for cost reporting purposes by invoice type. Select All Types, Add Charge, Chemical, Gainshare, Gas, Service, or Waste to limit the report to these categories.
