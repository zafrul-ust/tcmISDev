Receiving QC

+ Larger Font | - Smaller Font

Receiving QC is the process of verifying the information gathered during the receiving process. Typically, Receiving QC is performed by personnel who do not participate in the initial receiving process. The Receiving QC process involves taking the original receiving checklist and/or the printed transaction record from the receiving jacket and comparing the information on those documents to the physical materials in the Hub. Materials that have been received in tcmIS, but have not been QC'd are not eligible for picking until Receiving QC is complete and the material is put in Available Lot Status.

Watch the following movie to learn more about the Receiving QC process: https://www.tcmis.com/tcmIS/help/ReceivingQC/ReceivingQCMovie.htm.

Note that not all hubs utilize the Receiving QC functionality. If a hub elects to forego the Receiving QC process, the Lot Status is set and the receipt is confirmed from the Receiving page. No QC is necessary. The Tech Center should be contacted to turn on or off Receiving QC. By default it is turned on.

Receiving QC is completed in tcmIS on the Receiving QC page. This page can be accessed from the Supply Chain Menu. Select Stock Room Operations. Under Receiving, choose Receiving QC.

Uploading Documents

It is possible to upload a scanned image of the Receiving Jacket from this screen. Click the + symbol to display the Receipt Documents page. Click Add New Document.

Figure 11

Select the company for which the receipt was made to make these documents visible to the customer. If the receipt could be allocated to multiple customers, you must upload the docments for each customer that needs to see the delivery documents. Selecting Haas TCM prevents the customer from seeing the documents and allows only Haas TCM employees to view the documents. Enter a name for the Document in the Doc Name field. Click Browse and locate the file on your computer. Select the Doc Type Picture. Click Save to upload the document to tcmIS.

Figure 12

This process can also be accomplished from the Logistics page.

Printing Labels

Container labels and/or box labels can be printed from the Receiving QC page. If needed, change the number of labels to print in the # Labels field. Check the OK box for all lines that need labels. Click the Generate Labels link and/or the Receiving Labels link above the grid.

Figure 13
Confirm Receipt

To confirm receipt of all selected lines, check the OK box and click the Confirm link above the grid. Material is now pickable if it was moved to a pickable Lost Status.
