Invoice Print

+ Larger Font | + Smaller Font

The Invoice Print page is used to view and reprint invoices for Distribution Hubs in tcmIS. This page can be accessed from the Distribution menu by selecting Invoice Print. The Invoice Print page is launched in a new tab.

Figure 1

Select the Operating Entity and Hub from the corresponding drop-down lists. You may select an Inventory Group or choose My Inventory Groups to view invoices for all inventory groups at the hub for which you have permission.

The Ship Confirm Dates are preloaded with dates in the From and To boxes. To change these dates, select a date from the calendar by clicking in the From and To boxes. Select None to clear the dates.

By default the box labeled Show Not-Reprinted Only is checked. This will display only invoices that have not been reprinted. Uncheck this box to see all invoices regardless if they have been reprinted or not.

Check the box labeled Show Never Printed to display invoices that have never been printed.

Click Search to display all selected invoices. Click the Create Excel button to export the data displayed into Microsoft Excel format.

A summary of the invoice data is displayed including the Customer Name and PO Number, Invoice Number, date shipped, and Invoice Value and Total are displayed. In addition the date the invoice was first printed and by whom is shown. If the invoice has been reprinted, the reprint date and user are shown as well.

You may sort by any column by clicking the column header. Click the column header again to sort in descending order.

Figure 2

You may limit the results to a specific customer by clicking the labeled Customer. For more information on searching for customers, see the Customer Search help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerSearch.htm.

After the Customer is populated, click Search to view all invoices for the selected customer.

Figure 3

You may also limit the results to a specific customer Ship To location by clicking the labeled Ship To. After the Ship To is populated, click Search to view all invoices for the selected customer Ship To location.

Figure 4

To locate invoices for a specific Shipment ID, select Shipment from the Search drop-down list and enter the shipment ID in the Search text box. Click the Search button to display all invoices for the Shipment ID.

Figure 5

To locate invoices for a specific Material Request, select MR from the Search drop-down list and enter the MR Number in the Search text box. Click the Search button to display all invoices for the Material Request.

Figure 6

To locate a specific invoice, select Invoice from the Search drop-down list and enter the Invoice Number in the Search text box. Click the Search button to display the invoice.

Figure 7

To locate invoices for a Customer's PO Number, select Customer PO from the Search drop-down list and enter the PO Number in the Search text box. Click the Search button to display all invoices for the Customer's Purchase Order.

Figure 8

To view only invoices in a specific currency, select the currency abbreviation from the Currency box.

Figure 9

To print an invoice check the box in the OK column for each invoice to print. Click the Print Selected Invoices link. The selected invoices are displayed in PDF format and can then be printed.
