Deliveries

The Deliveries page is used to view information about deliveries from HGI and to access documentation associated with the delivery such as receiving checklists or supplier packing lists. The Deliveries page can be accessed from the Reports menu by selecting Deliveries. The Deliveries page is launched in a new tab.

Figure 1

The first two selection criteria for your report allow you to select a facility from the Facility list and a work area from the Work Area list. You may also leave either or both set at the default values.

The Delivered Between input boxes allow you to specify a starting and/or ending date for the deliveries you want to see reported. Clicking into either input box will cause a small calendar to be displayed from which you may choose the date you want.

You must provide something to at least partially identify the material you want using the Search by boxes. See Search functions for more information.

Click the Search button.

Click the Create Excel button on the page to export the data in Excel format.

Figure 2

The Deliveries report contains a convenient link when there are any actual delivery receipt documents for a record uploaded into tcmIS by the HGI Hub (when the materials were received at the hub). These documents may include receiving checklists, packing lists, certificates of conformance, or certificates of analysis. To the right of the Receipt number in the first column, a small ’document’ icon alerts you to the presence of an uploaded document for the record.

Right click anywhere on the row for a record to see a context menu with two menu items.

  • Select Move to Another Work Area to initiate the process to transfer material to another work area.
  • Select View Receipt Document and tcmIS will open a child window to display the Receipt Document form.
  • The document icon is not a link so clicking it does not invoke any action.
Figure 3

When you click on the Move to Another Work Area menu item, tcmIS will open a child window where you must detail the transfer. There are two sorts of work area destinations for a transfer.

  • Some work areas have defaults but allow you to override the defaults. These defaults will be in normal type. There will be multiple record rows to accommodate splitting the transfer among two or more charge numbers. (Even though multiple records are permitted, often there will be only one for a particular destination.)
  • Other work areas have a default set of charge information (one or more charge number records), set up so you cannot override it. These records will be in bold type.

In addition to verifying or updating the charge information, and adding records for split transfers, you must also verify or provide other information needed for the transfer.

  • You may use the to Work Area list to change the work area to which the transfer is to be made.
  • You must use the Quantity text box to specify the quantity of the item to be transferred.
  • If needed, use the Dock list to specify the dock for the transfer.
  • If needed, use the Deliver to text box to specify the location within the work area for the transfer.

When all the transfer information is provided, click on the Submit link in the upper-left corner of the window to complete the transfer process. Before you ever click the Submit link, you may abandon the process and disregard the transfer steps by clicking the Cancel link.
