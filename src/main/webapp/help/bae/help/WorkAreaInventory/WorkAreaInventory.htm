Work Area Inventory

The tcmIS Work Area Inventory page is used to view information regarding on hand inventory. It can be accessed from either of two menu selections.

  • The Work Area Inventory page can be accessed from the Supply Chain menu. Under Work Area Inventory select the Work Area Inventory menu item.
  • The Work Area Inventory page can be accessed from the Reports menu. Under Inventory select the Work Area Inventory menu item.

Using either menu selection, the Work Area Inventory page is launched in a new tab.

Figures 1 and 2

The Work Area Inventory page allows you to tailor your search for inventory. Select the facility which services the work area from the Facility list. Select the work area group from the Work Area Group list. The Work Area box will be populated with all active work areas for the selected group. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

By default, the search results (inventory items) are sorted and displayed by Part Number. You may choose to sort by Part Number or Item ID: select the desired sort option from the Sort By list.

You may provide something to at least partially identify the material you want using the Search by boxes. (Your search may be specified by Part Number, Item Desc[ription], or Item ID). See Search functions for more information.

You may also tailor the search to locate inventory which is expiring within a specified number of days. Enter the minimum number of days in the Expires In text box and, if wanted, the maximum number of days in the To ___ days text box.

Select the check box labeled Show Only Quantity > 0 to see only work area items that currently have stock.

Click the Search button to start the search.

Figure 3

After the search is run, you may sort by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Figure 4

There are three columns on the right end of the data grid that provide information about the inventory quantities in the Work Area.

  • The Last Count Qty column displays the inventory recorded in the most recent scan.
  • The Delivered Since Last Count displays the quantity added subsequent to the most recent scan.
  • The Total Qty is the sum of the previous two columns and represents the current inventory (some of which may already be consumed and will be accounted for in the next scan.

Notice that the expiration date will only be shown on the Work Area Inventory page when the Count Type is Receipt ID.

There is also a context menu available to view the inventory quantity in the HGI Hub. Right-click on a line and select the Show Inventory Detail menu item to display details on inventory of the item at the warehouse. The Inventory Details page will be displayed in a new window.

Figure 5

The Inventory Details page shows details on the inventory with sections for Item Description, On Hand Material, and material ’on the way’ (In Supply Chain).

The expiration date of HGI inventory is always shown on the Inventory details page.

Figure 6

In many cases when you hover the mouse pointer on data cells a tool tip will pop up and display the contents of the cell. This is particularly useful when the data in the cell does not fit the default size of the cell: it allows you to see the entire contents of the cell without having to adjust the width of the column.
