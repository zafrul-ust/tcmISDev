Work Area Labels

The Work Area Labels page is used to print labels for stock management in tcmIS. The Work Area Labels page allows you to generate these labels/reports

  • Baseline
  • Work Area Labels
  • Work Area and Bin Labels
  • All Labels

These labels should be printed and applied in the proper work area locations prior to the first cabinet scan. The Work Area Labels page can be accessed from the Supply Chain menu. Under Work Area Inventory, select Work Area Labels. The Work Area Labels page is launched in a new tab.

Figure 1

Select the facility where the work areas are located from the Facility list. You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list. The Work Area box will be populated with all active work areas for the facility or the work areas for the chosen work area group.

Select the specific work area(s) from the Work Area list. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

Click the Search button.

Figure 2

The selected work areas will be shown. You may sort the list by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Figure 3

Select the check box in the column labeled Print for each item where labels are to be printed.

  • Click the Generate Baseline link above the column labeled Print to print the Work Area Baseline report. The baseline report should be displayed in each work area.
  • Click the Generate Work Area Labels link to print the work area label. This label is scanned at the beginning of each work area scan.
  • Click the Generate Work Area & Bin Labels link to print the both the work area labels and all the bin labels. (See, also, bin labels, below.)
  • Click the Generate All Labels link to print whatever labels are available for the Work Area.
Figure 4

tcmIS will display a notification window while generating the base line report. When the report is run tcmIS will open another window to display the report for printing.
