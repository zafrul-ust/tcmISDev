Work Area Stock Management

The Work Area Stock Mgmt page is used to activate and inactivate parts in work areas, set stocking levels, and add parts to work areas in tcmIS. The Work Area Stock Mgmt page can be accessed from the Supply Chain menu: choose Work Area Inventory and select the Work Area Stock Mgmt menu item. The Work Area Stock Mgmt page is launched in a new tab.

Figure 1

Select the facility where the work areas are located from the Facility list. You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list. The Work Area box will be populated with all active work areas for the facility or the work areas for the chosen work area group.

Select the specific work area(s) from the Work Area list. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

Select the box labeled Show Inactive Parts to INCLUDE parts that have been inactivated. Select the check box in the Active column to activate an item. Unselect the check box in the Active column to inactivate an item. Click the Update link above the data grid to save any changes.

You may provide something to at least partially identify the material you want using the Search by boxes. (Your search may be specified by Part Number or Item ID.) See Search functions for more information.

Click the Search button.

Click the Create Excel button to export the data in Excel format.

Figure 2

There are two things that can be edited directly on the Work Area Stock Management page.

  • The Count Type column identifies the method used for inventory replenishment for the item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to record the expiration date and, in the case of expired material, beep loudly to notify the operator to remove the expired material.
    • KanBan This is included for a future enhancement.
  • When the box in the Active column is selected, the part is available within tcmIS. Clear the box to deactivate an item.

Click the Update link above the data grid to save any changes.

The Nonmanaged Materials functionality of tcmIS is not in use by BAE Systems at this time.

There are a number of local, or context, menus on the Work Area Stock Mgmt page for managing, editing, and getting more information about items.

  • Right click on any item in a Work Area column for a menu to Add Part, Generate Work Area Label, Generate Work Area & Part Labels, or Edit Work Area Directed Charge (this latter menu item appears only when specifically authorized for a work area).

  • Right click on any item in a Count Type, Active, Catalog, Part Number, RP/SL/RQ, KanBan Reorder Qty, or Lead Time in Days column for a menu to Set Part Level or Edit Part Directed Charge (this last menu item appears only when specifically authorized for a work area).

Selecting the Add Part menu item will cause the Add Part page to open in a child window. Also, in a situation where the search finds NO records that match your Search criteria, the display grid panel will display a message stating 'No Data Found' as well as an Add Part link.

In either case, click on the Add Part link to open a child window which displays the Add Part form.

  • Select the appropriate Catalog from the Catalog drop-down list. (There may be only a single option.)
  • Enter the correct Part Number in the Part Number text box.
    • Click the magnifying glass icon to open a child window to display the Part Number Search form.
    • Locate the Part Number by searching.
    • Select the desired Part Number and click the Return Selected Part link at the top of the form. tcmIS will close the child window and insert the selected part number into the Part Number text box on the Add Part form.
  • (None of these are required numbers but provide them if you can.)
    • Enter the minimum quantity to maintain in the work area in the Reorder Point text box.
    • Enter the maximum quantity to maintain in the work area in the Stocking Level text box.
    • Enter the quantity for a reorder for the work area in the Reorder Quantity text box.
    • Enter the quantity for a KanBan reorder in the KanBan Reorder Qty text box.
    • If known, enter the number of days which it takes to procure and ship the material in the Lead Time In Days text box.
  • The Count Type column identifies which base is used for inventory replenishment for an item. Choose the appropriate type from the drop-down list.
    • Item ID indicates that the scan process only the quantity of each item is entered; the individual containers are not scanned. Checking for and removal of expired material is done by eye.
    • Receipt ID indicates that during the scan process each individual container will be scanned. This enables the scanner to recognize expired material and beep loudly to notify the operator to remove the expired material.
    • KanBan This is included for a future enhancement.

Click the Add button to add the part to the Work Area. Use the Work Area Labels page to print a new Work Area Baseline report and Bin Labels.

Figure 6

Selecting the Set Part Level menu item will cause the Part Level page to open for the item on the selected row. There you may:

  • Enter or edit the minimum quantity to maintain in the work area using the Reorder Point text box.
  • Enter or edit the maximum quantity to maintain in the work area using the Stocking Level text box.
  • Enter or edit the reorder quantity to restock the work area using the Reorder Quantity text box.
  • Enter or edit the KanBan reorder quantity to restock the work area in the KanBan Reorder Qty text box. This is included for a future enhancement.
  • If known, enter or edit the number of days which it takes to procure and ship the material in the Lead Time In Days text box.
  • Enter or edit any comments in the Remarks text box.

Click the Update link at the upper-left of the data grid to save any changes.

Figure 7

Both the Edit Work Area Directed Charge menu item and the Edit Part Directed Charge menu item will open a child window to display the Charge Information form. This form is used to edit the existing directed charge information for the work area or part. (See also part). Make the needed changes directly on the form.

  • Click the Remove Charge Numbers link to delete the existing charge numbers (if any) from the part record for this work area.
  • Use the PO box, which may appear as a text box in which to type, or may appear as a drop-down list from which to select, to change the PO for the part record for this work area.

When done, click on the Submit link in the upper-left corner to save the changes: the new information will be used until any future change is made.

If directed charges have been entered for both a part and the work area, the part directed charge will be used rather than the work area directed charge.
