Catalog Add Request - HMRB tab

The Cat Add Req (Catalog Add Request) page with its HMRB tab is used to view and manage Hazardous Material Review Board information for the products on the QPL.

Five individual tabs in the lower portion of the Cat Add Req page correspond to the basic elements of the Cat Add Req process.

  • QPL
  •  
  • HMRB
  •  
  • Use Approval
  •  
  • Spec
  •  
  • Flowdown
  • QPL - used to view and manage products on or being requested for the QPL.
  • HMRB - used to view and manage Hazardous Material Review Form information for the products on the QPL.
  • Use Approval - used to view and manage use permissions on products on the QPL.
  • Spec - used to view and manage specifications for the products on the QPL.
  • Flowdown - used to view and manage flowdowns for the products on the QPL.

Flowdown is accessed after opening the Cat Add Req page (from the Catalog page) by clicking on the HMRB tab.

Figure 1

Record the Hazardous Material Review Board information on this tab.

  • Select the intended purpose of the material from the Used As list.
  • If the material is to, or may, leave the USA, check the To Be Sent Outside U.S. box.
  • Indicate the type of use for the material from the Type of Use list.
  • Enter the Type of Material in that text box.
  • If the type of material is an adhesive, check either the Autoclavable or Structural box, as appropriate.
  • Identify whether the material is used with flying or non-flying parts by selecting from the Applied to list.

When all the HMRB information is added, click the Save link in the upper-left of the HMRB section to store the information with the current Modify QPL request.
