List Management

The List Management Report page is used to create and manage list of chemicals. The List Management Report page can be accessed from the Administration menu by selecting the List Management menu item. The List Management Report page is launched in a new tab.

Figure 1

To manage an existing list, select the list name from the List box. Click on the Search button to cause tcmIS to display a data grid of the chemicals on the selected list. The data grid includes the CAS number, the assigned chemical name, and a delete column for each chemical on the list.

Click the Create Excel button on the page to export the data in Excel format.

Figure 2

To create a new list there are two options.

  • Use the Upload List button on the upper panel of the page. tcmIS will open an Upload List form.
    • Use the Upload List form to locate and upload an Excel file containing the list information to import. Click the Process button once the file is completely uploaded to incorporate that list information into tcmIS.
    • Click the Example button to download an Excel file which displays the needed format for the information to be imported properly.
Figure 3

The second option is to create a List directly within tcmIS. Creating a new list consists of two steps: creating the list and populating the list.

  • Click the New button on the List Management page, to the right of the List box. tcmIS will open a child window to display the New List form.
  • On the New List form provide the Name and Description in the text input boxes marked by red asterisks (required information).
  • You may also use the Reference text input box to record external reference information about the source, purpose, nature, etc. of the list.
  • Click the OK button to save the list or the Cancel button to disregard what you've started.
  • The list is only a name until you actually add specific chemicals to it. See the next section for details.
Figure 4

After a list is created, by importing or creating it directly, to add a chemical to a list, select the list on the List Management page data grid and click on the Add button. tcmIS will open a child window to display the Chemical (CAS) Search form. Use it to locate and select the chemical to be added. (See Lookup forms for more details.)

Clicking the Update link on the Chemical List page is required before the new record is added to the list.

Figure 5: Chemical search form (left) and data grid (right)

To remove a chemical from a list, display the list from the List Management search form.

  • Locate the chemical (or chemicals, you may delete more than one at a time) and select the Delete column check box on the row for the chemical to be removed.
  • Click the Update link at the upper left to save changes.
  • Clicking Update is required for either adding or deleting a chemical for the list.
