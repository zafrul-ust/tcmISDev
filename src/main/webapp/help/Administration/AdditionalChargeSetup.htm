Additional Charge Setup

+ Larger Font | + Smaller Font

The Additional Charge Setup page is used by Customer Service Managers to inactivate additional charges and set default prices for Distribution Hubs in tcmIS. The Additional Charge Setup page can be accessed from the Administration menu by selecting Additional Charge Setup. The Additional Charge Setup page is launched in a new tab.

Figure 1

Select the Operating Entity from the corresponding drop-down box. Check the box labeled Show Header Charges Only to limit the search results to only currently active Header Charges. Check the box labeled Show Line Charges Only to limit the search results to only currently active Line Charges. LEave both boxes unchecked to see all charges. click the Search button.

Figure 2

To make a charge inactive as a Header Charge unselect the box in the column labeled Header Charge. To make a charge inactive as a Line Charge unselect the box in the column labeled Line Charge.

To make a charge active as a Header Charge check the box in the column labeled Header Charge. To make a charge active as a Line Charge check the box in the column labeled Line Charge.

You may also enter a default price for each charge. Enter the price in the Default Price text box.

Click the Update link to save changes.

Figure 3
Note that only active charges are able to be selected and added to Material Requests by CSRs.
