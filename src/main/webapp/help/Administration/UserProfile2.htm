User Profile

+ Larger Font | + Smaller Font

The User Profile page is used to create new users and edit user information and certain permissions in tcmIS. You must have administrative permission to add or change user information or permissions in tcmIS. Contact the Technical Center to request administrative permissions. The User Profile page can be accessed from the Administration menu by selecting User Profile. The User Profile page is displayed in a new tab.

Figure 1

Your user account is displayed by default. You may update any of the information displayed including first name, last name, middle initial, phone number, fax number, mobile number, email, printer location, and font size. Click the Update Personal Info link to save any changes. You may also reset your password by clicking the Change Password link.

Figure 2

You may look up an existing tcmIS user and his or her personal information by clicking . Enter the first name or last name of the user in the Search box and click Search. Locate the user in the resulting list and click anywhere on the line to select the user. click the Return Selected User link to display that user's personal information on the User Profile page.

Figure 3

If you are a tcmIS administrator you may edit the user's personal information as described above. You may also create new users if you are a tcmIS administrator. Click the New button.

Figure 4

Select company Haas TCM to create a new user account for Haas TCM employees. Select a Logon ID for the new user and enter it in the Logon ID text box. Enter the user's first name, last name, email address, phone number, fax number, and cell phone number, if available, in the corresponding text boxes. Enter a password in the Password text box. Enter the password again in the Confirm Password text box. Select a printer location if this user works at a warehouse and needs to print labels. If the needed printer location does not exist or you are unsure which to select, contact the Technical Center. Specify a font size for tcmIS.

After all required personal information is completed, click the Create User link at the top left of the page. Click the Cancel link to abandon creation of this new user.

Figure 5

Click the Companies button to view and edit the list of companies that the selected user can access, their company admin privileges, and whether the user can create new users for each of these companies. Check the box in the Access column for all customers that the user needs to view. If they are an administrator for that company, check the box in the Admin column. You must have Grant Admin permission for that company to give others administrative permissions for that company.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 6

Click the Pages button on the User Profile page to view and edit the list of pages that the selected user can access and their page admin privileges. Select the company Haas TCM from the Company drop-down list to grant access to Haas TCM pages. Select a customer from the Company drop-down list to grant access to customer pages. Click Search. All active pages are displayed.

Check the box in the Access column for all pages that the user needs to view in tcmIS. Check the Admin box if the selected user needs to grant access to these pages to other users. Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 7

Click the Entity button on the User Profile page to view and edit the list of Operating Entities that the selected user can access and their Entity level permissions. Select an Entity from the Operating Entity drop-down list to view permissions for the selected Entity. Click Search.

All Entity level permissions are displayed. For the selected user, check all boxes corresponding to the permissions needed. Click the Update link to save changes.

Figure 8

Click the Edit List button to add or remove Entities from the selected user's profile. Check the box in the Access column for all Entities that the user needs to view in tcmIS. If the selected user is an Buyer for the Entity, check the box in the Purchasing column. If the selected user is works in a warehouse for the Entity, check the box in the Hub Operations column. Click the Update link to save any changes.

Figure 9

Click the Operations button on the User Profile page to view and edit the selected user's operational permissions. Select an Operating Entity, Hub, and Inventory Group from the corresponding drop-down lists to view permissions for the selected user. Click Search.

All Inventory Group level permissions are displayed. For the selected user, check all boxes corresponding to the permissions needed. Click the Update link to save changes.

Figure 10

To grant access to a new Operating Entity, click the Edit List button to the right of the Operating Entity drop-down list. Check the box in the Access column for all Entities that the user needs to view in tcmIS as shown above in Figure 9. If the selected user is an Buyer for the Entity, check the box in the Purchasing column. If the selected user is works in a warehouse for the Entity, check the box in the Hub Operations column. Click the Update link to save any changes.

To grant access to a new Hub, click the Edit List button to the right of the Hub drop-down list. Check the box in the Access column for all Hubs to which the selected user should have access. If the selected user should also be an Administrator for the Hub, check the box in the Admin column. Click the Update link to save any changes.

Figure 11

To grant access to a new Inventory Group, click the Edit List button to the right of the Inventory Group drop-down list. Select the Operating Entity and Hub from the corresponding drop-down box. Click Search to see all Inventory Groups for the selected Entity at the selected Hub.

Check the box in the Access column for all Inventory Groups to which the selected user should have access. If the selected user should also be an Administrator for the Inventory Group, check the box in the Admin column. Click the Update link to save any changes.

Figure 12

Click the Facilities button on the User Profile page to view and edit the list of facilities that the selected user can access corresponding to their companies, their facility admin privileges, and their operational permissions. Select a customer from the Company drop-down list to grant facility level permissions. Click Search. All facilities which the user currently has access to are displayed. Uncheck the box labeled Show current permission only to display all facility permissions for the selected customer.

Check the box in the Administrator column for all facilities for which the user should be a facility administrator in tcmIS. Check the box in the Cost Report column for all facilities for which the user should be able to view on the Cost Report page. Check the box in the Create New Chemical column for all customer facilities for which the user should be allowed to enter New Chemical Requests in tcmIS. Check the box in the List Approval Admin column for all facilities for which the user should be a List Approver in tcmIS. The List Approval function must be set up and turned on separately. Contact the Technical Center for assistance. Check the box in the Releaser column for all facilities for which the user should be a releaser in tcmIS. The Releaser function must be set up and turned on separately. Contact the Technical Center for assistance. Check the box in the Waste Manager column for all facilities for which the user should be a facility waste manager in tcmIS. Check the box in the Safety Label column for all facilities for which the user should be able to print safety labels in tcmIS. Check the box in the WA Management column for all facilities for which the user should be a Managed Work Area Approver in tcmIS. The Managed Work Area function must be set up and turned on separately. Contact the Technical Center for assistance.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 13

Click the Edit List button to add or remove facilities from the selected user's profile. Check the box in the Access column for all facilities that the user needs to view in tcmIS. If the selected user is an administrator for the facility, check the box in the Admin column. You must have Grant Admin permission for that facility to give others administrative permissions for that facility. Select a default facility by clicking the radio button in the Default column for the desired facility.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 14

Click the Work Area button on the User Profile page to view and edit the list of work areas that the selected user can access and whether the user can create Material Requests for these work areas. Select a customer from the Company drop-down list to grant access to customer work areas. Select a facility from the Facility drop-down list to grant access to customer work areas at a specific facility. Click Search.

Figure 15

All work areas which the user currently has access to for the selected facility are displayed. Uncheck the box labeled Show current permission only to display all customer work areas for the selected customer and facility. Click Search.

Check the box in the Access column for all work areas that the user needs to view in tcmIS. You may select All Work areas to grant access to all existing work areas at the selected customer facility. To grant permission to only selected work areas at a customer facility, uncheck the box in the Access column for the facility and all work areas and select the desired boxes for the specific facility and work area.

Check the box in the Create MR column for all facilities and work areas for which the user can enter Material Requests in tcmIS.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 16

Click the Docks button on the User Profile page to view and edit the list of docks (ship to addresses) that the selected user can access for Drop Ship Receiving and whether the user has receiving privileges for those docks. Select a customer from the Company drop-down list to grant access to customer delivery locations. Select a facility from the Facility drop-down list to grant access to customer docks at a specific facility. Click Search.

Check the box in the Dropship Receiving column for all facilities which the user should be allowed to enter receipts on the Dropship Receiving page.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

Figure 17

Click the Financial Approver button on the User Profile page to view and edit the list of Financial Approvers that the selected user is a member. Select a customer from the Company drop-down list view financial approval lists. Select a facility from the Facility drop-down list to view financial approval lists at a specific facility. Click Search.

To add a user to the financial approver list click the Add link.

Figure 18

Search for the user by clicking . Enter the user's designated financial limit for each order in the displayed currency. If the user is also a financial approver, enter their designated approval limit for each order. Select the user's designated approver from the Next Level drop-down box. Click OK to add the user to the financial approver list. All users who can create Material Requests for facilities that use the tcmIS financial approval function must be added to the financial approval list. The financial approval feature can be turned on or off for a facility by the Technical Center.

Figure 19

To edit a user's financial approval or order limits, select the user by clicking the name and click the Edit link. Update the selected user's Order Limit, Approval Limit, or Next Level Approver and click OK.

Figure 20

Select a user by clicking the user's name and click the Delete link to remove a user from the financial approval list.

Select a user by clicking the user's name and click the Alt Approver link to view and edit the user's alternate approver. Alternate approvers receive email notification from tcmIS and have the same approval authority as the approver. Click New Alt Approver to add a new alternate approver.

Figure 21

Enter the alternate approver's first or last name and click Search. Select the desired user by clicking the user's name and click the Return Selected User link.

Figure 22

Check the box in the Delete column and click the Delete button to remove the selected alternate approver.

Figure 23

Click the User Group button on the User Profile page to view and edit the list of User Groups that the selected user is a member. User groups can be used to limit who can order specific categories of material. Select a customer from the Company drop-down list view user group lists. Select a facility from the Facility drop-down list to view user group lists at a specific facility. Select This User's Access to view only user groups the selected user has access to already. Select All Users' Access to view all active user groups for the selected facility. Click Search.

Select the user group and click the Add User to Group link.

Figure 24

The selected user is added to the user group.

Figure 25

Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes.

To remove a user from a user group, first display all member of the group by clicking the + in front of the user group name. Click the user name of the user to remove from the group and click the Delete User From Group link.

Figure 26

Enter a name for the user group and a description. Click Add New User Group. The user group will now be displayed in the list and users can be added to it. Contact the Technical Center to add customer parts to the user group.

Figure 27

Click the Startup Pages button on the User Profile page to view and edit the list of pages that launch automatically when tcmIS is started for the selected user. to add a page to the user's startup pages, check the box in the Startup column for the desired pages. You may also set the order of the pages by entering number starting with 1 in the Startup Order column.

Click the Update link above the data grid to save any changes that were made. Click Create Excel to export the displayed data into Excel. Click Cancel to close this screen without saving changes
