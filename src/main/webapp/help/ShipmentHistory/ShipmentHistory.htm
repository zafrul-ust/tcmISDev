Shipment History

+ Larger Font | + Smaller Font

The Shipment History page is used to review deliveries from tcmIS, enter freight carrier and tracking numbers, and print Packing Lists or Consolidate Bill of Ladings. This page can be accessed from the Supply Chain menu by selecting Stock Room Operations and the choosing Delivery. Then select Shipment History. The Shipment History page is displayed in a new tab.

Figure 1

Select the hub and/or inventory group from the corresponding drop-down lists. Enter a date range to search for all shipments during the specified date range by clicking the From and To text boxes. Select the date from the calendar or click None to clear the date.

You may also select a facility from the Ship To drops-down box to view only shipments to the selected facility.

Click Search to display all shipments that meet your search criteria.

Figure 2

You may also enter a Shipment ID in the corresponding text box to locate a specific shipment. Click Search.

Figure 3

Freight carrier and tracking number information can be entered or updated on the Shipment History page. To update the freight carrier, select a carrier from the corresponding drop-down list. Enter the new tracking number or edit the current tracking number in the Tracking Number text box. Select the line that has been updated by checking the box on the far left of the data grid. Click the Update link above the data grid to save changes to all selected lines.

Figure 4

Packing lists can be printed from the Shipment History page. Select all shipments to be included on the packing list by checking the box on the far left of the data grid. Click the Print Packing List link above the data grid. The packing list is displayed in PDF format and can be printed or saved.

Figure 5

Consolidated Bill of Ladings can be printed from the Shipment History page. Select the shipment to be printed by checking the box on the far left of the data grid. Click the Consolidated BOL link above the data grid. The Bill of Loading is displayed in PDF format and can be printed or saved.
