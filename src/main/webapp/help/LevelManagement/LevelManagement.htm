Level Management

+ Larger Font | + Smaller Font

The Level Management page is used to review usage and determine appropriate stocking levels. This page can be accessed from the Supply Chain menu by selecting Stock Room Inventory and the choosing Inventory. Then select Level Management. The Level Management page is displayed in a new tab.

Figure 1

To view stocking levels across all Operating Entities, Hubs, and Inventory Groups for which you have permission, select My Entities, My Hubs, and My Inventory Groups from the corresponding drop-down lists. Select Item ID from the Search box and enter the Haas TCM Item ID. Click the Search button.

Figure 2

To limit the results, you may select the Operating Entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drop-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

If the Haas Item ID is known, select Item ID from the Search box and enter the Haas TCM Item ID for which inventory management is to be performed in the text box. Click Search.

If the Item ID is not known, you may also search by Item Description or Catalog Part Number (customer part number).

Figure 3

Check the box labeled Show Only Items Issued in the Last __ days and enter a number of days to display only materials that have been delivered from the selected Hub and Inventory Group in the number of days entered.

Figure 4

Check the box labeled Show Only On Hand Items to display only materials which currently have stock in the selected Hub and Inventory Group.

Figure 5

Various information about the item is displayed including current stocking method, reorder point and stocking level, quantity on hand and on hand value, total unallocated quantity, unpickable quantity, quantity on order and not allocated, quantity in purchasing and not allocated, date of last receipt, quantity issued in the last 30 days, quantity issued in the last 30-60 days, quantity issued in the last 60-90 days, Minimum, average, and maximum inventory value, and item description.

A summary of the data on the Inventory Level Management page can be created in Microsoft Excel format by clicking the Create Excel button.

Right-click on any line to view historical changes in the stocking type and levels or to change the stocking type and levels.

Figure 6

The Min Max Levels page is displayed. For more information on changing stocking levels, see the Min Max Levels help page: https://www.tcmis.com/tcmIS/help/MinMaxLevels/MinMaxLevels2.htm.
