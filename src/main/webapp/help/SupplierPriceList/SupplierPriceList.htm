Supplier Price List

+ Larger Font | + Smaller Font

The Supplier Price List page is used to view, edit, and create supplier price lists. The Supplier Price List page can be accessed from the Supply Chain menu by selecting Purchasing. Then select Supplier Price List. The Supplier Price List page is launched in a new tab.

Figure 1

Select the Operating Entity and Hub from the corresponding drop-down lists. Select an inventory group from the Inventory Group drop down list. Or select My Inventory Groups to see price list information for all inventory groups for which you have permission at the selected hub.

Check the box labeled Show Inactive to see include inactive pricing. The default is to show active pricing only.

To see price list information for a specific Item, select Item and is from the Search drop-down lists and enter the Item ID in the text box. Click the Search button.

Figure 2

If the Item ID is not known, select Item Desc and contains from the Search drop-down boxes. Enter a description of the material in the text box. Click Search.

Figure 3

To view price lists for a specific supplier click the to the right of the Supplier label.

Enter the supplier name in the Search Text box and select the country from the Country drop-down list. Click Search.

Select the desired supplier by clicking anywhere on the line. Click the Return Selected Supplier link to populate the Supplier field on the Supplier Price List page.

Figure 4

Click Search to locate all pricing for the selected supplier. Click the Clear button to remove the supplier.

Figure 5

If you have permission, you may update the Priority, Supplier Part Number, Lead Time, Freight Carrier, Incoterms, Consignment Status, Purchase Multiples, Rounding, Status, Currency, Quantity and Unit Price. Click the Update link to save any changes.

Figure 6

To add an Item to a Supplier Price List, Click the link labeled Add Item.

Select the Operating Entity, Hub, and Inventory Group from the corresponding drop-down boxes.

Figure 7

The Ship to is by default All. If the supplier pricing only applies to a specific Ship To location click the to the right of the Ship To label. Enter a portion of the Ship To location in the Search Text box and click Search. Select the desired Ship To location by clicking anywhere on the line and clicking the Return Selected Ship To ID link.

Figure 8

Enter the Item ID by clicking the to the right of the Item label. Enter the Item ID or part of the Item Description in the Search Text 1 and Search Text 2 boxes. Click Search. Select the desired Item ID by clicking anywhere on the line and clicking the Return Selected Item Id link.

Figure 9

Enter the supplier by clicking the to the right of the Supplier label. Enter the part of the supplier name in the Search Text box and select the country from the Country drop-down list. Click Search. Select the desired supplier by clicking anywhere on the line and clicking the Return Selected Supplier link.

Figure 10

Once the Ship To, Item, and Supplier are populated on the New Item Form click OK to add the supplier price record.

Figure 11

If multiple suppliers exist for this item, select the priority from the Priority drop-down list for each supplier.

Enter the supplier part number in the Supplier Part Num column, if known.

Enter the lead time in days in the corresponding text box.

If the material should ship using a specific carrier, enter it by clicking the .

If the material is to be purchased on consignment from the supplier, check the box in the Consignment column.

If the material must be purchased cases or multiples of a specific number, for example, in cases of 6, enter the quantity in the column labeled Multiples Of.

Select REVIEW from the Rounding drop-down to review rounding of buy orders based on the multiples. Select ROUND UP to round up buy orders based on the multiples. Select ROUND DOWN to round down on buy orders based on the multiples.

Select ACTIVE from the Status drop-down to set the item and supplier pricing. The Active Priority 1 pricing will be used when POs are created by Buyers from the Buy Orders page. Select INACTIVE to inactivate the price.

Enter the start date for the price by clicking in the Start Date column and selecting the date from the calendar.

Select the currency from the Currency ID drop-down list and enter the price for a quantity of 1. Click the Update link.

Figure 12

To enter a price break, right-click on an existing line in the Price List for the Item and select New Price Break.

Figure 13

Enter the quantity at which the price break applies and the price. You may repeat this process to create as many price breaks as needed. Click the Update link to save any changes.

Figure 14

You may enter a new start date and price for Items already in the price list by right-clicking on the line and selecting New Start Date.

Figure 15

A new row is added for the selected Item. Enter the start date for the new price by clicking in the Start Date text box and selecting a date from the calendar. Select the currency from the drop-down labeled Currency ID. Enter the price in the Unit Price column. Click the Update link to save any additions or changes.
