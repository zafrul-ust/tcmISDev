No Sales

+ Larger Font | + Smaller Font

The No Sales page provides information regarding material which has had no sales in a given time period. The No Sales page can be accessed from the Reports menu by selecting Inventory and then No Sales. The No Sales page is launched in a new tab.

Figure 1

Select the Operating entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drops-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

Enter a number of days in the within __ days box to display all inventory which has had no no sales in the entered number of days. Click the Search button.

The quantity in inventory, average cost and total value of each item is shown as well as the date of the last sale.

Figure 2

You may sort by any column by clicking on the column header you would like to sort by. Click the column again to sort in descending order.

Figure 3

To view more details regarding the inventory, right-click on any line and select Show Inventory Detail.

Figure 4

Details regarding the stock on hand including the Receipt ID, lot status, manufacturer lot number, expiration date, and warehouse bin location are displayed.

Figure 5

Click the Create Excel button on the No Sales page to export the results into Excel format.
