Catalog

The Catalog page contains information on materials approved for use in the specified facility and work area. The catalog page is used to create and submit requests for delivery of chemicals known as Material Requests in tcmIS. The Catalog page can be accessed from the Order Management menu by selecting Catalog. The Catalog page is launched in a new tab.

Figure 1

The first step in ordering a chemical is to search for it in the part catalog. Select the facility and work area where the material will be used from the corresponding drop-down lists. Information about the desired material can be entered in the Search box. Search criteria can include UTC part number, Haas TCM item ID, packaging, trade name, manufacturer, manufacturer's part number, or material description. Click the Search button to search the part catalog for the selected facility and work area.

Figure 2

The more specific the information entered, the more the search is narrowed. Multiple words as a logical expression can also be used to refine a search (separate words by AND, OR, NOT; for example, prc AND 1750). To search for all chemicals for a specific location, leave the Search box blank and click the Search button.

Figure 3

Some additional constraints can be added to a search of the part catalog. Selecting the Active for selected Facility/Work Area option displays only those chemicals associated with the selected facility and work area. Selecting the Approved Only check box limits the search to only those chemicals approved for use at the selected facility and work area.

Selecting the All Catalogs option displays chemicals for all UTC catalogs. This feature can be useful when searching for a new material to add to the part catalog. Selecting the Active Only check box limits the search to chemicals that are currently active. A chemical is inactive if it becomes obsolete for the part number, is no longer manufactured, or is not yet approved for the part number.

More information can be obtained for a specific chemical by right-clicking on the part details to display a menu of additional selections. These selections include information regarding the inventory, reorder point, stocking level, specifications, comments, and lead time.

Figure 4

To find out more about the inventory of a particular material, right-click on the part details and select Inventory. The inventory on hand at the Haas TCM warehouse is displayed. The current Reorder Point and Stocking Level is also shown. Typically only Min/Max items will have have on hand inventory. Select Part Inventory to see more details regarding the on hand inventory.

Figure 5

On hand inventory is displayed including the manufacturer's lot number and expiration date. Items that are on order are displayed below the on hand material . The estimated date the material will be ready to ship is listed.

Figure 6

Comments can be added to a particular part by right-clicking on the part details and selecting Comments. Then choose Part Comments. These comments will be available for viewing any time the part is displayed on the Catalog page for the catalog which the comment was added. Part Comments are printed on the Haas TCM pick list at the warehouse.

Figure 7

Enter a comment and click the Update link to save new comments. You must search the catalog again the see the comments.

Figure 8

To remove a comment, check the box in the OK column and click the Delete link.

Figure 9

Cart Comments can also be added to a part. When a Cart Comment is added to a particular part, the comment is saved. Any saved comments will be displayed every time the part is added to a shopping cart for the catalog that the comment was added to. This feature can be used to share information about excess chemicals with other users. This can help encourage efficient use of chemicals. Buyers and Haas TCM personnel will not be shown this comment. To add a Cart Comment to a part, right-click on the part details and select Comments. Then choose Cart Comments.

Figure 10

Enter your comment and click OK to save the comment. To remove a comment, simply delete the comment text and click OK.

Figure 11

The Cart Comment is displayed when the part is added to the shopping cart. Click OK to add the part to the shopping cart. Click Cancel to return to the Catalog page without added the part to the shopping cart.

Figure 12

Comments related to usage in a specific work area can be added or viewed by right-clicking on the part details and selecting Work Area Comments.

Figure 13

Add new comments by typing in the Comments box. Select the New check box and click the Submit Changes link to save the comment.

Figure 14

To remove a comment select the Delete check box and click the Submit Changes link.

Figure 15

To view a list of all work areas where a material is approved for use, right-click on the part details and select Comments then choose Approved Work Areas.

Figure 16

All work areas where the part is approved for use is displayed along with any usage restrictions.

Figure 17

Specifications associated with a part can be displayed by right-clicking on the part details and selecting Specification. All specs associated with the part are displayed. Select a specification name to display a copy of the spec.

Figure 18

The most current revision of the specification is displayed. You may view previous revisions by selecting a date from the Revision Date drop-down list.

Figure 19

Lead time information can be viewed by right-clicking on the part details and selecting Lead Time Plots. The median lead time in days is displayed.

Figure 20

Each delivery lead time is graphed as well as the median lead time, tenth percentile, and ninetieth percentile. This lead time represents the time between order placement in tcmIS and delivery to the requestor.

More information about the item ID can be displayed by right-clicking on the Component Description. You can view the Material Safety Data Sheet (MSDS) of a particular component by right-clicking on the component as shown below and selecting View MSDS.

Figure 22

Figure 23

The next step in ordering a chemical is to add it to the shopping cart. After locating a chemical in the part catalog, select it by clicking anywhere on the line and then click the Add To Cart link above the data grid.

Figure 24

If a material with a comment attached to it is added to the shopping cart, the comment will be displayed. After viewing the comment, clicking Continue will add the part to the shopping cart. Clicking Cancel will continue without adding the part to the shopping cart.

After adding chemicals to the shopping cart, the quantity can be changed by double clicking on the current Quantity and entering the quantity needed. The quantity can also be changed on the Material Request page which is described in depth below.

Enter the date the material is needed by clicking in the Date Needed text box and selecting a date from the calendar.

Figure 25

Notes can be entered by clicking in the Notes column. A text box appears where notes can be entered. Simply click outside of the text box to save the notes.

Figure 26

Select the check box in the Critical column to designate this material request as a critical order.

Figure 27

Searching and adding chemicals to the shopping cart can be repeated until all needed chemicals are in the shopping cart. Once a request has been added to the shopping cart, all further additions to that cart must be for delivery to the same facility but may be for different work areas.

Figure 28

An item can be removed from the shopping cart by clicking on the corresponding line and clicking the Delete Line link in the shopping cart.

Once all needed chemicals have been added to the shopping cart, the next step in completing the order is to check out. When the order is ready to submit, clicking on the Check Out link in the shopping cart will display the Material Request page.

Figure 29

The Material Request page allows the chemical requestor to assign the delivery dock and location and enter charge numbers. This must be done for each line item on the request.

Select each line item on the order and complete the required information. Select the dock and deliver to location from the corresponding drop-down lists.

The format of the charge number information varies depending on the facility. Some facilities have multiple accounting systems, and the requestor will be prompted to specify which accounting system to use immediately after clicking the Check Out link. If there are multiple accounting systems, one must be selected from the drop-down box labeled Select an Account System. The selected accounting system must be used for all items in the shopping cart. Enter the appropriate charge number information. For the % column, enter the percentage of the line item to be charged to that number. The percentages for each line item must total 100%. This allows you to split charges between multiple charge numbers.

The quantity ordered may be changed on this page by updating the number in the Qty column.

The need by date was entered from the shopping cart, but it may also be updated on this page. Note that the process of assigning a need date must be completed for each line item on the Material Request screen. There are two types of need dates: those that are needed on or before a specified date, and those that are needed exactly on a specified date or set of dates. The latter deliveries are referred to as Scheduled Deliveries. Select Deliver By or Schedule from the Delivery Instructions drop-down list. If an on or before date is needed, choose Deliver By and specify the requested deliver by date by clicking the current delivery date and selecting a different date from the calendar. This selection will inform Haas TCM to deliver the material on or before the specified date.

If a delivery needs to be scheduled for a specific date, whether it is a single delivery or multiple deliveries, select Schedule from the Delivery Instructions drop-down list and click the Schedule button.

Figure 30

The Delivery Schedule page is displayed. From the Delivery Schedule page, deliveries can be scheduled on a regular basis or on specified dates. To schedule a delivery on a specific date, click the blue diamond and select the delivery date from the calendar. Enter the quantity to be delivered in the Current Quantity column. Click the Update link to save this delivery.

Figure 31

To add more delivery dates, right-click and select Add row. Repeat the process to add as many deliveries as needed.

Figure 32

To remove a delivery date right-click on the delivery to remove and select Remove.

Figure 33

If deliveries should be scheduled on a set frequency, click the Frequency link. In the Quantity box, enter the number of items to be delivered for each of the scheduled deliveries and the frequency. For example, if one item should be delivered per month, enter the number 1 in the Quantity box, select Monthly, and enter 1 in the box labeled Every x Months. If one item should be delivered per week, enter the number 1 in the Quantity box, select Weekly, and enter 1 in the box labeled Every x Weeks. If deliveries need to be scheduled on a more frequent basis than monthly or weekly, the Daily option should be selected and enter a number of days in the Every x Business Days. The date of the first delivery should be entered in the Starting Date box by clicking the blue diamond and selecting a date from the calendar. Enter to total quantity to be delivered in the Total to Deliver box. Click OK to create the schedule.

Figure 34

If the total quantity of the order was changed in the delivery schedule process, a dialog box will appear notifying the requestor that the scheduled quantity is different than the ordered quantity. Clicking Yes updates the quantity in the original order and displays the delivery schedule. Clicking No displays the Delivery Schedule screen so the requestor can adjust the schedule.

Figure 35

Click the Show Calendar link to view and edit the schedule on the calendar. You may edit the schedule by clicking on a day and entering a quantity. To remove a delivery enter quantity zero.

Figure 36

Add deliveries by clicking on a date and entering the quantity to be delivered. Click Update to save the changes.

Figure 37

Click the Show Summary link to return to the Delivery schedule screen.

Figure 38

Click the Return to MR link to return to the Material Request page and complete the order.

The requestor may enter the End User Name and Department in the corresponding text boxes to indicate the point-of-use for the material request. This information applies to all line items on the request. The information from the End User and Department boxes will appear on the delivery ticket shipped with the material. These fields are flexible, and remarks can be customized to assist the end user once the item is delivered.

Figure 39

The requestor may also complete the Notes and Critical fields on this page. The Notes field provides a dialog box for entering information that needs to be communicated to the Haas TCM purchasing team. Selecting the Critical box notifies the Haas TCM purchasing team that this is a Priority 1 buy. A Priority 1 buy is an order in which a late delivery will shut down production or the item's lead-time is long enough to create a problem with operations.

Figure 40

The final step in placing a chemical or gas order is to submit the order. Before submitting the order, the requestor should review the order for accuracy and completeness. If the requestor does not have all of the required information, clicking the Save link will save the material request at draft status. Draft orders can be retrieved and completed through the Order Tracking page. Individual lines can also be removed from a material request by selecting the line and clicking the Delete Line link. The entire material request can be deleted by clicking the Delete link if the MR has not been submitted. If more parts need to be added to the order, the Return to Cart link can be used to return to the Catalog page. Click the Print This Page link to print a copy of the material request.

Check Scrap/Obsolete if this order should be filled from material in inventory that has been declared scrap or obsolete.

Click the Submit link to complete the process of placing the order. If the order is for a part that is an OOR whose item is currently in stock, tcmIS will prompt the requestor to allow the OOR shelf life requirement to we waived to allow the stocked item to be shipped immediately. The requestor is informed of the minimum remaining shelf life for the delivered items from stock. Often this material is currently owned by the requestor's company and may be a lost value if no one accepts the material. If this transaction is authorized, tcmIS will attempt to fill the request immediately. There are some occasions where shipping an OOR from stock could threaten the ability to ship the stocked item to the Min/Max requirement. In these cases, the order will revert to normal OOR processing.

Some facilities may use the financial approval, use approval, and/or releaser functions of tcmIS. If any of these features are used for the selected facility and work area, the material request may be sent to the designated approver or releaser prior to being released to Haas TCM. Check the Order Tracking page to see the status of your order.

If financial approval is needed, click the Ext Price to see the approver's contact information.
