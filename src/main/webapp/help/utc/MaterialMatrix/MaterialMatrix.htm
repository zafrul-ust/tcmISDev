Ad Hoc Material Matrix

The Ad Hoc Material Matrix report describes which materials were used at a specific facility much like the Ad Hoc Usage report. In addition, the Ad Hoc Material Matrix report lists which materials used at the specified facility appear on selected chemical lists. The format of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields as well as any of the lists in the Chemical Lists box. The Ad Hoc Material Matrix report can be accessed from the Report menu. Select Usage and then choose Ad Hoc Material Matrix. The Ad Hoc Material Matrix page is launched in a new tab.

Figure 1

The first step in creating an Ad Hoc Material Matrix report is to select a facility and work area from the corresponding drop-down boxes.

This report can be run for a specific part number, for all parts used in a given time period, or for all approved parts. To run the report for a specific part number, choose the Part Number option and enter the part number in the text box.

Figure 2
To run the Ad Hoc Material Matrix report for all parts used during a specified time period, select the All Parts Used option and enter the begin and end dates by clicking in the Begin Date and End Date text boxes and select the date from the calendar.

Figure 3

To run this report for all approved parts, choose the All Parts Approved option.

You may save a template of the report for use later. If this report will be used on a regular basis, saving a report template can decrease the amount of time spent configuring the report parameters.To save a template, once all parameters are set as desired, click the Save Template button. Enter a name for your template and click Save.

Figure 8
After saving a template, it can be retrieved by clicking Open Template. Select the desired template from the drop-down list. Click Open to retrieve the report template. Click Delete to permanently delete the template.

Figure 9

When all report parameters have been specified, click Generate Report to produce the report in Microsoft Excel format.

Select the Interactive option to run the report locally on your computer and wait for results. Enter a report name and click the Batch option to run the report on the Haas TCM server. An email will be sent to you when the report is ready. It can be retrieved from the Batch Reports page.

To clear all the report parameters, click Clear Template.
