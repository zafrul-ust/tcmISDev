Customer Purchasing

The Customer Purchasing page is used in the process whereby UTC manages purchasing and purchase orders while HGI continues to receive, manage, and issue the inventory. UTC buyers issue purchase orders and add the PO information to tcmIS. The inventory is owned by UTC so there is never any payable for UTC generated in tcmIS.

The Customer Purchasing page can be accessed from the Order Management menu by selecting the Customer Purchasing menu and then clicking on the Customer Purchasing sub-menu item. The page is launched in a new tab.

The buyer makes the final decisions in creating the needed purchase orders and updates tcmIS information on the Customer Purchasing page, including the addition of the relevant PO number.

Use the Customer Purchasing page to identify items that need to be purchased; tcmIS adds items to the page based on both stocking level (MM) and Material Request (OOR) information. To see only these records, select the check box labeled Show only Open Requests.

  • Select the Inventory Group from the Inventory Group drop-down list, or use the default All selection.
  • Select the check box labeled Show only Open Requests to limit the data to items for which a PO is still needed; leave it unselected to view both Open and Processed records
  • You may search for a specific buy order or group of buy orders by entering some or all of the UTC PO number in the Search text box. See Search functions for more information.
  • The records will displayed in ascending order of the Item ID as that is presently the only option in the Search By drop-down-list.

Click the Search button to display the records.

Open Records

To handle Open records

  • Enter the PO number in the PO text box.
  • Modify the quantity, if needed to match the PO, in the Qty text box.
    • The quantity entered should be the quantity on the PO for that buy order, not the total quantity for that item on the UTC PO.
  • The Promised Date defaults to today; click on the blue diamond icon to open a small calendar from which to choose the actual date committed by the supplier, or click the None button.
  • Identify the supplier in the Supplier text box.
  • Select the check box in the OK column for each record you want to update.
  • Click the Process link at the top of the data grid to mark all checked records as completed.

Once the buyer has updated the record it is no longer Open and becomes a customer buy order in tcmIS. The HGI Hub anticipates the delivery; and processes and inventories the material on arrival; Items are issued as needed by UTC. (See, also, the Customer Buy Orders page.)

Please note that there may be multiple buy orders for the same item. These will be grouped together by default. Multiple buy orders can be grouped and placed together on the same UTC PO.

Click the Create Excel File button to export the data in Excel format.

Processed Records

To view details of, or update a processed record, click the PO number which is a link. tcmIS will open a child window and display the record. This form will set the current user as the Entered By identification: if it is not your record, choose the appropriate buyer name, or Any Buyer, from the Entered By drop-down list and click the Search button to display the record you want.

If changes are needed, make them in the appropriate boxes, select the check box in the OK column, and click the Update link at the top of the data grid.
