MARS Request/Incoming Testing

A MARS Request for Incoming Testing becomes a required part of the Receiving process when a Part is added to the Catalog AND the Incoming Testing checkbox is selected. That selection is recorded in tcmIS.

After a part is approved through the Catalog Add Request process, it is possible to assign one or more tests selected from those available in tcmIS. These tests are performed whenever needed for newly Received items or for recertification.

If a part is not originally requested with Incoming Testing selected, please get in touch with the HGI Catalog team if Incoming Testing is needed.

Figure 1

Select the Test Definition menu item (the third item on the context menu on the left half of a Catalog Report data grid) to access test records. tcmIS will open a child window displaying the Test Definition form. (See also, Catalog Report.)

Figure 2

The Test Definition form displays a record for each test for the part. On this form you may

  • Click the Add row link to insert an empty record in which to record another test.
    • On the new row in the first column, select the additional test from the drop-down list.
      • Optionally, you may:
      • enter a value in the Unsuccessful Tests for Skip column;
      • Select the check box for Test Allowed Skip; and/or
      • Specify the Test Type in the text input box of that column.
  • Click the Delete link to remove a record.
  • Click the Edit Test link while a record is selected to open the Add/Edit [test] Type form to view and manage all the tests defined in tcmIS.
  • Click the Update link to record any additions or changes. They are not permanent until this is done.
Figure 3

The Add/Edit [test] Type form displays a record for each test defined in tcmIS. On this form you may

  • Click the Add link to insert an empty record in which to record a new test.
    • On the new row in the first column, provide a Short Name for the test.
    • Click into the Description column to open a drop-down, multi-line text input box and enter the description of the test.
    • Click into the Test Criteria column to open a drop-down, multi-line text input box and enter criteria for the test.
  • Click the Delete link to remove an empty or new record. (This link is not available unless a new record is currently selected.)
  • Click the Update link to record any changes. Changes are not permanent until this is done.
