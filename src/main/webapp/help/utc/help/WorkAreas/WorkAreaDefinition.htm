Work Area Definition

The Work Area Definition page is used to view, manage, and print information about work area bins and parts set up in tcmIS. The Work Area Definition page can be accessed from the Supply Chain menu. The Work Area Definition page is launched in a new tab.

Figure 1

Select the facility for the work area(s) from the Facility list. You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list.

Click the Search button. The work areas will be displayed in the data grid.

Figure 2

Each work area has almost two dozen characteristics which can be viewed and managed on the Work Area Definition page. For clarity these are described in sections.

  • Active status
    • A work area may be active or inactive. If it is inactive it will not appear on the Work Area list on the Catalog page: no orders can be placed and no parts added. On the Work Area Definition page data grid the first (left-most) column is the Active column. If the check box is selected for the line, the work area on that line is active. If the box is cleared, the work area is inactive.
  • Work Area
    • Every work area must have a name (and must belong to a work area group; see below). The name uniquely identifies the work area to distinguish it from all other work areas. The work area name may be edited directly in the text box in the Work Area column.
  • Drop Ship
    • If the check box is selected materials are delivered to the work area directly from the manufacturer; if it is clear all deliveries will be from an HGI Hub.
  • Offsite
    • If the check box is selected the work area is physically located away from the facility itself; if it is clear the work area is physically located within the facility.
  • Work Area Group
    • The work area group associates a work area with others with which it is operationally or organizationally tied. The work area group is identified in the Group column.
      • To locate or edit a work area group, click on the magnifying glass icon in the Lookup column right of the Group column. The Work Area Groups form will open in a new window.
      • This form is NOT the standard Lookup Form because there is no Search function, but it may be used to choose a Work Area Group, add a new Work Area Group, or edit an existing Work Area Group.
  • Ordering Method
    • Ordering method specifies the means by which inventory may be obtained for the work area. It is indicated and edited using the list in the Ordering Method column. There are three choices:
      • Manual Ordering indicates the replenishment occurs when a customer places an order using the Catalog page.
      • Stock Level Ordering indicates the replenishment occurs when a work area scan indicates that additional inventory is needed.
      • Manual & Stock Level indicates that both manual and stock level ordering are used for obtaining needed inventory.
  • Pulled Within Days
    • The number of days prior to expiry when material should be removed from the work area for Stock Level Ordering Work Areas. When a work area is scanned and a material’s expiry date is within the Pulled Within Days, the scanner is instructed to remove the material.
  • Days Between Scan
    • Enter the number of business days that should elapse between successive scans (e.g., 5 days indicates a weekly scan).

Changes made on the Work Area Definition page are not saved until the Update link is clicked.

Click the Create Excel button to export the data in Excel format.

Figure 3: leftmost fields
  • Area The Area fields are not in use by UTC. Area is a somewhat formal description of a work area or delivery point, corresponding to plant maps and the like, while delivery point and work area are frequently colloquial names. The four columns are:
    • Area: A physical location at the facility or department.
    • Building: A building name or number within a facility.
    • Room: A room name or number within a building.
    • Lookup: A link to a form to find an Area.
  • Inventory Group
    • Select the Inventory Group, at the HGI Hub which should be the source of materials for delivery to the work area from the drop-down list in this column.
  • Account[ing] Sys[tem]
    • UTC facilities each use only one accounting system so the default here is the correct and only choice.
  • Dock Name
    • Select the Dock Name used for the work area from the drop-down list in this column.
  • Dock Fixed
    • If the check box is selected the dock may not be altered for the work area on an MR; if it is clear the dock may be selected from a list when the MR is created.
  • Delivery Point Name
    • Select the Delivery Point Name used for the work area by clicking on the magnifying glass icon in this column. tcmIS will open a child window to display the Manage Delivery Point page to select, edit, and/or create a delivery point record.
  • Delivery Point Fixed
    • If the check box is selected the delivery point may not be altered for the work area when submitting an MR; if it is clear the delivery point may be assigned when the MR is created.

Changes made on the Work Area Definition page are not saved until the Update link is clicked.

Figure 3A: middle fields
  • Contact Name: Enter the name of the contact person for the work area in the text box.
  • Phone: Enter the telephone number of the contact person for the work area in the text box.
  • Email: Enter the e-mail address of the contact person for the work area in the text box.
  • Location: Enter the location of the contact person for the work area in the text box.
  • Customer Work Area Id: Enter the customer work area ID, if any, of the contact person for the work area in the text box.

Changes made on the Work Area Definition page are not saved until the Update link is clicked.

Figure 3B: rightmost fields

In addition to the Update link, there are two other links in the upper-left of the data grid:

  • The New Work Area link may be clicked to add a new work area line to the data grid. You must edit this line to complete the information needed, and click the Update link to save the new record.
  • The Show Legend link will cause a small window to open, on top of the current page, to display a color key to any shading used to characterize the records in the data grid.

Work Area Groups form

The Work Area Groups form lists all the existing work area groups to which any particular work area may be assigned. There are three things that can be done on this page.

  • Locate and select an existing work area group
    • Select a work area group by clicking the record row;
    • Insert the work area group into the Work Area Definition by clicking the Return Work Area Group link at the top of the form.
    • The Work Area Groups form will close and the selected group will be inserted into the Group column on the Work Area Definition page
  • Edit one or more of the existing work area group names.
    • The work area name may be edited directly in the text box in the Work Area Group column.
  • Create a new work area group.
    • Click the New Work Area Group link at the upper-left of the page. A blank work area group line will be added at the bottom of the list: you may have to scroll down the list to see it.
    • Enter a work area name directly into the text box in the Work Area Group column.

No changes made on the Work Area Groups form are saved until the Update link on this page is clicked.

Figure 5

Edit Work Area Directed Charge

Selecting the Edit Work Area Directed Charge context menu item for any record will open a window to display the Directed Charge form. This form is used to view or edit the existing directed charge information for the work area or the part.

If directed charges have been entered for both a part and the work area, the part directed charge will be used rather than the work area directed charge.

Figure 6

The Edit Work Area Directed Charge form allows modifications to the default charge distribution associated with a work area. Enter or choose the appropriate charge numbers in the left column and the percentage values that are appropriate in the right text column; click the Submit link to record them in tcmIS. This capability is only available to those who have the appropriate responsibility and authority.
