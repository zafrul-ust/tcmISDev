Work Area Count

If manual entry of work area inventory is necessary, use the Work Area Count page. The Work Area Count page can be accessed from the Supply Chain menu. Under Work Area Inventory, select Work Area Count. The Work Area Count page is launched in a new tab.

Figure 1

Select the facility where the work areas are located from the Facility list. You may choose to narrow the range of work areas by selecting a specific work area group from the Work Area Group list.

The Work Area box will be populated with all active work areas for the facility or the work areas for the chosen work area group. Select the specific work area(s) from the Work Area list. You may select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area in the Work Area box, or you may select All work areas.

Click the Search button to display the work area count information to be shown in the data grid.

  • Enter or edit the quantity in the work area for each item as needed.
  • Click the Update button to save any changed quantities.

Material Requests for inventory replenishment will be created based on the information you have provided but the Work Area Inventory page will not be updated until the next work area scan is uploaded to tcmIS.

In many cases when you hover the mouse pointer on data cells a tool tip will pop up and display the contents of the cell. This is particularly useful when the data in the cell does not fit the default size of the cell: it allows you to see the entire contents of the cell without having to adjust the width of the column.

Click the Create Excel button to export the results of the report in Excel format. This file can be saved and printed to assist in manually doing an inventory.
