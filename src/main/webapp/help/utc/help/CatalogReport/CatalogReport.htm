Catalog Report

The Catalog Report page can be used to create reports of the contents of any catalog to which you have access. The Catalog Report page can be accessed from the Reports menu by selecting the Catalog Report menu item. The Catalog Report page is launched in a new tab.

The Catalog Report page is highly similar to the Catalog page but does not provide any means to order materials -- it is just a report.

Figure 1

The first step in running the Catalog Report identify the facility, catalog, and work area with their respective drop-down selection lists.

Enter details about the desired material in the Search box. Search criteria can include UTC part number, HGI item ID, packaging, trade name, manufacturer, manufacturer’s part number, or material description. The more specific the information entered, the more the search is narrowed. Multiple words as a logical expression can also be used to refine a search (separate words by and, or, but not; for example, prc AND 1750). To search for all parts/items/chemicals leave the Search box blank.

Select the Facility Approved Only check box to limit the report to those items in the catalog which are on the QPL (qualified product list) for the selected facility.

Initiate the report by clicking the Search button.

The Catalog report is displayed in a data grid within the same window, below the specification panel. Items are displayed in Part number order.

Click the Create Excel button on the page to export the data in Excel format.

Figure 2

Right click on any of the first four columns in the data grid to display a context menu that provides access to additional information.

  • The first context menu item is the Part No. for the item in the record.
  • The second context menu item, Comments, leads to two additional menu items.
    • Choose Part Work Area Comments to access comments on this particular item, specific to this facility and work area. These comments are not shown to other facilities or work areas, nor outside UTC.
    • Choose Part Comments to view or add comments for all users of the Catalog page. Part Comments are printed on the HGI pick list at the warehouse.
    • Select the Test Definition menu item for viewing or managing Incoming Test (MARS Request) details for the record. (This is only available for parts or materials for which incoming testing is required.)
Figure 3

Right click on any of the last seven columns in the displayed data for a context menu to view the part’s MSDS. Click on the View MSDS menu item to see the MSDS in a child window. The View MSDS menu item is only available if tcmIS has a copy of the MSDS on file.

Figure 4

Click on the View MSDS menu item to see the MSDS for an item. tcmIS will open a child window and display the current version of the MSDS. (For more information see the MSDS help page.)
