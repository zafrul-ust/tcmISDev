Charge Numbers Set Up

The Charge Numbers Set Up page is used to view, manage (add, edit, inactivate), and print information about valid charge number combinations set up in tcmIS. The Charge Numbers Set Up page can be accessed from the Supply Chain menu. The Charge Numbers Set Up page is launched in a new tab.

Figure 1

Select the facility for the charge number(s) from the Facility box.

In some cases you may also be able to choose the accounting system or the charge type from the respective lists. In other cases there is only one possibility and only one value in the box.

Additionally, for some facilities, you may narrow your search by specifying all or part of the charge number particulars in one or more text boxes on the right side of the search parameter panel. Depending on your own situation these possibilities may include one or more of the specific elements included in your charge system, for example Cost Center, Charge #, Network, Work Order, etc. (Some locations have no charge number options.)

Click the Search button. The work areas will be displayed in the data grid.

Figure 2: Charge Numbers Set Up search parameter panel

Figure 2A: Various Charge Numbers examples

tcmIS will display the result of your search in a data grid. Each row in the grid has a check box to indicate whether the Charge Number(s) is Active and one column for each element in the charge number, e.g. one column for Network, or one for Cost Center and one for Charge #, etc. When there is more than one element, it is the combination of all elements that make up a single, valid, charge number.

You may sort the list of charge numbers by any of the elements, ascending or descending, by clicking in the header of the data grid for the element you want. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Figure 3

A charge number may be active or inactive. If it is inactive it cannot be used to place orders or add inventory.

On the Charge Numbers Set Up page data grid the first (left-most) column is the Active column. If the check box is selected for the line, the charge number on that line is active. If the box is clear, the charge number is inactive; it is also highlighted with a dark background on the row.

Figure 4

When you attempt to inactivate a charge number (unselect the check box and click the Update link), tcmIS will notify you if the number is ’in use’ (meaning the charge number is the default charge number for a work area or a part number). A small message window will be shown. You must click the OK button on the message window to proceed with an inactivation in this situation.

Figure 5

In order to add a Charge Number to tcmIS you must click the New link on the upper-left of the data grid. tcmIS will add a new, blank row to the data grid where you enter the values for all the elements of the charge number combination in their respective columns. (The active check box is selected by default.)

New charge numbers and any changes made on the Charge Numbers Set Up page are not saved until the Update link is clicked.

Note, also, that a Delete link is available to eliminate the newly created charge number record if you realize it is incorrect or unneeded.

Figure 6

Any charge number may be edited directly in the row and column for the record and number element. No changes made on the Charge Numbers Set Up page are saved until the Update link on this page is clicked.

In the figure below, the highlighted section shows the change that has been made before the Update link has been clicked.
