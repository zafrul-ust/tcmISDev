Spec Update

The Spec Update page is used to view and edit specifications associated with UTC part numbers. The Spec Update page can be accessed from the Order Management menu by selecting Spec Update. The Spec Update page is launched in a new tab.

Figure 1

Select a catalog from the Catalog drop-down box. Enter a UTC part number in the text box and click Search.

Figure 2

Select Specification and contains from the drop-down lists and enter a specification number in the text box. Click Search.

Figure 3

Click the link in the Specification column to display a copy of the specification in PDF format.

Figure 4

Edit certification requirements by checking or unchecking the boxes in the CoC (certificate of conformance) or CoA (certificate of analysis) columns. Check the box in the Delete column to remove the specification. Click Update to save any changes.

Figure 5

Right-click on a line and select View Part History to display the history of specifications for the selected part.

Figure 6

The specification history is displayed in a new window.

Figure 7

Right-click on a line and select View Spec History to display the history for the selected spec.

Figure 8

The specification history is displayed in a new window.

Figure 9

Right-click on a line and select Add Row to add a new specification.

Figure 10

Select the specification from the drop-down list and check the CoC and/or CoA boxes depending on your requirements. Click Update to save the new specification.

Figure 11

Click Create Excel File to export the date from the Spec Update page to Excel.
