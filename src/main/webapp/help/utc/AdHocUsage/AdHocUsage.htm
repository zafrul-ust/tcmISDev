Ad Hoc Usage

The Ad Hoc Usage report describes which materials were used at a specific facility.The format of this report is not predetermined and can be altered to include any of the fields included in the list of Base Fields. The Ad Hoc Usage report can be accessed from the Report menu. Select Usage and then choose Ad Hoc Usage. The Ad Hoc Usage page is launched in a new tab.

Figure 1
The Ad Hoc Usage report can report on usage of all chemicals, a single chemical by CAS Number, or chemicals on a specific chemical list. To report on all chemicals used at a facility, select the All option.

Figure 2

To report only on a specific chemical, select the Single Chemical option and enter a CAS number in the text box.

Figure 3

You can search for a CAS number by clicking the Search button. Enter a description of the chemical in the Search text box and click Search. Locate the desired chemical and highlight it by clicking anywhere on the line.The CAS Number is populated in the Selected CAS Number box. Click OK to return to the Ad Hoc Usage page. The selected CAS Number is populated in the CAS Number text box.

Figure 4

To report on chemicals that are included on a chemical list like SARA313 or Ozone Depleting Substances, select the List option and select the desired list from the drop-down box.

Figure 5

You can view the list by clicking the View List button. All chemicals that are included on the selected list are displayed.

Figure 6
Once the chemicals to report on have been selected, specify a facility, work area, dock, and delivery point from the corresponding drop down lists. Select the time period to report by clicking in the Begin Date and End Date boxes and select the date from the calendar.

If desired, the report can be limited to a specific part number by entering the part number in the text box labeled Part Number.

Figure 7
The results of an Ad Hoc Usage report can be further limited by material category and manufacturer. To report only on chemicals in a specific category, select the category from the Material Category drop-down list.
Figure 8
To limit the results of the report to a specific manufacturer, enter the manufacturer name in the Manufacturer text box.
Figure 9

Finally, select the base fields that you would like included in the report. Click a field from the Base Fields list on the left and move it to the Report Fields list on the right by clicking . Similarly, you can remove fields from the Report Fields list by clicking .

Figure 10

A description of the base fields is available by clicking the Description link above the Base Fields box.

Figure 11
You may save a template of the report for use later. If this report will be used on a regular basis, saving a report template can decrease the amount of time spent configuring the report parameters. To save a template, once all parameters are set as desired, click the Save Template button. Enter a name for your template and click Save.

Figure 12

After saving a template, it can be retrieved by clicking Open Template. Select the desired template from the drop-down list. Click Open to retrieve the report template. Click Delete to permanently delete the template.

Figure 13

When all report parameters have been specified, click Generate Report to produce the report in Microsoft Excel format.

Select the Interactive option to run the report locally on your computer and wait for results. Enter a report name and click the Batch option to run the report on the Haas TCM server. An email will be sent to you when the report is ready. It can be retrieved from the Batch Reports page.

To clear all the report parameters, click Clear Template.
