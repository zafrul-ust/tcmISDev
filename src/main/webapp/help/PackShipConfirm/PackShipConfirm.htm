Pack Ship Confirm

+ Larger Font | + Smaller Font

The Pack Ship Confirm page is used to update tracking information and confirm shipments to the U.S. Government. This page can be accessed from the Supply Chain Menu. Under Stock Room Operations, select Delivery then select Pack Ship Confirm. The Pack Ship Confirm page is launched in a new tab.

Figure 1

Select the hub and transportation mode from the corresponding drop-down lists. Click Search to see all shipments that have been packed.

You may update the Lead Tracking Number or the Box Tracking Number by typing in the corresponding text boxes. Check the box in the Update column and click the Update Shipment link above the data grid to save any changes.

When the material has shipped, it must be confirmed in tcmIS. Select all orders which have shipped and click the Confirm Shipment link above the data grid.

Figure 2

Enter the date delivered in the corresponding text box. Check the box in the Confirm column for all shipments to confirm. Click the Confirm Shipment & Return to Main link above the data grid to confirm all selected shipments.
