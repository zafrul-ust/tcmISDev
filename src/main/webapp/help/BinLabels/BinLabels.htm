Bin Labels

+ Larger Font | + Smaller Font

The Bin Labels page is used to print bin (storage location) labels for a warehouse in tcmIS. This page can be accessed from the Supply Chain Menu. Under Stock Room Inventory, select Reconciliation then select Bin Labels. The Bin Labels page is launched in a new tab.

Figure 1

Select the hub and room where the bins are located. Click Search. All active bin locations are displayed. Check the box in the Print column for all bin labels to be printed. Click Generate Labels.
