Transfers

+ Larger Font | + Smaller Font

The Transfers page is used to view details regarding Transfer Requests between hubs in tcmIS. This page can be accessed from the Reports menu by selecting Orders and the choosing Transfers. The Transfers page is displayed in a new tab.

Figure 1

To view all Transfer Requests across all Operating Entities, Hubs, and Inventory Groups for which you have permission, select My Entities, My Hubs, and My Inventory Groups from the corresponding drop-down lists for both the Source and Destination. Click the Search button.

Details regarding the Transfer Requests are displayed. The details include the source Operating Entity and Inventory Group, the destination Operating Entity and Inventory Group, the Item ID and Item Description of the material being transferred, the quantity requested to be transferred, the quantity received at the destination, and the quantity in transit. The freight carrier and tracking number may be displayed as well, if entered by the shipper.

Click the Create Excel button to export the data displayed into Microsoft Excel format.

Figure 2

To limit the results, you may select the Operating Entity from the corresponding Source and Destination drop-down lists. You may also select the Hub and Inventory Group from the corresponding drop-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

Figure 3

If the Haas Item ID is known, select Item ID from the Search box and enter the Haas TCM Item ID in the text box. Click Search.

Figure 4

If the Item ID is not known, you may also search by Item Description. Select Item Desc from the Search box. Be sure to change the search criteria from is to contains. Enter a portion of the Item Description in the Search text box and click the Search button.

Figure 5

You may also limit the results using the date the Transfer Request was made. Enter dates in the Request Date between fields by clicking in the text boxes and selecting a date from the calendar. Click the Search button to see all Transfer Requests submitted between the dates entered.

Figure 6

Finally, you may also limit the results of your search by using the Status of the Transfer Request. Select Open from the Status drop-down list to view only Transfer Requests which have not been completed. Select Closed from the Status drop-down list to view only Transfer Requests which have been completed. Select All from the Status drop-down list to view all Transfer Requests regardless of their status.
