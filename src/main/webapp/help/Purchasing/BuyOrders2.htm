Buy Orders

+ Larger Font | + Smaller Font

Buy Orders are generated by tcmIS as the result of a Material Request by the client for an Order On Request item or to restock the warehouse when on hand quantities for a Min/Max item fall below the minimum stocking level. The Buy Order directs the buyer to obtain material via Purchase Order to a supplier.

Buyers are responsible for securing materials at competitive prices while meeting all quality requirements of the client. The Procurement Department has a goal of 2 days for placing a Purchase Order once a Buy Order is received.

Information on Buy Orders can be found from the Supply Chain menu under Purchasing. Select Buy Orders to display the Buy Orders page. The Buy Orders page should be checked daily by Purchasing Staff. All Purchase Orders must be created from the Buy Orders page.

The Buy Orders page can be used to locate Buy Orders and related information and create Purchase Orders. There are many search parameters that can be used to locate a specific Buy Order.

If searching for Buy Orders related to a specific Hub, select that Hub from the drop-down box labeled Hub. If the Hub is not known or if searching for Buy Orders for all Hubs, select All from the Hub drop-down box.

If searching for Buy Orders related to a specific client, select that client from the list box labeled Company. If the client is not known or if searching for Buy Orders for all clients, select All from the Company drop-down box.

If searching for Buy Orders related to a specific inventory group, a Hub must be selected from the drop-down box labeled Hub. Choose the desired Inventory Group from the Inventory Group drop-down box. If the inventory group is not known or if searching for Buy Orders for all inventory groups, select All from the Inventory Group drop-down box.

If searching for Buy Orders related to a specific facility, a Hub and inventory group must be selected. Choose the desired facility from the Facility drop-down box. If the facility is not known or if searching for Buy Orders for all facilities, select All from the Facility drop-down box.

If searching for Buy Orders related to a specific buyer, select the buyer's name from the drop-down box labeled Buyer. If the buyer is not known or if searching for Buy Orders for any buyer, select All Buyers from the Buyer drop-down box.

The Search drop-down list may be used in conjunction with the text box labeled For to locate Buy Orders for which certain information is known. If the client's internal purchase order number is known, select Customer PO from the Search drop-down box and enter the customer's PO number in the For text box. If the Item ID is known, select Item ID from the Search drop-down box and enter the Item ID in the text box labeled For. If a Purchase Order has already been associated with a buy order and the Haas TCM Purchase Order number is known, select PO from the Search drop-down list and enter the Purchase Order number in the For text box. If the Material Request ID is known, select MR from the Search drop-down list and enter the Material Request ID in the text box labeled For. If the manufacturer of the material is known, select Manufacturer from the Search drop-down list and enter the manufacturer name in the For text box. If the Buy Order number is known, select PR Number from the Search drop-down list and enter the Buy Order number in the text box labeled For. If the client's part number is known, select Part Number from the Search drop-down list and enter the part number in the For text box. If the last supplier is known, select Last Supplier from the Search drop-down list and enter the supplier name in the For text box. You may want to change the box which currently displays "is" to "contains","starts with", or "ends with" to accommodate your search criteria.

If the status of the Buy Order is known, or if looking for only Buy Orders with a certain status, select a status from the Status drop-down list. Choose All Statuses if the status is not known.

Select a supply path from the drop-down box labeled Supply Path. Select Purchasing to find all Buy Orders that follow the normal purchasing supply path. Select Dbuy to find all Buy Orders that are processed through the Direct Buy supply path. To find Buy Orders with any supply path, select All from the Supply Path drop-down list.

Select a field to sort by from the Sort drop-down list. Buy Orders will be sorted according to this selection.

Select the check box labeled Only Unconfirmed POs to search for only Buy Orders that have unconfirmed Purchase Orders attached to them.

Select the check box labeled Only With No assigned PO#s to search for only Buy Orders that do not yet have Purchase Orders attached to them.

Select the check box labeled Only Confirmed PO#s to search for only Buy Orders that have confirmed Purchase Orders attached to them.

Select the check box labeled PO Confirmed but Association not Confirmed to search for only Buy Orders that have confirmed Purchase Orders attached to them, but the association with the Buy Order has not been confirmed.

Once all desired search parameters have been set, click the Search button.

To create a Purchase Order, select the Buy Order by checking the box labeled Grab. Then click the Create PO for supplier link above the data grid. A tcmIS PO is created. If the material has been purchased before, the last supplier, price, shelf life requirements, and specification requirements will be automatically completed. For first time purchases this information must be completed as described on the Purchase Order page help.

Figure 14

To create a PO with multiple Buy Orders attached check the grab button for all lines to include on the PO. Note that the preferred supplier is displayed on the Buy Orders page. When a Buy Order is selected to be included on a PO, the selected supplier is displayed. The selected supplier is by default the same as the preferred supplier for the first Buy Order grabbed. Click the More button to see alternate suppliers and pricing, if any.

After checking the Grab box for all Buy Orders to include on the PO, click the Create PO for supplier link. For more information on completing Purchase Orders see the Purchase Orders help page: https://www.tcmis.com/tcmIS/help/Purchasing/Purchasing.htm.
