Buyer Item Notes

+ Larger Font | + Smaller Font

Purchasing notes can be associated with an Item ID. These notes are displayed on the Purchase Order page when a PO is generated for the Item ID. The Buyer Item Notes page can be used to view and edit item notes. The Buyer Item Notes page can be found from the Supply Chain menu under Purchasing. Select Buyer Item Notes. The Buyer Item Notes page is launched in a new tab.

Figure 1

Enter the Item ID in the corresponding text box and click Search. All notes for the item ID are displayed. Enter a new note by typing in the Comments box and clicking the Update link above the data grid.

Figure 2

Delete old comments by checking the box in the Delete column and clicking the Update link above the data grid. Please be respectful of other people's data and delete only your own comments.
