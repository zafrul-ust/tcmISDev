Supplier Search

+ Larger Font | + Smaller Font

The Supplier Search page is used to look up supplier information in tcmIS. All new supplier requests should be started by first checking to see if the supplier already exists in tcmIS by using the Supplier Search page. The Supplier Search page can be accessed from the Supply Chain menu. Under Purchasing, select Supplier Search. The page is launched in a new tab.

Figure 1

You may also access the Supplier Search page from the Reports menu by selecting Purchasing and Supplier Search.

Figure 2

Enter the supplier name in the Search Text box. Select the country from the corresponding drop-down list or select All to search all suppliers worldwide.

Check the box labeled Active Only to eliminate inactive suppliers from the results.

Click Search. All suppliers that match the search criteria are displayed.

Figure 3

Right-click on a supplier and select Supplier Contact to display contact information for the supplier.

Figure 4

All available contact information for the selected supplier is displayed.

Figure 5

To request a new remit to address for an existing supplier, right-click on the supplier name and select New Remit To Address.

Figure 6

Enter the required address information and click the Okay button.

Figure 7

To request a change in the supplier's default payment terms, right-click on the supplier and select Request Change Payment Terms.

Figure 8

Select the new payment terms from the Default Payment Terms drop-down list. Enter notes explaining the need to change the default payment terms in the Notes box. Click the Submit link at the top left of the page to submit the request.

Figure 9

To change or modify other supplier information, right-click on the supplier and select Modify Supplier.

Figure 10

Update the necessary data and click the Submit link in the top left of the page. More details on the fields on this page can be found in the New/Modify Supplier Request help page.

Figure 11

To create a new supplier, first search the Supplier Search page to be sure the supplier does not already exist. Click the New button in the header.

Figure 12

The New Supplier Request page is displayed. Complete the required information and click the Submit link in the top left of the page. More details on the fields on this page can be found in the New/Modify Supplier Request help page.
