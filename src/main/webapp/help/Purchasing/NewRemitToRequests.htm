New Remit To Requests

+ Larger Font | + Smaller Font

The New Remit To Requests page is used to view requests for new supplier remit to addresses in tcmIS and assign or create new vendor codes for each remit to address. The New Remit To Requests page can be accessed from the Supply Chain under Purchasing. Select New Remit To Requests. The New Remit To Requests page is launched in a new tab.

Figure 1

You may search for a specific supplier by entering the supplier name in the Search text box. It is recommended to select contains, starts with, or ends with if the exact supplier name is not entered. Leave the Search box blank to view all requests for new remit to addresses. Click Search.

To reject a request to add a new remit to address for a supplier, right-click on the request line and select Reject.

Figure 2

To look up an existing vendor code, click the button in the SAP Vendor Code column. Enter the vendor name in the Search box and click Search. If the vendor already exists, click the line and click the Return Selected Data link.

Figure 3

The SAP Vendor Code is populated on the New Remit To Requests page. Check the OK box and click the Update link to associate this remit to request with the selected SAP Vendor Code.

Figure 4

If a new SAP Vendor Code is needed, check the box in the New column and click the Update link. This will assign the next valid SAP Vendor Code to the supplier remit to address.

Figure 5

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 6

You may resize any column by clicking and dragging the column dividers.

Figure 7

The data on the New Remit To Requests page can also be sorted by certain columns by clicking the column name. The results will be sorted by the selected column in ascending order. Click the column name again to sort in descending order.

Figure 8

Click Create Excel File to export the data on the New Remit To Requests page to Excel.
