Printed PO List

+ Larger Font | + Smaller Font

The Printed PO List page is used to see when a purchase order was printed and view the image that was printed at that time. The Printed PO List page can be accessed from the Supply Chain Menu. Under Purchasing, select Printed PO List. The page is launched in a new tab.

Figure 1

Enter a purchase order number in the text box and select PO Number from drop-down box. Click Search. Each printing of the purchase order is listed. Click the link in the PO/BO column to view the image that was printed.

Figure 2

Enter a buy order number in the text box and select BO Number from drop-down box. Click Search. Each printing of the purchase order which the buy order is associated with is listed. Click the link in the PO/BO column to view the image that was printed.
