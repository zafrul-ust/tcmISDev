New Supplier Request

+ Larger Font | + Smaller Font

The New Supplier Request page is used to enter new suppliers in tcmIS. The New Supplier Request page can be accessed from the Supply Chain under Purchasing. Select New Supplier Request. The New Supplier Request page is launched in a new tab.

Figure 1

Click New to enter a new supplier. Please note that you should use the Supplier Search page to determine if the supplier already exists in tcmIS prior to adding a new supplier.

Figure 2

Enter all required information related to the supplier. Required fields are denoted by an asterisk. Required information includes the supplier name, address and phone number. Optional information includes information related to the main contact person at the supplier location.

Figure 3

Although the Federal Tax ID is not denoted with an asterisk, it is a required fields. For suppliers outside the U.S., use this field to enter the VAT number. The State Tax ID is optional and should be completed for suppliers in the U.S. if the number is available from the supplier.

Select the default payment terms for the supplier. If the payment terms needed are not listed, contact the Technical Center.

Select the type of purchase. This is typically Chemical. Other options include Waste, Gas, Computer Hardware, and Other.

Select the supplier's qualification level. New suppliers should be set to New. If the customer has directed Haas to purchase from this supplier select Customer Directed-New.

If the supplier is a Small Business, Minority Owned, Disadvantaged, or Women Owned, select the corresponding check boxes.

If the supplier has issued Haas TCM an account number, enter it in the box labeled Haas Account Number.

If this supplier replaces a previous supplier due to a name change, enter the old name in the box labeled Former Supplier Name.

Enter any notes directed to new supplier approvers that should be included based on your team's specific work processes.

After all required information has been completed, click the Submit button at the top of the screen.

Figure 4

A new supplier request will currently be sent to two approvers before being added to tcmIS. Approvers will be sent an email asking them to review and approve the supplier. Currently, there are two approval roles for all new suppliers. The first approval role is Payment Terms, approved by Haas TCM Sourcing group The second approval role is a general supplier approval and is performed by the Haas TCM Finance Group.

To locate all supplier requests that are pending Payment Terms approval, select Status Pending Payment Terms and click List. To locate all supplier requests that are pending general approval, select Status Pending Approval and click List.

Figure 5

To approve a request for a new supplier, click the Request ID link. Click Approve or Reject. You must be a designated approver to perform this action. Contact the Technical Center to add or change approvers.

Figure 6

An email will be sent to the requestor when a request is approved or rejected. If the supplier is approved, it is now possible to add the supplier to a Purchase Order.
