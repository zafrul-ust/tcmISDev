Price List

+ Larger Font | + Smaller Font

The Price List page is used to view, edit, and create customer price lists. If a material is on a customer price list and the price list has been assigned to the selected ship to location, the price will be automatically populated when added to a Scratch Pad, Quote, or Material Request. The Price List page can be accessed from the Distribution menu by selecting Price List. The Price List page is launched in a new tab.

Figure 1

To see all Parts in a Price List, select the Operating Entity and the Price List from the corresponding drop-down lists. Click the Search button.

Figure 2

To see how a part is defined across all Price Lists for an Operating Entity, select All from the Price List drop-down. Select Part and contains from the Search drop-down boxes. Enter the Part Number or part of the Part Number in the text box. Click Search.

Figure 3

If the Part Number is not known, select Part Desc and contains from the Search drop-down boxes. Enter a description of the material in the text box. Click Search.

Figure 4

Check the box labeled Show Expired to see expired pricing only.

You may update the Baseline, Currency ID, or Unit Price. Click the Update link to save any changes.

Figure 5

To add a Part to a Price List, select the Price List and search to make sure the item is not already included in the price list. Click the link labeled Add Part To Price Group.

Figure 6

Locate the material to add to the Price List. You may search by Part Description, Part Number, Text (Material Notes), or Synonym (legacy Part Number). Click on the group of specifications to which to price applies. Select the currency and click the link labeled Add Catalog Item.

Figure 7

The selected Item and Spec Group is added to the Price List. Select the date that the price begins by clicking in the Start Date text box and selecting a date from the calendar. Enter a baseline price in the Baseline column, if known. Select the currency from the drop-down labeled Currency ID. Enter the price in the Unit Price column. Click the Update link to save any additions or changes.

Figure 8

You may enter a new start date and price for Items already in the price list by right clicking on the line and selecting New Start Date.

Figure 9

A new row is added for the selected Item. Enter the start date for the new price by clicking in the Start Date text box and selecting a date from the calendar. Enter a baseline price in the Baseline column, if known. Select the currency from the drop-down labeled Currency ID. Enter the price in the Unit Price column. Click the Update link to save any additions or changes.

Figure 10

To delete a start date that has not yet passed, right-click on the line and select Delete Start Date. Click the Update link to save any changes.

Figure 11

To enter a price break, right-click on an existing line in the Price List for the Item and select New Price Break.

Figure 12

Enter the quantity at which the price break applies and the price. You may repeat this process to create as many price breaks as needed. Click the Update link to save any changes.

Figure 13

New Price Lists can be added by clicking the New button on the Price List page.

Enter a short name for the price list in the Price List Name field. Enter a description of the price list in the Price List Desc field.

Select the Operating Entity that owns the Price List from the drop-down box labeled Controlling Operating Entity.

Select the default currency for the price list from the corresponding drop-down list.

Enter the date that the Baseline Prices should be reset, if applicable, by clicking in the Baseline Reset Date box and selecting a date from the calendar.

Click Submit.

Figure 14

Add items to the new price list as described above.
