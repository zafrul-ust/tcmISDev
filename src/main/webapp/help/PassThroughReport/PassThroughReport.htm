Pass Through Report

+ Larger Font | + Smaller Font

The Pass Through Report page displays invoice details for counting inventory groups regarding pass through at cost materials. The Pass Through Report page can be accessed from the Reports menu by selecting Cost Report. Then select Pass Through Report. The Pass Through Report page is launched in a new tab.

Figure 1

Select the hub and inventory group from the corresponding drop-down boxes. Enter a date in the Used Since box to report all pass through items used since the date entered. Click the blue diamond to select the date from the calendar. Results may be sorted by Use Date and Customer PO, Item ID and Invoice Date, Item ID and Use Date, Part Number and Invoice Date, or Part Number and Use Date by selecting the desired sort option from the Sort By drop-down list.

Click Search to display all pass through items used. Export the results to Excel by clicking Create Excel File.
