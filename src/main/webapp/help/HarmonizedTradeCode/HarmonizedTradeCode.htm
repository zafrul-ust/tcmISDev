Harmonized Trade Code

+ Larger Font | + Smaller Font

The Harmonized Trade Code page provides information regarding the Harmonized Trade Code associated with tcmIS items. This page also facilitates the entry of Harmonized Trade Codes. The Harmonized Trade Code page can be accessed from the Reports menu by selecting Item and then Harmonized Trade Code. The Harmonized Trade Code page is launched in a new tab.

Figure 1

You may search for items using a description of the product. Select Item Desc from the Search drop-down list, select you search condition, and enter the item description in the text box.

Check the box labeled Show Items Without Harmonized Code Only to display only items which have not been assigned a Harmonized Trade Code in tcmIS.

Click the Search button.

Figure 2

You may also search for a specific item ID by selectingItem ID from the Search drop-down list and entering the item ID in the text box. Click Search.

Figure 3

Select the appropriate code from the Harmonized Trade Code drop-down list. Check the OK box and click the Update link to associate the code with the selected item ID in tcmIS.

Figure 4

Click the Create Excel button to export the results into Excel format.
