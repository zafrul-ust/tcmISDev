Customer Return Tracking

+ Larger Font | + Smaller Font

The Customer Return Tracking page is used by Haas TCM Distribution Managers to view and approve Customer Return Requests. The Customer Return Tracking page can be accessed from the Distribution menu by selecting Customer Return Tracking.

Figure 1

Select the Operating Entity and Hub from the corresponding drop-down boxes. Select an Inventory Group from the Inventory Group drop-down list to see only Customer Return Requests for the selected Inventory Group. Select My Inventory Groups to see all Customer Return Requests for Inventory Groups at the selected Hub for which you have access.

Select a status from the Status drop-down list to see all requests in the selected status. SelectAll to see all requests. Select Pending Approval to see all requests that have been submitted but not yet approved. Check the box labeled All Requests within __ days to see all requests entered in the number of days entered. Click Search.

Figure 2

To locate Customer Return Requests for a specific Material Request, select MR from the Search drop-down lists and enter the MR number in the Search text box. Click Search.

Figure 3

To locate Customer Return Requests for a specific Customer, select Customer and contains from the Search drop-down lists and enter a portion of the customer name in the Search text box. Click Search.

Figure 4

To locate Customer Return Requests for a specific customer PO number, select Customer PO and contains from the Search drop-down lists and enter a portion of the customer PO number in the Search text box. Click Search.

Figure 5

To locate Customer Return Requests using RMA number, select RMA and is from the Search drop-down lists and enter the tcmIS RMA number in the Search text box. Click Search.

Figure 6

To view or approve the Customer Return Request right-click on the line and select View RMA.

Figure 7

If you are an approver you may edit the request. Scroll down to the Approval Detail section of the request. Enter the quantity approved for return in the Return Quantity Authorized box.

You may add fees like restocking fees by clicking the Add Line link above the section labeled New Fees. Remove charges by clicking on the line to remove and then clicking the Remove Line link. For more information on adding charges see the Additional Charges help page: https://www.tcmis.com/tcmIS/help/OrderEntry/AdditionalCharges.htm.

Any header charges or line additional charges that have been invoiced are displayed in the Original Order Fees box. Any of these fees which need to be credited should be added under the New Fees section as negative amounts.

Figure 8

Scroll to the top of the request and click the Approve link to approve the request. If material is requested to be shipped back to the warehouse, it will appear on the receiving page.

Click Deny to reject and cancel the request. Click Save to save any changes without approving or rejecting.

Figure 9

Click the Print link to print a copy of the RMA Authorization which can be provided to the customer.
