Customer Return Request

+ Larger Font | + Smaller Font

The Customer Return Request process is used by Haas TCM staff to initiate a return from a Distribution customer in tcmIS. To begin the Customer Return Request process, first look up the Material Request which the customer would like to return. Use the Order Entry Lookup screen to locate and open the MR. For more information on the Order Entry Lookup process see the Order Entry Lookup help page: https://www.tcmis.com/tcmIS/help/OrderEntry/OrderEntryLookup.htm.

Figure 1

The MR line must be delivered in order to begin a Customer Return Request. Right-click anywhere on the MR line and select Customer Return Request.

Figure 2

Enter the contact information for the person who is requesting the return in the Return Requestor, Email, and Phone fields.

Click the Add Line link and select the reason for the return. You may add as many lines as needed to explain the reason for the return. To remove a line, click anywhere on the line and click the Remove Line link.

Enter the quantity the customer has requested to return in the text box labeled Return Quantity Requested.

If the customer is asked to keep the material select Yes to the right of the question Do we want the to keep the material? If the material should be returned to the warehouse select No.

If the warehouse shipped the Item that was ordered select Yes to the right of the question Did we ship the correct item? If the warehouse shipped a different item than was ordered select No.

Figure 3

If a different item was shipped, click the Add Line link beneath the question. Locate the Receipt ID that was actually shipped. This may require inspection in the warehouse and/or information from the customer to determine which Receipt ID they received which is still showing as in stock in tcmIS. You may search by Receipt ID, Manufacturer's Lot Number, Item ID, or Item Description. Choose your search option from the Search drop-down list. Select is or contains and enter your search criteria. ClickSearch.

For each Receipt ID shipped enter the quantity the customer received in the Quantity Shipped column and check the OK box. Click the Return Receipts link.

Figure 4

The page will remain open but the Receipt is added to the Customer Return Request page. You may add as many Receipt lines as needed. To remove a line, click anywhere on the line and click the Remove Line link.

Figure 5

If the customer wants replacement material select Yes to the right of the question Does Customer want replacement? in the Replacement Detail section. If the customer does not want replacement material select No.

Figure 6

If replacement material is required and it is a different material than originally ordered click the to the right of the Part Number. Lookup the replacement item. For more information on looking up items, see the Add Line help page: https://www.tcmis.com/tcmIS/help/OrderEntry/AddLine.htm.

Figure 7

If replacement material is needed, enter the new price in the text box labeled New Unit Price.

Enter the customer's need date in the Need Date box by clicking in the box and selecting a date from the calendar. Enter your promised ship date in the Promised Date box by clicking in the box and selecting a date from the calendar.

Enter any notes in the Comments text box.

You may add fees like restocking fees by clicking the Add Line link above the section labeled New Fees. Remove charges by clicking on the line to remove and then clicking the Remove Line link. For more information on adding charges see the Additional Charges help page: https://www.tcmis.com/tcmIS/help/OrderEntry/AdditionalCharges.htm.

Any header charges or line additional charges that have been invoiced are displayed in the Original Order Fees box. Any of these fees which need to be credited should be added under the New Fees section as negative amounts.

Figure 8

Click the Save link to save the request without submitting it.

Click the Submit link to submit the request for approval. Approvers will be notified and will review and approve the request using the Customer Return Tracking page.

Click the Delete link to delete requests that have not been submitted. Deleted requests can not be retrieved.
