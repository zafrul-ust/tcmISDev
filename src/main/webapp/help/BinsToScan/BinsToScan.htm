Bins to Scan

+ Larger Font | + Smaller Font

The Bins to Scan page is used to print lists of bins to include in warehouse inventory scans. The Bins to Scan page can be accessed by selecting Stocking Room Inventory from the Supply Chain menu. Select Reconciliation then Bins to Scan. The Bins to Scan page is launched in a new tab.

Figure 1

Select the hub and room from the corresponding drop-down boxes. Click Search to display all bins for the selected hub and room. Click Create Excel File to export the data on the Bins to Scan page to Excel. Print the Excel file and use it to scan the listed bins using TCM Scan. TCM Scan is a proprietary software application which runs on Portable Data Terminals (PDTs) equipped with bar code scanners. See the Inventory Count Upload help page for more information on inventory scans.

Figure 2

Enter a number of days in the text box labeled Receipts issued in the last __ days to display only bins where material has been issued in the number of days entered. Click Search.

Figure 3

Enter a number of days in the text box labeled Bins that have not been counted in the last __ days to display only bins where material has been scanned in the number of days entered. Click Search.

Figure 4

Enter an Item ID in the text box labeled Item to display only bins that have contained the Item ID. Click Search.

Figure 5

Enter dollar values in the text boxes labeled Items with Unit Cost > $ __ and < $ __ to display only bins that contain materials with a unit cost within the entered range.

Figure 6

Enter dollar values in the text boxes labeled Items with Inventory Value > $ __ and < $ __ to display only bins that contain materials with inventory value within the entered range.
