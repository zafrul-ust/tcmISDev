The Item Count page is used to enter inventory counts into tcmIS for counting inventory groups. Haas TCM owned material, client owned material, and consigned material should all be included in inventory counts. The Item Count page can be accessed from the Supply Chain menu by selecting Stock Room Operations. Then choose Counting and Item Count.

Figure 1

To enter a count, select the desired hub from the Hub drop-down list. Choose the plant from the drop-down list labeled Inv Grp. Click Start New Count.

Figure 2

Please note that when a new count is started, the Count Start Date is displayed above the materials to be counted. Verify that this date and time is current. If it is in the past, click Close and then Start New Count.

If there are any black lines displayed, click Close and then Start New Count. Black lines indicate that there has been a receipt made since the count was started. This means the on hand quantity displayed on the page is not accurate and needs to be refreshed by starting a new count.

The date the inventory count was taken should be entered in the text box labeled Actual Count Date.

Figure 3

All materials which need to be counted will be displayed. These include all MM items and any OORs which have on hand inventory. The quantity on hand as of the last count plus any receipts is displayed in the Qty column. Enter the actual quantity remaining on hand in the Actual Count column. It is important to enter your actual count quantity in terms of the unit of measure displayed in the Count UOM column. If you would like to change the Count UOM for counting, please contact the Technical Center for assistance.

Click Add Item to add OOR items to the count list.

When done entering counted quantities, click Process. To close the count without processing the data, click Close.

An Excel spreadsheet of the count data may be created by clicking Create XLS.
