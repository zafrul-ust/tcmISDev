MR Release

+ Larger Font | + Smaller Font

The MR Release page is used to view Material Requests which are pending release for Distribution Operations. Cash Sales, Credit Card Sales, and Inter-Company Sales need to be released using the MR Release page. MRs on Quality Hold can also be reviewed and release from the MR RElease page. The MR Release page can be accessed from the Distribution menu by selecting MR Release. The MR Release page is launched in a new tab.

Figure 1

Select the Operating Entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drops-down lists. Click Search to see all Material Requests pending Release.

Figure 2

You may limit the results by the date the MR was confirmed. Click in the Confirm Date text boxes and select dates from the calendar. Click Search.

Figure 3

To limit the results to MR entered by a specific person, possibly yourself, click the to the right of the Entered By field.

Enter the the name you are looking for in the Search text box and click the Search button. Select the desired user by clicking anywhere on the line. Click the Return Selected User link.

Click the Clear button to see all MRs regardless who entered it.

Figure 4

You may search for Material Requests for a specific customer by clicking the labeled Customer. Use the Customer Search page to look up the customer and populate the Customer field on the Order Entry Lookup page. Click Search.

For more details on the Customer Lookup process, see the Customer Lookup help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerLookup2.htm.

Click the Clear button to see MRs pending release for any customer.

Figure 5

To limit the results further, you may select a status from the Status list box. Select All to see MR pending release. You may select multiple statuses by holding CTRL and clicking on multiple statuses. Click Search.

Figure 6

If known, you may enter the MR Number in the Search box. Click the Search button to locate the specified MR.

Figure 7

To view the details of a Material Request, right-click on the line and select Open MR. The MR is opened in a new tab. For MRs pending quality hold, open the MR to view more details.

Figure 8

If the MR is on Quality Hold, scroll to the far right of the MR Release page to view which MR line is on quality hold. For each line, an X in the Spec column indicates the material allocated to this line has arrived without the proper certifications. An X in the SL column indicates the material allocated to this line has arrived without the proper shelf life remaining.

To release an MR line from Quality Hold, right-click on the line and select Release Line.

Figure 9

To enter or edit the Note column, right-click on the line and select Create/Edit Line Note.

Figure 10

A Pro Forma invoice can be printed by right-clicking on a MR and selecting Print Pro Forma.

Figure 11

To approve and release an MR pending any type of Finance Hold (Credit Hold, Pending CC Auth, Pending Cash, etc.), right-click on the anywhere on the MR line and select Release MR.

Figure 12

To reject and cancel the MR, right-click on the anywhere on the MR line and select Cancel.

Figure 13

Click Create Excel to export the data on the MR Release page in Excel format.
