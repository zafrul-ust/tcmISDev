Daily Inventory Report

+ Larger Font | + Smaller Font

The Daily Inventory Report page can be used to view the inventory quantity and value at a hub for any date. The Daily Inventory Report page can be accessed from the Reports menu by selecting Inventory. Then select Daily InventoryReport. The Daily Inventory Report page is launched in a new tab.

Figure 1

Select the hub and inventory group from the corresponding drops down lists. Select the date by clicking in the On Date text box and selecting a date from the calendar. Click Search.

Figure 2

All inventory that was on hand on the give date is displayed. The Material Value columns represent the cost of the material not including additional charges like freight, certification fees, etc. The Full Value columns represent the cost of the material including all associated charges.

Material Value and Full Value columns are each further subdivided into Haas, Consigned, Customer, and Total. The Haas column represents the value of Haas owned material in the warehouse. The Consigned column represents the value of inventory in the warehouse owned by suppliers and sold to Haas on consignment. The Customer column represents the value of customer owned material in the warehouse (often zero value). The Total column is a sum of the Haas, Consigned and Customer columns.

Click the Create Excel File button to export the displayed data to Excel.
