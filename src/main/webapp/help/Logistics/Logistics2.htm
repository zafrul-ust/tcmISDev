Logistics

+ Larger Font | + Smaller Font

The Logistics page is used to manage inventory in the Haas TCM warehouse. This page can be used to change the lot status of a material, print labels, change the expiration date of a material, and view a variety of other information regarding the state of a material. The Logistics page can be accessed from the Supply Chain menu by selecting Stock Room Inventory. Select Inventory then Logistics.

Figure 1

Select a Hub from the Hub drop-down box. Click Search to display all receipts and their current status. If a specific receipt is not listed, change the number of days in the Exp Within text box to a larger number and click Search.

The results may be sorted by Item/Status/Receipt ID, Bin/Receipt ID, Owner/Bin/Receipt ID, Receipt ID, or Age/Status/Item. Select the desired sort option from the Sort By drop-down list. The default is to sort by Item/Status/Receipt ID.

Figure 2

To locate materials for a specific Inventory Group, select the desired Inventory Group from the Inv Grp drop-down list and click Search.

Figure 3

A specific receipt can be located on the Logistics page by entering the Receipt ID in the text box labeled For and by setting the Search drop-down lists to Receipt ID and Equals. Click Search.

Figure 4

To locate all materials in a specific bin, enter the Bin name in the text box labeled For and set the Search drop-down lists to Bin and Equals. Click Search.

Figure 5

To locate al materials with a specific word in the item description, enter the key word in the text box labeled For and set the Search drop-down lists to Item Desc and contains. You may use the percent sign (%) as a wildcard in your search string. Click Search.

Figure 6

To locate all materials with a specific Haas TCM item ID, enter the item ID in the text box labeled For and set the Search drop-down lists to Item ID and Equals. Click Search.

Figure 7

To locate all materials with a specific Lot Status, enter the Lot Status in the text box labeled For and set the Search drop-down lists to Lot Status and Equals. Click Search.

Figure 8

Alternatively, you may select a lot status from the list box labeled Lot Status. Select more than one lot status by holding CTRL and clicking multiple statuses. Click Search to see all inventory in the selected statuses.

Figure 9

To locate all materials from a specific Manufacturer's Lot, enter the lot number in the text box labeled For and set the Search drop-down lists to Mfg Lot and Equals. Click Search.

Figure 10

A material with a specific Original Receipt ID can be located on the Logistics page by entering the Original Receipt ID in the text box labeled For and set the Search drop-down lists to Orig Receipt ID and Equals. This is useful if a material is returned or transferred. When this happens, the material a new Receipt ID is created but the Original Receipt ID is preserved and can be used to locate the material on the Logistics page.

Figure 11

To locate material for a specific Purchase Order, enter the PO Number in the text box labeled For and set the Search drop-down lists to PO and Equals. Click Search. The Haas PO Number and line on which the material was received is displayed on the far right of the Logistics page.

Figure 12

For operations that have been migrated to tcmIS from another legacy system, your Legacy Receipt ID can be used to locate receipts in tcmIS. Enter the Legacy Receipt ID in the text box labeled For and set the Search drop-down lists to Legacy Receipt ID and contains. Click Search. The Legacy Receipt ID is displayed on the far right of the Logistics page.

Figure 13

To display only Receipts which need labels printed, select the check box labeled Show Only Receipts to be Printed. Click Search.

Figure 14

To display information about materials that are no longer in inventory, select the check box labeled Include History. Click Search. The quantity in inventory is displayed in the column labeled Quantity. Directly below the quantity on hand is the quantity received.

Figure 15

To view and edit an item's DOT shipping name and other shipping information, click the link in the Item column. The DOT Shipping Name page is displayed.

Figure 16

To view all transactions associated with a Receipt ID, click the link in the Receipt ID column. The Transactions page is displayed.

Figure 17

Click the link in the Quantity column (if there is a positive quantity available) to spilt the receipt. It may be necessary to spilt a receipt if the some of the items received need to be assigned different lot statuses. For instance, if one of the 3 items received under Receipt ID 369492 was damaged transit, but the other 2 were available, it would be necessary to split the receipt in order to assign the proper lot status to the correct materials.

Enter the Qty to Split and select the new lot status for the split item from the Status drop-down list. Change the inventory group and bin, if necessary. Enter the Quantity to Split. Click OK.

Figure 18

You may request that a receipt be written off by changing the Lot Status to Write Off Requested. Spilt off a portion of the Receipt as described above if only some of the Receipt needs to be adjusted. Select the root cause and responsible company for the write off request from the corresponding drop-down boxes. Enter any notes in the Justification text box and click OK. Click the Update button on the Logistics page to save the Write Off Request.

Figure 19

You may write on or increase the quantity of a Receipt ID from the Logistics page by clicking the + symbol under the Quantity. Write On Requests will be sent to the Inventory Adjustment page for approval.

Figure 20

Enter the quantity to add to the Receipt ID and enter comments to explain why the material is being added. Click the OK button. Write On Requests will be sent to the Inventory Adjustment page for approval.

Figure 21

A summary of the data on the Logistics page can be created in Microsoft Excel format by clicking Create Excl File.

To print container labels for a receipt, select the material by checking the OK box in the far left column of the Logistics page. Adjust the number of labels to be printed, if necessary, by entering a number in the #Labels text box. Choose the label size and click Container Labels.

To print all label types defined for the customer, click Generate All Labels.

To update the lot status of a receipt, select the new lot status from the Status drop-down list. Click Update.

The shelf life of a material can be updated by entering an expiration date in mm/dd/yyyy format in the Exp Date text box. Click Update.

In addition, clicking the blue diamond to the right of the date will display a calendar. Select the proper month and year from the drop-down lists at the top. Choose a date by clicking on the desired date on the calendar. Click Update.

Figure 22

To change the location of a receipt, select the bin where the material is to be stored from the Bin drop-down list. Click Update.

The drop-down list in the Bin column lists all the bins in which this material has previously been stored. If the bin where the material is to be stored is not on the drop down menu, click the + to the left of the drop-down list. The Pick New Bin page will appear. This page displays a list of all bins in the hub. Select the location of the bin and click OK. The bin will be added to the drop-down list on the Logistics page.

Figure 23

If the bin is not on the Pick New Bin page, it is necessary to create a new bin. To create a new bin, click the Add New Bin button near the top of the screen. The Add New Bin page will appear. Select the hub where the bin is located from the Hub drop-down list and enter a name for the bin. Be sure the At the hub check box is selected. Click OK. The bin will be added to the Bin drop-down list on the Logistics page.

Figure 24

To change the manufacturer's lot number, enter the new lot number in the Lot text box. Click Update.

To update the number of recertifications, enter a number in the Recert# text box. Click Update.

Notes can be added to the Logistics page by clicking . The Receipt Notes page is displayed.

Figure 25

Enter any notes and click Add.

Uploading Documents

You may view or upload a scanned image of the Receiving Jacket from this screen. To view a document on the Logistics page, click the icon. All uploaded receipt documents are listed. Click the link in the Name field to display the document. You may also add new documents from here by clicking Add New Document. You may delete documents by checking the Delete box and clicking Update.

Figure 26

To upload new documents if there are no existing documents, click the + symbol to display the Receipt Documents page. Click Add New Document.

Figure 27

Select the company for which the receipt was made to make these documents visible to the customer. For CMS Operations, If the receipt could be allocated to multiple customers, you must upload the documents for each customer that needs to see the delivery documents online. For Distribution Operations, select ALL COMPANIES. Selecting Haas TCM prevents the customer from seeing the documents and allows only Haas TCM employees to view the documents. Enter a name for the Document in the Doc Name field. Click Browse and locate the file on your computer. Select the Doc Type that best describes the document you are uploading. Click Save to upload the document to tcmIS.
