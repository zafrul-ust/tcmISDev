Reconciliation

+ Larger Font | + Smaller Font

The Reconciliation page facilitates counting and recording discrepancies between physical inventory and tcmIS. This page is typically used when performing yearly physical inventory counts or during periodic cycle counts. Discrepancies reported on the Reconciliation page are sent to approvers to release from the Inventory Adjustments page. For more information on approving Inventory Adjustment requests see the Inventory Adjustments help page: https://www.tcmis.com/tcmIS/help/InventoryAdjustment/InventoryAdjustment.htm. The Reconciliation page can be accessed from the Supply Chain menu by selecting Stock Room Inventory. Choose Reconciliation. Then select Reconciliation. The Reconciliation page is launched in a new tab.

Figure 1

Select the Operating Entity, Hub and Inventory Group for which inventory reconciliation is to be performed. You may select My Inventory Groups in the Inventory Group list to see all inventory in the Hub. Select one or more inventory groups by clicking Inventory Group names while holding down the CNTRL key. Click Start New Count or select a date from the Count Date box.

It is important to start a new count if the date displayed is old. Starting a new count refreshes the on hand inventory quantity displayed.

Figure 2

Select the Count Type. Choose All Inventory to create a count list for all items in inventory for the selected Inventory Groups. Select Items to Reconcile to create a count list for all items that have been marked for reconciliation. Choose Receipts to Reconcile to create a count list for all Receipt IDs that have been marked for reconciliation. Choose Single Item and enter the Item ID to create a count list for a specific Item ID. Click the OK button.

Figure 3

All Receipt IDs matching your search criteria are displayed. Enter the actual on hand quantity of each receipt ID in the Actual On Hand column. Click the Update link to save your entries.

Figure 4

The data on the Reconciliation page can be exported into Excel format for printing by click the button labeled Create Excel. These pages can then be taken to the warehouse and used to record the actual inventory levels for entry on the Reconciliation page.

Figure 5

To locate all Receipts for a specific Item ID select Item ID from the Search drop-down list and enter the Item ID in the text box. Click Search

Figure 6

To locate a specific Receipt ID select Receipt ID from the Search drop-down list and enter the Receipt ID in the text box. Click Search

Figure 7

To locate all Receipts using the Item Description select Item Desc and contains from the Search drop-down lists and enter a portion of the Item Description in the text box. Click Search

Figure 8

If you are an approver, check the box labeled Show Problem Receipts Only to see Receipts which have been been counted and the quantity entered does not match the quantity in tcmIS. To approve all of the Inventory Adjustment Requests for the selected count click the Approve link. To approve each request individually, use the Inventory Adjustments page. For more information on approving Inventory Adjustment requests see the Inventory Adjustments help page: https://www.tcmis.com/tcmIS/help/InventoryAdjustment/InventoryAdjustment.htm.

Figure 9

When you are done entering count information click the Close Count button.
