Reconciliation

+ Larger Font | + Smaller Font

The Reconciliation page facilitates counting and recording discrepancies between physical inventory and tcmIS. This page is typically used when performing yearly physical inventory counts or during periodic cycle counts. The Reconciliation page can be accessed from the Supply Chain menu by selecting Stock Room Inventory. Choose Reconciliation. Then select Reconciliation. The Reconciliation page is launched in a new tab.

Figure 1

Select the hub and inventory group for which inventory reconciliation is to be performed. You may select All in the Inventory Group list or select one or more inventory groups by clicking the inventory group names which holding down the CNTRL key. Click Start New Count.

Figure 2

All receipt IDs in inventory are displayed. Enter the actual on hand quantity of each receipt ID in the Actual On Hand column. Click Update to save your entries.

Figure 3

The data on the Reconciliation page can be exported into PDF format for printing by click the button labeled Create PDF. These pages can then be taken to the warehouse and used to record the actual inventory levels for entry on the Reconciliation page.

Figure 4

The data on the Reconciliation page can also be exported into Excel format for printing by click the button labeled Create XLS. These pages can then be taken to the warehouse and used to record the actual inventory levels for entry on the Reconciliation page.
