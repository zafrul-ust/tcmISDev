Expiration Forecast

+ Larger Font | + Smaller Font

The Expiration Forecast page provides information regarding inventory which will expire in six months or less. The Expiration Forecast page can be accessed from the Reports menu by selecting Inventory and then Expiration Forecast. The Expiration Forecast page is launched in a new tab.

Figure 1

Select the Operating Entity from the corresponding drop-down list. Select the Hub and Inventory Group from the corresponding drops-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission.

Select All from the Lot Status list box to included all inventory in the result.

Click Search to find all inventory which is expected to expire in six months or less or has already expired.

Figure 2

You may include multiple lot statuses in the results by holding the CTRL key and clicking multiple items in the Lot Status list box.

Figure 3

You may exclude specific lot statuses from the results by selecting Exclude as opposed to Include.

Figure 4

In the results data grid, the quantity in inventory, shelf life of the item and the expiration date are displayed. The average cost of the item is displayed as well.

The total cost of expired material is shown above the Expired column header. The average cost of each line is shown in the Expired column.

Figure 5

The total cost of material which is forecasted to expire a given month is shown above the column header for each month. The average cost of each line is shown in the column for each month. It is possible to sort by the month by clicking in the header for the month you would like to sort by. Click the header again to sort in descending order.

Figure 6

Click the Create Excel button to export the results into Excel format.
