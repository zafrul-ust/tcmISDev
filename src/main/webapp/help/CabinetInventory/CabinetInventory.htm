Cabinet Inventory

+ Larger Font | + Smaller Font

The Cabinet Inventory page is used to view information regarding on hand inventory in cabinets in tcmIS. The Cabinet Inventory page can be accessed from the Supply Chain menu. Under Cabinet Inventory, select Cabinet Inventory. The Cabinet Inventory page is launched in a new tab.

Figure 1

Select the hub which services the cabinets from the Hub drop-down box. Select the facility and/or work area where the cabinet is located from the corresponding drop-down boxes. The Cabinet box will be populated with all active cabinets for the selected facility and work area. You may select multiple cabinets by holding CNTRL and clicking multiple cabinets in the Cabinet box.

Check the box labeled Show Only Quantity > 0 to see only cabinet items that currently have stock.

Click Search. All active items in the selected cabinets are shown. Details of each Receipt ID in the cabinet are displayed including manufacturer lot number, expiration date, last count date, last count quantity, quantity delivered since the last count, total quantity, unit cost, and stocking levels.

Figure 2

You may display all cabinet inventory of a customer part number by selecting Part Number from the drop-down box and entering the part number. Click Search.

Figure 3

You may also search for cabinet inventory by item description. Select Item Desc from the drop-down box and enter a description of the product. Click Search.

Figure 4

You may display all cabinet inventory of an Item ID by selecting Item ID from the drop-down box and entering the item number. Click Search.

Figure 5

You can use the Cabinet Inventory page to locate material in the cabinets which are expiring soon. Enter the number of days in the Expires In and To boxes as shown below in Figure 6. Click Search.

Figure 6

Results can be sorted by Cabinet ID, Part Number, or Item ID by selecting the desired sort option from the Sort By drops-down list.

You may also sort by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order.

Figure 7

Clicking on the link in the Item column displays detail on inventory of the item at the warehouse that serves the cabinets.

Figure 8

The same inventory details can be viewed by right-clicking on any line and selecting Show Inventory Detail.

Figure 9

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 10

Click Create Excel File on the Cabinet Inventory page to export the results into Excel.
