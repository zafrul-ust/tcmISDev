Waste Transfer Management

Click on the button named Pickup to get to the Waste transfer Management Screen as shown below.
This will take you to the screen shown below.
  

This screen displays to current status of all waste container transfer requests. The transfer request date is shown in the second column. Once the transfer is complete, and the waste is in the 909-day storage area, then the box marked Transferred is checked by the 90-day area manager (access to this box will only be granted to the 90-day operator or their designee). This indicates that waste has already been moved from the point of generation to the 90-day generator storage area. Upon Update Now, the container will no longer appear.

In many cases the waste container is created in the 90-day area by consolidating waste into a drum or other container in the storage area. In this case, the Waste Manager can check the Transferred box on the Waste Management Request screen while generating the request.

You can search for waste containers that were picked up from the point of generation within a specified time period by checking the box by Picked up within ______ Days and indicating in the window the time period you wish to view. 

If a request was not submitted - the requestor did not click the Submit button on the Waste Request Pickup screen - then it will be held as a draft request. No date appears in the request date column for draft requests. The requestor can go back and complete the draft request, and submit it, by highlighting that line then clicking the Waste Request Detail button. The same button can be used to view details on any of the waste requests.

 The Waste Manager is allowed to modify the pickup date by double-clicking on that column and modifying the date.
