List Old Jackets

+ Larger Font | + Smaller Font

The List Old Jackets page is used to reprint copies of the receiving checklist, also known as a receiving jacket, that have already been printed. The receiving checklist is a tool that may be used by your hub as part of the Haas TCM ISO quality process. The receiving checklist is used to record the receipt of materials ordered through tcmIS. The receiving checklist is uniquely identified by a purchase order number and a line item number. The receiving checklist is typically printed from the Receiving page, but the List Old Jackets page can be used to look up and print receiving checklists when the purchase order is no longer visible on the Receiving page. The List Old Jackets page can be accessed by selecting Stocking Room Operations from the Supply Chain menu. Select Receiving then List Old Jackets. The List Old Jackets page is launched in a new tab.

Figure 1

Check the have box the list receiving checklists which have printed before.

Select the hub from the drop-down box labeled for.

You may want to limit the results by selecting date ranges for the purchase order date and/or the promised delivery date.

If you know the purchase order number of the receiving checklist to be printed, enter the po number in the text box labeled where the tcmIS PO is. Click Submit Query.

Figure 2

If you know the Item ID of the material for which the receiving checklist to be printed, enter the Item ID in the text box labeled and the tcmIS Item ID is. Click Submit Query.

Figure 3

You may also limit your search by supplier by entering a supplier name in the text box labeled and the Supplier contains. Click Submit Query.

Results can be sorted by Item ID, po number, or supplier in either ascending or descending order by selecting the sort option from the Sort by drop-down list and selecting either Ascending or Descending. Click Submit Query to sort the results.

Figure 4

Click the link in the column labeled PO Number (Part table @ end) to display the receiving checklist formatted with the list of active parts for the Item ID at the end of the receiving checklist. Right-click on the receiving checklist and select Print to print the checklist. Right-click on the receiving checklist and select Back to return to the List Old Jackets page.

Figure 5

Click the link in the column labeled PO Number (Part table @ top) to display the receiving checklist formatted with the list of active parts for the Item ID at the beginning of the receiving checklist. Right-click on the receiving checklist and select Print to print the checklist. Right-click on the receiving checklist and select Back to return to the List Old Jackets page.

Figure 6

You may mark a receiving checklist as printed or not printed by selecting either Y or N in the column labeled Marked printed. Click Set Print Status to save changes.
