Min Max Levels

+ Larger Font | + Smaller Font

The Min Max Levels page is used to set and change the stocking type, reorder point, and maximum stocking levels for Haas TCM items. The Min Max Levels page can be accessed from the Supply Chain menu by selecting Stock Room Inventory and the choosing Inventory. Then select Min Max Levels. The Min Max Levels page is displayed in a new tab.

Figure 1

Select the hub for which Min Max levels are to be changed from the Hub drop-down list. Select Item ID or Part Number from the drop down box and enter an Item ID or Part Number in the corresponding text box. If desired, you may also select an Inventory Group. Click Search.

Figure 2

Details regarding previous changes to stocking types or levels can be displayed by right-clicking on any line and selecting Show History.

Figure 3

Details regarding previous changes to stocking types and levels for the selected Part Number and Inventory Group are displayed.

Figure 4

To change the stocking type for a Part Number, select the new stocking type from the Stocking Method drop-down list on the Min Max Levels page.

Selecting OOR indicates that the part is not to be stocked at the hub, but should be ordered only when requested by the customer. For OORs, the Reorder Point and Stocking Level should be zero.

Selecting MM indicates that the part should be stocked at the hub. The Stocking Level represents the maximum quantity that will be kept in stock at any time. The Reorder Point is the point at which a Buy Order is generated to replenish the stock back to the Stocking Level. For MMs, the Reorder Point and Stocking Level should be non-zero. The Reorder Point should be less than the Stocking Level.

Selecting exitMM indicates the part was stocked at the hub, but should be converted to an OOR when the remaining stock is depleted. Changing a MM to an OOR will automatically change the status to exitMM until all stock on hand or on order has been depleted. After all inventory is depleted, the stocking type will automatically be changed to OOR.

Selecting virtualMM indicates that the part is stocked at a Virtual Hub or supplier stocking location. VirtualMM materials are treated as OORs. This stocking level indicates to customers that material is not available in the warehouse, but is stocked at another location.

To change the Reorder Point, Stocking Level, Reorder Quantity or Look Ahead Days, enter a new number in the corresponding text box and enter an explanation of the change in the Remarks text box.

You may designate a material as a direct delivery or drops ship material by checking the box in the column labeled Drop Ship. Designating a material as Drop Ship results in direct delivery of all purchases of the material for the selected Inventory Group. The material will not be shipped to the Hub and all quality and labeling procedures will be foregone.

For all lines updated, check the box in the column labeled OK. Click the Update link above the data grid to save changes for all lines selected.

Figure 5

For Counting Inventory Groups, you may update the Receipt Processing Method and Billing Method as well.

The Receipt Processing Method controls when material in invoiced to the customer. Select Customer Owned if the material should be invoiced on receipt and stored in inventory as owned by the customer. Select Haas TCM owned if the material should be invoiced when it is counted missing and stored in inventory as owned by Haas. Select Issue On Receipt if the material should be invoiced on receipt and should not be stored in inventory.

The Billing Method indicates whether a material should be invoiced to the customer or not. If the material is a contract or CPU item and should not be invoiced to the customer, select Not Billed. If the material is a Pass Through item and should be invoiced to the customer, select Pass Through.

Select the proper values from the Receipt Processing Method and Billing Method drop-down lists.

For all lines updated, check the box in the column labeled OK. Click the Update link above the data grid to save changes for all lines selected.
