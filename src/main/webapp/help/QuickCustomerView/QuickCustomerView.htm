Quick Customer View

+ Larger Font | + Smaller Font

The Quick Customer View page is a convenient way to get a complete overview of Distribution items in HGI locations, including HGI owned material, client owned material, and consigned material. The Quick Customer View can be accessed in two ways:

  • From the Distribution menu by selecting the Quick Customer View menu item. It will be opened in a new tab.
  • From the context menu on any record row shown on a Scratch Pad or a Material request select the Quick Customer View menu item. It will be displayed in a child window.

Figure 1

In the upper panel of the Quick Customer View tab page there are there are three criteria and two options for specifying the item for which you want details. (On the child window form since you have started from a specific customer and item all these choices have already been accounted for: the upper panel displays only the Inventory Group and Item Desc[ription] information.)

  • Select the operating entity from the Operating Entity drop-down list.
  • Select the HGI hub from the Hub drop-down list.
  • Select the inventory group from the Inventory Group drop-down list.
  • You must specify a particular Customer:
    • begin typing the name and tcmIS will display a drop-down list of matching customers (see Type ahead search); or
    • click on the magnifying glass icon to use the Customer Lookup form. See Lookup forms for general information and see below for more details.
  • You may use the within ___ days text box to limit the search to customers withing a recent period.

Click the Search button to display all the items that match the criteria.

Figure 2

Unlike most pages in tcmIS, the Quick Customer View page displays four data grids. Click on any of these data grid titles to see more information:

  • MR History (upper left)
  • Invoice (upper right)
  • Price Quote History (lower left)
  • Contact (lower right)

Figure 3

Customer Lookup form

The Customer Lookup form includes five different sets of search fields; each be used alone or in combination with one or more of the others. When combinations are used search results will display only those items that meet all criteria.

The Customer ID search allows only exact matching, that is where the entered search value Is exactly the same as the field value. The remaining four of the five set of searches allows matching on Is, Contains, Starts with, and Ends with. (See tcmIS Search/Seach by for more details.) The fields are

  • Customer ID
  • Customer Name
  • Synonym
  • Bill To
  • Ship To

Figure 4

MR History data grid

In the MR History panel use any of the text boxes in the column headers to filter the display and show only those inventory items matching the text you enter. The match will be made when any or all of what you enter matches the data in that column, including when you enter text in more than one column .

  • Inventory Group
  • MR-Line
  • Customer Part
  • Desc[ription]
  • Customer PO

Figure 5

Right click on any record in the MR History panel for a context menu that includes four menu items.

  • The Open menu item will display the MR for the record in a new tab. See the Create MR page help for more details.
  • The MR Line Allocation menu item opens a child window to display the MR Allocation report for only the selected record. See help for the MR Allocation Report for details.
  • The Create New Quote menu item display a new Quote page in its own tab, with much of the information already filled in based on the existing quote. See the Quote page for details.
  • The Create New MR menu item will display a new MR in its own tab, based on the existing MR. See the Create MR page help for more details.

Figure 6

Price Quote History data grid

In the Price Quote History panel use the text box in the Inventory Group column to filter the display to display only those inventory items matching the text you enter.

Right click on any record in the Price Quote History panel for a context menu that includes three menu items.

  • The Open menu item will display the Quote for the record in a new tab. See the Quote page help for more details.
  • The Create New Quote menu item display a new Quote page in its own tab, with much of the information already filled in based on the existing quote. See the Quote page for details.
  • The Create New MR menu item will display a new MR in its own tab, based on the existing MR. See the Create MR page help for more details.

Figure 7

Invoice data grid

In the Invoice panel use the text box in the Customer PO column to filter the display to display only those PO’s matching the text you enter.

Right click on any record in the Invoice panel for a context menu that includes one menu item.

  • The Print Invoice menu item will send a copy of the selected invoice you your printer.

Figure 8

Contact data grid

In the Contact panel there is a New contact link. Click on it to display the New Customer Contact form in a child window. To record a new customer contact"

  • Enter as much information as you have in the various text boxes (First Name, Last Name, etc.): text boxes marked with red asterisks indicate require information;
    • Notice that there is a Default Contact check box at the bottom of the second column of text boxes. If the new record you are creating is to be the default contact, select this box.
  • From the Primary Job Function drop-down list select the closest match: note that this is also required information;
  • Using the check boxes, select any (and all) Other Job Functions that apply;

Click the OK button to store the new contact information in tcmIS. Click the Cancel button to discard the information and close the form.

Figure 9

  • The Customer Details menu item will display the Edit Customer Customer Details page in a new tab.
  • Email menu item will use your e-mail software open an e-mail message addressed to the selected contact person.
