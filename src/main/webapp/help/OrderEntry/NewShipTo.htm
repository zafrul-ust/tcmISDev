New Ship To Address

+ Larger Font | + Smaller Font

The New Ship To Address process is used to add new customer Ship To Addresses to tcmIS for Distribution Hubs. You may begin the New Ship To process from the Scratch Pad by clicking the labeled Ship To.

All current customer Ship To Addresses for the customer selected on the Scratch Pad are displayed. If a new customer Ship To Address needs to be added, click the New Ship To Address link at the top left.

Figure 1

The New Ship To Address page is displayed. All fields marked with an asterisk are required fields and must be completed.

Enter a name for the Ship To Address in the Ship to Name text box.

Enter the Ship To Address in the Address text boxes. These are free form fields and you may enter the address in any format as long as the City and Postal Code are included.

Select the operating entity and default inventory group from the corresponding drop-down boxes.

If applicable, select the state, province, or region from the drop-down box labeled State/Region.

Enter the City and Postal Code in the corresponding text boxes.

Select the country from the Country drop-down box.

Figure 2

The Field Sales Rep and Sales Agent are not required fields. To enter either field, click the corresponding . Enter a name in the Search box and click Search. Select the desired Sales Representative by clicking anywhere on the line and clicking the Return Selected User link above the data grid. The selected Sales Representative is populated on the New Ship To Address page.

Figure 3

You may also select a price list for this Ship To Address by selecting the appropriate price list from the Price List drop-down list.

After all required fields have been completed, add the new customer Ship To Address by clicking the OK button. The new Ship To Address is populated on the Scratch Pad, Quote, or MR header.

To update existing customer Ship To Addresses, you should access the Customer Search page from the Distribution menu. For more information on viewing and updating existing customer information, see the Customer Search help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerSearch.htm.
