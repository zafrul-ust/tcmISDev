The Order Entry page is the starting point for creating Quotes and Material Requests (Sales Orders) in tcmIS for Distribution Hubs. The Order Entry page can be accessed from the Distribution menu by selecting Order Entry.

Figure 1

The Scratch Pad is the starting point for all Quotes or Material Requests. Think of the Scratch Pad as an electronic piece of paper on your desk which you use to record key information regarding potential quotes or material requests.

When the Order Entry page is launched your existing Scratch Pads are displayed.

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 2

To create a new Scratch Pad click the New link in the top left above the data grid. A new tab is launched for each Scratch Pad opened.

Figure 3

Click the icon in the bottom right of the header to display the entire search header. Click the icon in the bottom right of the header to collapse the search header.

Figure 4

The first step in using the Scratch Pad is to select a customer. You may begin the Customer Lookup process from the Scratch Pad by clicking the labeled Customer. For more information on the Customer Lookup process, see the Customer Lookup help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerLookup.htm.

For cash sales, select the appropriate Staff Cash Sales or House Cash Sales customer. For more information on the Cash Sales process, see the Cash Sales help page: https://www.tcmis.com/tcmIS/help/OrderEntry/CashSales.htm.

After completing the Customer Lookup process, the Customer, Ship To Name, Credit Terms, Credit Limit, and Ship To address is populated on the Scratch Pad. Various other customer details are also displayed including the customer's Ship Complete preference, whether or not to Charge Freight, Default Shelf Life, Currency, Default Inventory Group, and Field Sales Representative.

Next, select the customer contact person by clicking the labeled Contact. For more information on the Customer Contact Lookup process, see the Customer Contact Lookup help page: https://www.tcmis.com/tcmIS/help/CustomerSearch/CustomerContactLookup.htm

After completing the Customer Contact Lookup process, the Contact, Telephone, and Email is populated on the Scratch Pad. Click the Clear button to erase the selected customer contact information.

Figure 5

The Ship To Address is populated based on the Ship To Address selected during the Customer Lookup process. However, you may select a different Ship To Address by clicking the labeled Ship To.

All Ship To Addresses for the customer are displayed. Select the desired Ship To Address by clicking anywhere on the line and then clicking the Return Selected Address link. The selected Ship To Address will be populated on the Scratch Pad.

If a new customer Ship To Address needs to be added, click the New Ship To Address link at the top left. For more information on adding new customer Ship To Locations, see the New Ship To Address help page: https://www.tcmis.com/tcmIS/help/OrderEntry/NewShipTo.htm.

Figure 6

You may click the buttons labeled One Time to enter a Ship To Address and use it on the Scratch Pad without permanently adding it to tcmIS. This should be used for one time shipments only.

Enter the Ship To Address in the Full Address text boxes. These are free form fields and you may enter the address in any format as long as the City and Postal Code are included. Select the operating entity and default inventory group from the corresponding drop-down boxes. If applicable, select the state, province, or region from the drop-down box labeled State/Region. Enter the City and Postal Code in the corresponding text boxes. Select the country from the Country drop-down box.

After all required fields have been completed, click the OK button to add the Address to the Scratch Pad.

Figure 7

The customer reference number, for example purchase order number or request for quote number, may be entered in the Customer Ref text box but is not required for a Scratch Pad.

If there are any permanent Bill To Notes or Permanent Ship To Notes, you may view them by clicking the corresponding links in the Notes area.

Figure 8

At any time you may save your Scratch Pad by clicking the Save link at the top left. Click the Delete link at the top left of the Scratch Pad to permanently delete the Scratch Pad. Click New to create a new Scratch Pad. Click Refresh to reopen the Scratch Pad.

The next step in creating a Quote or Material Request is to add material lines by clicking the Add Line link below the header. For more information on adding lines to the Scratch Pad, see the Add Line help page: https://www.tcmis.com/tcmIS/help/OrderEntry/AddLine.htm.

Figure 9

Customer's specification requirements can be specified when adding a line to a Scratch Pad. From a Scratch Pad, click the in the Specification column to lookup specifications and add them as requirements for the selected material line. For more information on adding specification requirements see the Specifications help page: https://www.tcmis.com/tcmIS/help/OrderEntry/Specifications.htm.

Figure 10

Additional charges like freight and packing fees can be added to the Scratch Pad header or to individual material lines. Click the button labeled Hdr Charges in the header to add additional charges to the Scratch Pad header. These charges apply to the entire order as opposed to additional line charges which apply only to specific lines on the Scratch Pad. Header additional charges can be used to create miscellaneous credits and debits witch are not associated with material lines.

Figure 11

You may add additional charges to individual material lines by right-clicking anywhere on the line and selecting Additional Charges. These charges apply to the selected material line only as opposed to header additional charges which apply the entire Scratch Pad. For more information on adding additional charges to a Scratch Pad, see the Additional Charges help page: https://www.tcmis.com/tcmIS/help/OrderEntry/AdditionalCharges.htm.

Figure 12

You may delete a line on the Scratch Pad. Highlight the line to be deleted by clicking anywhere on the line. Click the Delete Line link above the data grid.

Figure 13

The highlighted line is deleted from the Scratch Pad.

Figure 14

You may duplicate a line on the Scratch Pad. Highlight the line to be duplicated by clicking anywhere on the line. Click the Dup Line link above the data grid.

Figure 15

The highlighted line is duplicated on the Scratch Pad.

Figure 16

You may view and/or allocate inventory for the inventory group associated with the material line by right-clicking on the line and selecting Show Allocatable for InvGrp. To view and/or allocate inventory regionally or globally, right click on the line and select Show Allocatable for Region or Show Allocatable for Global.

Figure 17

Note that inventory allocated to a Scratch Pad will be automatically unallocated after 10 hours if a Confirmed Material Request is not created.

For more details on allocating inventory to a Scratch Pad, see the Allocation help page: https://www.tcmis.com/tcmIS/help/OrderEntry/Allocation.htm.

Right-click on any line and select Show Previous Orders For Customer to view all pervious orders for the selected Catalog Item and customer.

Right-click on any line and select Show All Previous Orders to view all pervious orders for the selected Catalog Item for any customer.

Right-click on any line and select Show PO History to view all Purchase Orders for the selected Catalog Item.

Right-click on any line and select Show Quotation History for Customer to view all historical Quotes for the selected Catalog Item and customer.

Right-click on any line and select Show All Quotation History to view all historical Quotes for the selected Catalog Item.

Right-click on any line and select Show Price List Info to view customer price list details and price breaks for the selected Catalog Item.

Right-click on any line and select Show Current Sourcing Information to view supplier price list details for the selected Catalog Item.

Right-click on any line and select Lead Time Plots to view actual lead time details for the selected Inventory Group and Catalog Item.

Right-click on any line and select Item Notes to view global notes for the selected Item ID.

Right-click on any line and select Catalog Item Notes to view global notes for the selected Item ID and Spec Group.

After adding material lines to the Scratch Pad, it is possible to create Quotes or Material Requests from the Scratch Pad.

To create a Quote, select the material lines on the Scratch Pad to include on the quote by checking the box in the Grab column. Click the Create Quote link at the top left of the Scratch Pad. A new Quote is created in a new tab and the Scratch Pad remains active. Click the Create Quote-Delete Scratch Pad link at the top left of the Scratch Pad to create a Quote in a new tab and delete the Scratch Pad. For more information on creating Quotes from the Scratch Pad, see the Create Quote help page: https://www.tcmis.com/tcmIS/help/OrderEntry/CreateQuote.htm.

Figure 18

To create a Material Request, select the material lines on the Scratch Pad to include on the quote by checking the box in the Grab column. Click the Grab box in the line header to select all lines on the Scratch Pad. Click the Create MR link at the top left of the Scratch Pad. A new Material Request is created in a new tab and the Scratch Pad remains active. Click the Create MR-Delete Scratch Pad link at the top left of the Scratch Pad to create a Material Request in a new tab and delete the Scratch Pad. For more information on creating Material Requests from the Scratch Pad, see the Create Material Request help page: https://www.tcmis.com/tcmIS/help/OrderEntry/CreateMR.htm.
