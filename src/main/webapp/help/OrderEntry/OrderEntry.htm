Order Entry

+ Larger Font | + Smaller Font

The Order Entry page is the starting point for creating Quotes and Material Requests (Sales Orders) in tcmIS for Distribution Hubs. The Order Entry page can be accessed from the Distribution menu by selecting Order Entry.

Figure 1

The Scratch Pad is the starting point for all Quotes or Material Requests. Think of the Scratch Pad as an electronic piece of paper on your desk which you use to record key information regarding potential quotes or material requests.

When the Order Entry page is launched, all of your existing Scratch Pads are displayed.

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 2

To create a new Scratch Pad click the New link in the top left above the date grid. For more information on using the Scratch Pad see the Scratch Pad help page: https://www.tcmis.com/tcmIS/help/OrderEntry/ScratchPad.htm.

Figure 3

To open an existing Scratch Pad check the box in the OK column for all Scratch Pads you would like to open. Click the Open link in the top left above the date grid.

Figure 4

A new tab is launched for each Scratch Pad opened. For more information on completing the Scratch Pad see the Scratch Pad help page: https://www.tcmis.com/tcmIS/help/OrderEntry/ScratchPad.htm.

Figure 5

You may also right-click on any line on the Order Entry page and select Open to open a Scratch Pad.

Figure 6

You may delete Scratch Pads by checking the OK box for each line to delete. Click the Delete link at the top left of the Order Entry page above the data grid. It is recommended that you delete all unneeded Scratch Pads occasionally. You are limited to 999 open Scratch Pads. As with real papers on your desk, your Order Entry screen can become cluttered with old unneeded Scratch Pads and should be periodically cleaned up.

Figure 7

Click the Lookup link at the top left of the Order Entry page to search for and open existing Material Requests, Quotes, or Scratch Pads. For more information on looking up Material Requests, Quotes, or Scratch Pads. see the Order Entry Lookup help page: https://www.tcmis.com/tcmIS/help/OrderEntry/OrderEntryLookup.htm.

Figure 8

Click the Refresh link on the Order Entry page to refresh your list of Scratch Pads.
