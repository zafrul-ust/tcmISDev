Add Line

+ Larger Font | + Smaller Font

The Add Line process is used to add material lines to a Scratch Pad, Quote, or Material Request. At least one line must be added to a Scratch Pad in order to create a Quote or Material Request from the Scratch Pad.

Click the Add Line link below the header to begin adding material lines to a Scratch Pad, Quote, or Material Request.

Figure 1

Select the inventory group the material should be shipped from from the Inventory Group drops-down box.

You may lookup the product to add to the Scratch Pad, Quote, or Material Request using a variety of different search criteria. A common search is by Part Number or Catalog Item. If the exact Haas Catalog Item is known, select is from the Part Number drop-down list and enter the Part Number in the corresponding text box. If the exact Catalog Item is not known, select contains, starts with, or ends with from the Part Number drop-down list and enter a portion of the Part Number in the corresponding text box. Click the Search button.

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the column.

Figure 2

You may also use the Part Description to locate products to add to a Scratch Pad, Quote, or Material Request. Select contains from the Part Description drop-down list and enter a description of the material in the corresponding text box. Click the Search button.

Figure 3

You may also use the Specification to locate products to add to a Scratch Pad, Quote, or Material Request. Select contains from the Specification drop-down list and enter a portion of the specification name in the corresponding text box. Click the Search button.

Figure 4

You may also use the Customer Part Number field to search for products in a similar fashion if a customer part number has been entered.

Figure 5

Use the Text field to search text notes entered for a Catalog Item.

The Synonym field can be used to search by your legacy part number. Select contains from the Synonym drop-down list and enter a portion of the legacy part number in the corresponding text box. Click the Search button.

Figure 6

By default a standard manufacturer's C of C is added to all material lines, Click the Clear button if this certification is not required. Click the in the Specification List column to view and select specification requirements for the Catalog Item. Check the box to the left of the specification to add. Check the box in the CoC column if a Certificate of Conformance is required. Check the box in the CoA column if a Certificate of Analysis is required.

You may select as many specifications and certifications as required by the customer. Click the Return Selected link at the top left of the Specification Look Up page to return to the Add Line page.

Figure 7

The specifications needed are populated in the Specification List column. Click the Clear button to delete the specification requirements.

Figure 8

For more information on adding specification requirements see the Specifications help page: https://www.tcmis.com/tcmIS/help/OrderEntry/Specifications.htm.

Right-click on any line and select Show Allocatable for InvGrp to view and allocate available inventory for the selected Catalog Item and default customer inventory group. To view and/or allocate inventory regionally or globally, right click on the line and select Show Allocatable for Region or Show Allocatable for Global.

For more details on allocating inventory to a Scratch Pad or Material Request see the Allocation help page: https://www.tcmis.com/tcmIS/help/OrderEntry/Allocation.htm.

Once you have located the product to add to a Scratch Pad, Quote select the line to add to the Scratch Pad, Quote, or Material Request by clicking anywhere on the line and clicking the Add Catalog Item link.

The selected Part Number is added to the Scratch Pad, Quote, or Material Request. You may continue to add as many lines as necessary by repeating the process.

Figure 18

You may remove a line from a Scratch Pad, Quote, or Material Request by clicking anywhere on the line to delete and clicking the Delete Line link.

You may duplicate a line on a Scratch Pad, Quote, or Material Request by clicking anywhere on the line to duplicate and clicking the Dup Line link.
