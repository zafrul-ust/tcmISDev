Customer's specification requirements should be specified when adding a line to a Scratch Pad, Quote or Material Request. By default a standard manufacturer's C of C is added to all material lines, Click the Clear button if this certification is not required.

From Add Line page, click the in the Specification column to lookup specifications and add them as requirements for the selected material line.

Figure 1

From a Scratch Pad, Quote, or Material Request, click the in the Certification column to lookup specifications and add them as requirements for the selected material line.

Figure 2

Check the box to the left of the specification to add. Check the box in the CoC column if a Certificate of Conformance is required. Check the box in the CoA column if a Certificate of Analysis is required.

You may select as many specifications and certifications as required by the customer. Click the Return Selected link at the top left of the Specification Look Up page to return to the Scratch Pad, Quote, MR, or Add Line page.

Figure 3

The specifications and requirements selected are now attached to the material line. Click the Clear button to remove all specs from the line.

Figure 4

If the specification needed by the customer does not appear in the list click the Add Specs link.

Figure 5

Enter the Spec name in the box labeled Spec and click Search. If the specification is located, enter any Type, Grade, or Class information in the Detail box. Click anywhere on the line and click the Return link. If the desired specification in not found, click the New Specs button.

Figure 6

Select the spec date by clicking in the box labeled Spec Date and select the date from the calendar. Enter the name of the specification in the Spec text box. This is the only required field.

Enter the revision, amendment, and any additional details like type, grade, or class if needed. Also enter the name of the organization or company that produced the specification in the Spec Org box.

You may upload a copy of the specification by clicking the Browse button and locating the file on your PC.

Click the Return Spec link to add the specification. Click the Close link to exit without adding the specification.

Figure 7

The new specification is now displayed on the Specification Look Up page. You may select it and add certification requirements as described above.
