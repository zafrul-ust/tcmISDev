Cabinet Labels

+ Larger Font | + Smaller Font

The Cabinet Label page is used to print labels for Cabinet Management in tcmIS. Labels available from the Cabinet Labels page include Cabinet Baselines, Cabinet Labels, and Cabinet Bin Labels. These labels should be printed and applied in the proper cabinet locations prior to the first cabinet scan. The Cabinet Labels page can be accessed from the Supply Chain menu. Under Cabinet Inventory , select Cabinet Labels. The Cabinet Labels page is launched in a new tab.

Figure 1

Select the hub which services the cabinets from the Hub drop-down box. Select the facility and work area where the cabinet is located from the corresponding drop-down boxes. Click Search.

Figure 2

Check the box in the column labeled Print for all cabinets which labels are to be printed. Click the Generate Baseline button the print the cabinet baseline. The cabinet baseline should be displayed on every cabinet.

Figure 3

A sample cabinet baseline is shown in Figure 4.

Figure 4

Click the Generate Cabinet Labels button the print the cabinet label which is scanned at the beginning of each cabinet scan. Click the Generate Cabinet & Bin Labels button to print the cabinet label plus all bin labels.

Click the Generate All Labels button to print the cabinet baseline, cabinet label, and all bin labels.

To print only specific bin labels, click the link in the Cabinet ID column. Check the box in the column labeled Print for each bin label to be printed. Select the labels size and click Generate Bin Labels.
