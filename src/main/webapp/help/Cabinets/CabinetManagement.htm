Cabinet Management

+ Larger Font | + Smaller Font

The Cabinet Management page is used to activate and inactivate parts and bins in cabinets, set cabinet stocking levels, and add new bins and parts to cabinets in tcmIS. The Cabinet Management page can be accessed from the Supply Chain menu. Under Cabinet Inventory, select Cabinet Management. The Cabinet Management page is launched in a new tab.

Figure 1

Select the hub which services the cabinets from the Hub drop-down box. Select the facility and/or work area where the cabinet is located from the corresponding drop-down boxes. If a work area is selected, the Cabinet box will be populated with all active cabinets in that work area. You may select multiple cabinets by holding CNTRL and clicking multiple cabinets in the Cabinet box.

Click Search. All active items in the selected cabinets are displayed.

Figure 2

You may display all cabinet bins containing a customer part number by selecting Part Number from the drop-down box and entering the part number. Click Search.

Figure 3

You may display all cabinet bins containing an Item ID by selecting Item ID from the drop-down box and entering the item number. Click Search.

Figure 4

Select the box labeled Show Inactive bins to search for cabinet bins that have been inactivated. Check the box in the Active column to activate the bin. Uncheck the box in the Active column to inactivate the bin. Click the Update link above the data grid to save any changes.

Figure 5

Click the link in the Item column to edit the reorder point, stocking level, and lead time for the cabinet bin. Enter the minimum quantity to maintain in the cabinet in the Reorder Point text box. Enter the maximum quantity to maintain in the cabinet in the Stocking Level text box. If known, enter the number of days which it takes to procure and ship the material in the Lead Time In Days text box. Enter any comments in the Remarks box. Click the Update link above the data grid to save any changes.

Figure 6

To add a new bin to a cabinet, from the Item Management Page, click Add New Bin. Select the work area and cabinet from the corresponding drop-down boxes, If necessary, select a catalog from the Catalog drop-down box. Enter a name for the new bin in the Bin Name text box. Enter an active customer part number in the Part Number text box. Enter the minimum quantity to maintain in the cabinet in the Reorder Point text box. Enter the maximum quantity to maintain in the cabinet in the Stocking Level text box. If known, enter the number of days which it takes to procure and ship the material in the Lead Time In Days text box. Click Add to add the bin and part to the cabinet. Use the Cabinet Labels page to print anew Cabinet Baseline and Bin Labels.

Figure 7

Data from the Cabinet Management page can be exported to Excel by clicking the Create Excel File button.
