Cabinet Count

+ Larger Font | + Smaller Font

If manual entry of cabinet inventory is necessary, use the Cabinet Count page. The Cabinet Count page can be accessed from the Supply Chain menu. Under Cabinet Inventory, select Cabinet Count. The Cabinet Count page is launched in a new tab.

Figure 1

Select the hub which services the cabinets from the Hub drop-down box. Select the facility and/or work area where the cabinet is located from the corresponding drop-down boxes. If a work area is selected, click Get Cabinet to display all active cabinets for the selected work area in the Cabinets box. You may select multiple cabinets by holding CNTRL and clicking multiple cabinets in the Cabinet box.

Click Search. All active bins and items in the selected cabinets are displayed.

Figure 2

Enter the quantity in the cabinet for each item and click the Update button. Material Requests for cabinet replenishment will be created, however, the Cabinet Inventorypage will not be updated until the next cabinet scan upload to tcmIS.

Data on the Cabinet Count screen can be exported to Excel by clicking the Create XLS button. This file can be saved and printed to assist in manually taking cabinet inventory.
