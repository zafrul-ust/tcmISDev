Work Area Definition

The Work Area Definition page is used to view, manage, and print information about work areas set up in tcmIS. The Work Area Definition page can be accessed from the Supply Chain menu by selecting the Work Area Inventory menu item, then the Work Area Definition item. The Work Area Definition page is launched in a new tab.

Figure 1

You must choose a specific facility for the work area(s) from the Facility list. You may choose to narrow the range of work areas by selecting from the Physical Layout portion of the selection panel and/or from the Organizational Layout section.

  • Physical Layout: refers to the physical location of the Work Area.
    • You may select a specific Area from the Area list.
    • You may also further narrow the selection by selecting a specific building from the Building list.
  • Organizational Layout: refers to the work area position in the company hierarchy.
    • You may select a specific work area from the Work Area Group list.
    • You may select a specific department from the Dept. list.

Click the Search button. The work areas will be displayed in the data grid. (Because the grid includes so many columns it is illustrated in left and right sections, below.)

Click the Create Excel button to export results in Excel format.

Action links

In addition to the Update link, there are two other links in the upper-left of the data grid:

  • The New Work Area link may be clicked to add a new work area line to the data grid. You must edit this line (as described in the sections below) to complete the information needed, and click the Update link to save the new record.
  • The Show Legend link will cause a small window to open, on top of the current page, to display a color key to any shading used to characterize the records in the data grid.

Context menu links

Right click on any column of a record for a context menu.

  • Select the Edit Work Area Directed Charge link to view or edit these details. See Edit Charge on the Catalog page for details.
    • If directed charges have been entered for both a part and the work area, the part directed charge will be used rather than the work area directed charge.
  • Select the Change History to display a child window listing the records of changes made for the part.
Figure 4

The Work Area Groups page simply lists all the existing work area groups to which any particular work area may be assigned. There are three things that can be done on this page.

  • Edit one or more of the existing work area group names. The work area name may be edited directly in the text box in the Work Area Group column. No changes made on the Manage Work Area Groups page are saved until the Update link on this page is clicked.
  • Create a new work area group by clicking the New Work Area Group link at the upper-left of the page. A blank work area group line will be added at the bottom of the list: you may have to scroll down the list to see it. Enter a work area name directly into the text box in the Work Area Group column. No changes made on the Manage Work Area Groups page are saved until the Update link on this page is clicked.
  • Assign a work area group to the work area you selected on the Work Area Definition page when you clicked the magnifying glass icon . A new link, Return Work Area Group, will appear at the top of the page. Click this link. The Manage Work Area Groups page will close and the selected group will appear in the Group column. (Since you will be back on the Work Area Definition page, remember, no changes made are saved until the Update link is clicked.)
