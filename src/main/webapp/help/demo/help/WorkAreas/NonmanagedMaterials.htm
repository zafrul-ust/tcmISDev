Nonmanaged Materials

From the Work Area Stock Mgmt page it is possible to add non-managed materials (often referred to as Tracker items). The process begins by clicking on the Add Nonmanaged Mat[erial]l link at the top left of the data grid.

Figure 1

tcmIS will open a child window to display the Please choose a Work Area form. You must specify for which Work Area the material is to be added.

The available work areas are displayed in a drop-down list: the areas included are based on the Search criteria used to create the data grid just displayed on the Work Area Stock Mgmt page.

Once the Work Area is chosen from the list, click the Select button to proceed.

Figure 2

tcmIS will change the child window to display the Add Nonmanaged Material search form. This is used to identify the company-approved material that, while not supplied through HGI, is to be inventoried for a location and tracked in tcmIS.

Select the Catalog from which you want to select the material. (In many instances there is only one Catalog available.)

You may narrow the range of the search with the three Search by boxes, using one of the listed fields: Material ID, Material Description (the default), Manufacturer, or MSDS Number.

Click the Search button to see the usage of consigned material including PO number, quantity issued, and unit price. Provide this report to the supplier listed for accurate invoicing to HGI.

Click the Create Excel button to export the results of the report in Excel format.

Figure 3

tcmIS will run the search and display the records in a data grid. The first of these records is selected by default but you may locate and select the exact item you want. Click on the row to select an item and it will be incorporated into the Return Selected Material link at the top of the data grid.

Next, click on the Return Selected Material link. tcmIS will open an additional child window to display another Add Nonmanaged Material form where you must supply final details.

Figure 4

In this second Add Nonmanaged Material form you must supply additional details. All the elements are required.

  • The Work Area is supplied based on your earlier choice.
  • Provide the MSDS Number
  • Specify the expected Average Amount of the material to be stored in the work area.
  • Indicate the Max[imum] Amount allowed in the work area for this material.
  • Specify the Largest Container Size allowed in the work area for this material, as measured in the Unit specified below.
  • For the Unit choose the appropriate values from the list.
  • For the Tier II Storage Method choose the appropriate values from the list.
  • For the Tier II Storage Temperature choose the appropriate values from the list.

When you have supplied the information for the form, click the Add button. The child window will close and the material will be added to the end of the Work Area Stock Mgmt data grid.

Figure 5

You may repeat the steps above until all needed materials have been added to the data grid.

The Add Nonmanaged Material process is not complete, however, until you click on the Update link at the top left of the data grid. When you click the Update link, the materials are incorporated into the permanent record.
