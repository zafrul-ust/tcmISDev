Charge Numbers Set Up

The Charge Numbers Set Up page is used to record (and manage) valid numbers or number combinations for Direct or Indirect charges on requests. A single charge number or number combination may be assigned to more than one approver for different work areas or work area groups, etc., as needed for the organization’s accounting purposes. See Charge Number Management for details on specifying the approver(s) for each valid charge number.

The Charge Numbers Set Up page can be accessed from the Supply Chain menu.

Figure 1

Select the facility for the charge number(s) from the Facility list.

In some cases you may also be able to choose

  • the accounting system and/or
  • the charge type

from the respective boxes. Your choices depend on how the Facility has defined charge number set up requirements. In many cases there is a single “choice”.

Additionally, you may narrow your search by specifying all or part of the charge number particulars in one or more text boxes on the right side of the search parameter panel. Depending on your own situation these possibilities may include one or more of the specific elements included in your charge number combinations, for example Cost Center or Network.

Click the Search button. The work areas will be displayed in the data grid.

Click the Create Excel link to export the displayed data in Excel format.

Figure 2: Charge Numbers Set Up search parameter panel

tcmIS will display the result of your search in a data grid. Each row in the grid presents one valid combination of charge numbers and each column, except the first, presents one charge number element. It is the combination of all elements that make up a single, valid, charge number.

Notice in the figure how the specification of parts of two elements affected the results displayed.

In addition, you may sort the list of charge numbers by any of the elements, ascending or descending, by clicking in the header of the data grid for the element you want. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

A charge number may be active or inactive. If it is inactive it cannot be used to place orders or add inventory.

On the Charge Numbers Set Up page data grid the first (left-most) column is the Active column. If the check box is selected for the record, the charge number on that row is active. If the box is cleared, the charge number is inactive; the selected row is also highlighted with a dark background.

Figure 4

When you attempt to inactivate a charge number, tcmIS will notify you if the number is in use, that is, the charge number is the default charge number for a work area or a part number. A small message window will be shown. You must click the OK button on the message window to proceed with an inactivation in this situation.

Figure 5

Any charge number may be edited directly in the row and column for the record and number element. No changes made on the Charge Numbers Set Up page are saved until the Update link on this page is clicked.

In order to add a Charge Number to tcmIS you must click the New link on the upper-left of the data grid. tcmIS will add a new, blank row to the data grid where you enter the values for all the elements of the charge number combination in their respective columns. (The active check box is selected by default.) Enter the needed information and click the Update link. No addition made on the Charge Numbers Set Up page ise saved until the Update link on this page is clicked.

  • The New and Update links are only shown if you have permission to make additions or changes.

Note, also, that a Delete link is available to eliminate the newly created charge number record if you realize it is incorrect or unneeded.
