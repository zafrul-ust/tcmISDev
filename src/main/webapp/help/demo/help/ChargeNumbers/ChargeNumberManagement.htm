Charge Number Management

The Charge Number Management page is used to identify (and manage) authorized approvers for use of individual Direct or Indirect charge numbers on requests. The authorized approver may be different for different work areas or work area groups, etc., as needed for the organization’s accounting purposes. tcmIS automatically routes Material Requests to Approver(s) for the needed authorization. See Charge Numbers Set Up for details on recording valid charge numbers and number combinations.

The Charge Number Management page can be accessed from the Administration menu. The Charge Number Management page is launched in a new tab.

Figure 1

Select the facility for the charge number(s) from the Facility list.

In some cases you may also be able to choose

  • the accounting system and/or
  • the charge type

from the respective boxes. Your choices depend on how the Facility has defined charge number set up requirements. In many cases there is a single “choice”.

Additionally, you may narrow your search by specifying all or part of the charge number particulars in one or more text boxes on the right side of the search parameter panel. Depending on your own situation these possibilities may include one or more of the specific elements included in your charge number combinations, for example Cost Center or Network.

Click the Search button. The work areas will be displayed in the data grid. Sort the data, ascending or descending, by clicking in the header of the data grid for the item you want. A small upward- or downward-pointing triangle will appear in the header to indicate the sort.

Click the Create Excel link to export the displayed data in Excel format.

Figure 2: Charge Number Management search parameter panel

On the Charge Number Management page data grid the center column is the Active column. If the check box is selected for the line, the WBS Element-Approver combination on that line is active. If the box is cleared, the combination is inactive. If it is inactive it will not be used to route orders.

Figure 3

In order to add a WBS Element-Approver combination to tcmIS you must click the Add link on the upper-left of the data grid. tcmIS will add a new, blank row to the data grid where you enter the values for the WBS Element and the Approver Name in their respective columns. (The active check box is selected by default.)

  • Both the WBS Element text box and the Approver Name text box are Type Ahead Searches to allow easy location of valid Elements and Names.

Any added (or changed) record on the Charge Number Management page is not saved until the Update link is clicked.

Click the Create Excel button on the Catalog Add Tracking page to export results in Excel format.

Figure 4

In order to edit or delete a Charge Number-Approver combination in tcmIS you must right-click the record you want to edit or delete. tcmIS will display a context menu with two menu items: Update Approvers and Remove Line.

  • Select the Update Approvers menu item to edit the existing combination. tcmIS will redisplay the record and make it possible to change the Approver. Just as in Adding a record, the Approver text box also serves as a Type Ahead Searches to locate a valid Approver Name.
  • Select the Remove Line menu item and tcmIS will immediately remove the WBS Element-Approver Name combination permanently.
