MSDS Analysis Report

The MSDS Analysis Report is used to locate and display specific Material Safety Data Sheets (MSDS) in order to analyze sets of materials. The MSDS Analysis report is accessed by selecting the MSDS Analysis item on the Report menu.

Figure 1

The MSDS Analysis report page consists primarily of ways to specify the criteria for the needed report.

  • Select a Facility or accept the default All from the Facility drop-down list.
  • Select a Work Area or accept the default All from the Work Area list.
  • You may narrow the range of the search with the three Search by boxes, using one of the listed fields:
    • Item ID
    • Material ID
    • Material Description
    • MSDS [number]
    • Part Number
    • Specification
    • Synonym
    • Trade Name
      • As with any MSDS search you may use the search words ’AND’, ’OR’, or ’BUT NOT’ to combine multiple search terms. See MSDS Viewer help for details and examples.
  • There are three check boxes to further characterize the material for the search.
    • The Approved check box, which is selected by default, will exclude unapproved materials from the report unless it is unselected.
    • The Stocked check box will restrict the search to items kept in inventory for the work area if it is selected.
    • The Full Database check box will extend the search to the entire database if it is selected.
    • The Kit MSDS will restrict the report to materials which are a part of some Kit MSDS if it is selected.
  • The Manufacturer text box is used to select a specific manufacturer for the report data. Typing in the box will automatically begin a search to match the entered text. When the correct manufacturer appears, move on to the next element to specify.
  • The Physical State drop-down list allows specification of that by selecting from the list or accepting the default All.

Physical characteristics

  • Six more physical characteristics can be requested explicitly using these sets of search boxes:
    • pH
      • The first box allows a choice for pH of > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second pH box.
    • Flashpoint
      • The first box allows a choice for Flashpoint of < (less than), > (greater than), >= (greater than or equal), or <= (less than or equal) as compared to a value you specify in the second Flashpoint box.
      • Use the Flashpoint drop-down list to indicate the F (Fahrenheit) or C (Centigrade) scale.
    • Vapor Pressure
      • The first box allows a choice for vapor pressure of > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Vapor Pressure box.
      • Use the Vapor Pressure scale drop-down list to indicate the units of measure.
    • VOC
      • The first box allows a choice for the desired percentage or quantity of Volatile Organic Compounds (VOC) of > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second VOC box. Specify the percentage number between 0 and 100, or a quantity number. Choose the units of measure from the drop-down list.
    • VOC LWES
      • The first box allows a choice for the desired percentage or quantity of Volatile Organic Compounds (VOC) Less Water and Exempts of > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second VOC LWES box. Specify the percentage number between 0 and 100, or a quantity number. Choose the units of measure from the drop-down list.
    • % Solids
      • The first box allows a choice for the desired percentage of solids of > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second % Solids box. Specify the percentage number between 0 and 100.

NFPA parameters

  • The NFPA section allows the specification of any of the four NFPA parameters to be included on the report.
    • Health
      • The first box allows a choice for the desired Health rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Health box.
    • Flammability
      • The first box allows a choice for the desired Flammability rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Flammability box.
    • Reactivity
      • The first box allows a choice for the desired Reactivity rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Reactivity box.
    • The Hazard drop-down list allows a choice for the desired Hazard description.

HMIS parameters

  • The HMIS section allows the specification of any of the four HMIS parameters to be included on the report.
    • Health
      • The first box allows a choice for the desired Health rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Health box.
    • Flammability
      • The first box allows a choice for the desired Flammability rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Flammability box.
    • Reactivity
      • The first box allows a choice for the desired Reactivity rating of = (equals), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to a value you specify in the second Reactivity box.
    • The Personal Protections drop-down list allows a choice for the desired Personal Protections Code (and description).
  • The final section of the report specification page may take two forms depending on whether the List option is selected or the CAS Number is selected. (See below.)

When all desired criteria are specified, begin the Search.

List selection

The selection of the List option (which is the default) makes it possible to specify values for a particular list as a parameter for the report. For example, one could report on materials where the Arsenic Compounds had a minimum value greater than 10%.

  • Select a list by clicking on it. To select more than one, hold down the CTRL key while selecting.
  • Using the first of the three boxes on the right of the list display, choose whether the comparison value is a Min[imum], Max[imum], or Average value.
    • If more than one list is selected any comparison value will apply identically to all lists.
  • Using the second of the three boxes, choose whether the min/max/average is = (equal), > (greater than), >= (greater than or equal), <= (less than or equal), or < (less than) as compared to the value you specify in the third box. Specify the percentage number between 0 and 100.

When any desired List items are specified, begin the Search.

Figure 3

CAS Number selection

The selection of the CAS Number option makes it possible to specify the proportion of one or several materials, as defined by CAS Numbers, as a parameter for the report. For example, one could report on materials where CAS #763-69-9 (Ethyl 3-Ethoxypropionate) made up at least 9% and CAS #123-86-4 (n-Butyl Acetate) made up no more than 15%.

  • Note: If there is more than one CAS Number in the criteria, choose at the head of the list whether to
    • select ’and’ to restrict material on the report to items which meet the criteria for ALL OF the indicated CAS Numbers or
    • select ’or’ to restrict material on the report to items which meet the criteria for ANY OF the indicated CAS Numbers or
  • Enter the first material CAS number in the left-hand CAS Number 1 box.
    • Click on the small magnifying glass icon to use a Lookup form to find the CAS Number.
    • In the second box choose whether the comparison value is a Min[imum], Max[imum], or Average value.
    • In the third box, choose the comparison operator >= (greater than or equal), > (greater than), = (equal), <= (less than or equal), < (less than), or NULL for the min/max/average value.
    • In the fourth box specify the percentage number (between 0 and 100) for the comparison.
  • Repeat the process for each additional CAS Number needed
  • If more lines are needed to add CAS Numbers, click on the Add Another CAS Number link to insert them at the end of the list. Scroll down the list if necessary.

When any desired CAS Number items are specified, begin the Search.

Begin the search

When all desired criteria are specified, click the Search button to begin the search. When the report is composed, tcmIS will redisplay the page and display the report in the lower panel.

Click on the Create Excel button to export the report in Excel format.

If at any time you want to start over on the report criteria, click the Clear All Fields button to wipe out all the values or entries made anywhere on the criteria form.

On the MSDS Analysis report in the leftmost column, following the Material ID, there will be a small page/paper icon . Clicking on that icon will open a child window to display the View MSDS page. See MSDS Viewer for further details.

Similarly, right click on any record in any column and there will be a context menu displayed. The first item on this context menu is View MSDS which will also open a child window to display the View MSDS page for the selected record.

Figure 6

The second item on the context menu is Show Storage Locations. Click on that item and a child window will be opened to display the Show Storage Locations form. If you have run the report for a specific Work Area, only storage locations for that Work Area are shown. If you are looking at a report for all work areas in a Facility, multiple storage locations may be shown.
