Lookup/Run Reports

The Lookup/Run Reports page is used as a centralized location to find and run saved reports in tcmIS. The Lookup/Run Reports page can be accessed from the Reports menu by selecting the Lookup/Run Reports menu item. The Lookup/Run Reports page is launched in a new tab.

Figure 1

The Lookup/Run Reports page begins with a search form that allows you to identify the report type or types that you want, using a set of check boxes, as well a text box to enter part or all of the name of any report you want. There are seven types of reports and you may select as few as one or as many as seven for your search. The report types are:

  • Material (M)
  • Usage (U)
  • Inventory (I)
  • Cost (C)
  • Waste (W)
  • Orders (O)
  • Admin[istration] (A)

When you have specified characteristics of the report(s) you want, click the Search button. tcmIS will list the saved reports that match your criteria. The report names begin with the letter of the type (shown in parenthesis in the list above) and a number, for example, ’C123 - Weekly Cost Report’.

Figure 2

tcmIS will produce a data grid listing all the reports matching the criteria you have specified.

Click the Create Excel button to export the results of the report in Excel format.

Figure 3

Within the data grid you may right-click on any row to get a context menu that allows you to act upon the report. The choices on the context menu depend on your 'relationship' to the report.

  • If you are the creator of the report your context menu will include Open Report as well as Delete Report.
  • If you are not the creator of the report your context menu will only include Open Report.
