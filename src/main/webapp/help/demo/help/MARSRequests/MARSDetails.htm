MARS Details

The MARS Details page is part of the Receiving and Receiving QC processes as well as the recertification process. It is rather long and includes four sections:

  • General Information in the top panel.
  • Sample Information in the second panel.
  • Chain of Custody information in the third panel.
  • Test Results in the bottom panel

The MARS Details page is accessed from the MARS Requests page. The MARS Details page opens in its own tab.

Figure 1

Except for the General Information section each panel may be collapsed or expanded to make it easier to view and/or work on only the section of immediate interest. Notice the upward- or downward-pointing chevrons ( or ) on the right side of the page, one corresponding to each of the three lower panels.

  • If the section panel is collapsed the indicates that it can be expanded downward.
  • If the section panel is expanded the indicates that it can be collapsed.
Figure 2

As you can see by reading through the panels, each contains specific detail information about various aspects of the MARS Request.

On the Sample Information panel, in addition to information about the sample used for the testing, there is a View/Add Receipt Documents link. Click that link and tcmIS will open a child window to display a list of any existing receipt documents associated with this particular test.

Figure 3

The Receipt Document window displays a list of any existing documents stored in tcmIS for this MARS Request. Click on the (underlined) name of the document to display it in a child window. Select the check box in the first column of a record and click the Update link to remove the document from tcmIS.

Click the Add New Document button to open another child window to display the Add Receipt Document form. On that form, provide a Doc Name and select the type of document to be stored from the Doc Type list. Use the Browse button to locate the document and upload it to tcmIS. Click the Save button to complete the process or the Cancel button to abandon the addition.
