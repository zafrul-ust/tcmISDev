List Management

The List Management Report page is used to create and manage list of chemicals. The List Management Report page can be accessed from the Administration menu by selecting the List Management menu item. The List Management Report page is launched in a new tab.

Figure 1

To manage an existing list, locate the list name using the Search box. Click on the Search button to cause tcmIS to display a data grid of the matching lists. The data grid includes the list name and description as well as up to three thresholds (for list inclusion) and their parameters.

Click the Create Excel button on the page to export the data in Excel format.

Figure 2

To create a new list there are two options.

  • Use the Upload List button on the upper panel of the page. tcmIS will open an Upload List form.
    • Use the Upload List form to locate and upload an Excel file containing the list information to import. Click the Process button once the file is completely uploaded to incorporate that list information into tcmIS.
    • Click the Example button to download an Excel file which displays the needed format for the information to be imported properly.
Figure 3

The second option is to insert a new List directly into the data grid. Creating a new list consists of two steps: creating the list and populating the list.

  • Click the New button on the List Management page, to the right of the List box. tcmIS will add a new row to the bottom of the data grid where you may enter all the available information.
    • Provide a Name and Description in those text input boxes (required information).
    • Enter other information using the text boxes and selection lists in the remaining columns.
  • Click the Update link to save the list or close the List Management tab to disregard what you've started.
  • The list is only a name until you actually add specific chemicals to it. See the next section for details.
Figure 4

After a list is created, by importing or creating it directly, to add a chemical to a list, right click on the row and select the Manage List menu item from the context menu. tcmIS will open a child window to display a List Management form.

  • The context menu Manage List menu item is shown on most or all records on the List Management tab. Except for those lists you have created, selecting this link simply displays the list of chemicals on the list. Editing is not allowed.

On the context menu, select the Delete Line menu item to discard the new list if you conclude you do not want to keep it in tcmIS.

Figure 5

Until at least one item is added to the list no records can be displayed. The only possible actions are to click the Add link or to close the child window.

Click the Add link to add an item to the list. tcmIS will open another child window to display the CAS Search form which may be used to locate the specific item to be added to the List.

Figure 6

The CAS Search is a standard three-box text search. You may search on CAS Number, EC Number, or Preferred Number. Locate the item you want to add to the list and select the record. The Return Selected CAS Number: [number] link can be clicked to add the item to the list.

tcmIS will close the child window.

Figures 7 and 7A

Continue using the Add link until all the needed items are added to the List. When you have added the last needed item, click the Update link at the upper-left of the data grid.

tcmIS will redisplay the child window along with a superimposed caution that the actual data base update will be made over night. Close that notice window.

To remove a chemical from a list, notice that tcmIS has added a check box in the Delete column for every row. Select that check box and again click the Update link to remove any items that should not be on the list.

Close the child window to be returned to the List Management tab. The List is now ‘populated’.
