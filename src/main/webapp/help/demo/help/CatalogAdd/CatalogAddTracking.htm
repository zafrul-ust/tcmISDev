Catalog Add Tracking

The Catalog Add Tracking page enables users to view the status of a Catalog Add request. This page also allows approvers to locate Catalog Addition requests needing their approval.

The Catalog Add Tracking page can be accessed from the Order Management menu by selecting the Catalog Add menu item, then the Catalog Add Tracking menu item. The Catalog Add Tracking page is opened in a new tab.

Figure 1

The Catalog Add Tracking page allows you to tailor your search for requests for additions to the catalog and view the status of those requests.

  • The Requester box can be used to limit the search to requests by a specific individual. There is no default Requester.
    • Leave the Requester box blank to see requests regardless of who placed the request.
    • Enter the name of the person who placed the request in the Requester box to see only those requests. Start typing and tcmIS will display up to ten names that match your entry. (See Type ahead searches.)
    • By default the Active Only check box is selected to limit the type-ahead search to active accounts; to include deactivated accounts, unselect that check box.
  • Similarly, the Approver box can be used to limit the search to those request that have to be approved by specific individual. There is no default Approver.
    • Leave the Approver box blank to see requests regardless of who is supposed to approve.
    • Enter the name of the person to approve the request in the Approver box to see only those requests. Start typing and tcmIS will display up to ten names that match your entry. (See Type ahead searches.)
    • By default the Active Only check box is selected to limit the type-ahead search to active accounts; to include deactivated accounts, unselect that check box.
  • You may search within all facilities for which you have authority by selecting My Facilities from the Facility list or select one specific facility;
  • likewise, the work area can be all work areas for which you have authority by selecting My Work Areas from the Work Area list or select one specific work area.
  • You may provide something to at least partially identify the material you want using the Search by boxes. See Search functions for more information.
    • The available Search by fields are Part (LM part number), Part Desc[ription], [HGI] Item ID, or [Catalog Add] Request ID.
  • Checking the Show all request needing my Approval box limits the search to only those requests needing your approval (see below).

There is a set of check boxes to limit the search to only those requests in a certain status. Select one or more of the Status check boxes at the right to use these.

  • Draft (New Material): requests for addition of new material still in draft status.
  • Draft (New part No): requests for addition of a new part number still in draft status.
  • Pending QC: submitted requests awaiting completion by the HGI Catalog Team (gathering required data [including the MSDS] and validation of the material with the manufacturer).
  • Pending Pricing: submitted requests awaiting complete baseline pricing information.
  • Pending Resubmit: a resubmission of a rejected request has been drafted but not completed and resubmitted.
  • Ready to Order: fully approved requests; materials are now available for ordering.
  • Draft (New Size/Pkg): requests for addition of an existing material in a new size or new packaging still in draft status.
  • Draft (New Approval): requests for a new work area approval of an existing material still in draft status.
  • Pending Approval: submitted requests awaiting approval through your company's internal processes.
  • Pending Catalog: submitted requests awaiting final verification by the HGI Catalog Team of quality requirements (specifications) after all other approvals are complete.
  • Resubmitted: a rejected request for which a new submission has been made.
  • Rejected: disapproved requests.

Search for requests by clicking the Search button.

Click the Create Excel button on the Catalog Add Tracking page to export results in Excel format.

Figure 2

The status of a request can be read in the Status column. In addition, depending on the status of the request, a context menu (right click on the line) will display additional choices.

  • Select the View/Approve Eval menu item to begin the Engineering Evaluation approval process.
  • Select the View/Approval Cat Add menu item to begin the Catalog Add approval process.
  • Further information about the status in the approval process can be viewed by selecting the Approval Detail menu item.
  • Select the Resubmit menu item to request a reconsideration of the Cat Add.
Figure 3

Approval Detail

When you select the Approval Detail menu item from the context menu, the Approval Detail page will be opened in a new window on top of the Catalog Add Tracking page.

The Approval Detail page displays the material details along with the status and date of every approval associated with the request.

Figure 4

Approvers

Approvers should check Show all requests needing my approval before a search to display all Catalog Add requests awaiting their approval.

For each record in the display grid, the Cat Add Request page can be viewed by right-clicking anywhere on the line and selecting the View/Approve Cat Add menu item (see Figure 3_. For additional details, see Help on the Cat Add Request page.

Resubmit a Cat Add Request

To initiate a request to reconsider a Cat Add when it has been rejected, right-click anywhere on the line and selecting the Resubmit menu item. tcmIS will open a new child window and display the Cat Add request as a new request. For additional details, see Help on the Cat Add Request page.
