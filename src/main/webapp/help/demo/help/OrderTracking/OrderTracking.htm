Order Tracking

The Order Tracking page provides information about and status of MR’s (Material Requests = Orders) in tcmIS. It is accessed from the Order Management menu. Select the Order Tracking menu item. The Order Tracking page is launched in a new tab.

  • For viewing and approving MR’s—see help for the MR page.
  • For seeing details of individual MR-Lines—see help for the MR-Line Allocation form.
Figure 1

The Order Tracking page provides numerous selection criteria to support locating the desired MR’s. Some criteria must be specified but many are optional.

  • If you have a choice you must specify the company for your search from the Company drop-down list;
  • Choose a specific Facility from the drop-down list or keep the default My Facilities choice.
  • Choose a Work Area from the drop-down list or leave it set at the default My Work Groups.
  • You may want to or need to provide something to at least partially identify the order you want using the Search by boxes. (Your search may be specified by User PO [number], PO[number]-Line [number], Part #, Part Desc[ription], Material Request No, or Item ID). See Search functions for more information on search methods.
  • The Requestor box can be used to limit the search to requests by a specific individual.
    • The default Requestor is the logged in user: you.
    • Clear the Requestor box to see requests regardless of who placed the order.
    • Enter the name of the person who placed the request in the Requester box to see only those requests. Start typing and tcmIS will display up to ten names that match your entry. (See Type ahead searches.)
    • By default the Active Only check box is selected to limit the type-ahead search to active accounts; to include deactivated accounts, clear that check box.
  • Checking the Orders needing my Approval box limits the search to only those requests needing the current user's approval.
  • Checking the Critical Only box limits the search to only those requests that are currently marked critical.
  • A search must be specified by the state of the request in the order process by selecting one of the four option buttons.
    • To find requests with any status, select the Any option.
      • You will be required to provide some Search by detail (described above) with this option.
    • To find requests that were saved, but not yet submitted, on the Material Request page, select the Draft option.
    • To find requests that are pending financial approval or pending cancellation, select the Pending option.
    • To find requests that have been submitted but not yet delivered, select the Open option.
  • A search may be specified based on some past activity
    • To find those requests that were released to HGI in the past select the Released within __ days option and enter the number of days in the past in the text box.
    • To find those requests that were delivered in the past select the Fully Delivered within __ days option and enter the number of days in the past in the text box.
    • To find those requests that were canceled or rejected in the past select the Canceled within __ days option and enter the number of days in the past in the text box.

Click the Search button to display the records. Click the Create Excel button to export the displayed data in Excel format. Click on the Show Legend link (at the top of the data grid) to view a key to the background coloring of records.

The initial display of the Order Tracking page usually displays only about half of the detail columns. Use the slider bar at the bottom of data grid to display columns on the right side of the data grid.

To view or work on an MR, right click the record line and select the context menu item. tcmIS will display the needed page or form. The context menu items available depend on the Status and Type of the order.

  • Select MR Allocation to view the MR Allocation report for the entire order, including the selected record, or select MR Line Allocation to view the MR Allocation report for only the selected record. See help for the MR Allocation Report for details.
  • If the line has a Type of SCH select the MR Line Schedule item to display the Delivery Schedule page (for the line) in a child window. There you may view the delivery schedule or edit future scheduled deliveries. See the Delivery Schedule page help for details.
  • Select View/Approve MR to open the Material Request. See the help for the MR page.
  • Select Approval Detail to cause tcmIS to display a form which lists those responsible for approvals, superimposed on the Order Tracking page. (See figure below.)
