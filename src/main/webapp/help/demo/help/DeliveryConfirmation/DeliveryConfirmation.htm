Delivery Confirmation

The Delivery Confirmation page is used to manually confirm deliveries after they have been received, and to view any comments that have been entered on received deliveries. The Delivery Confirmation page can be accessed from the Order Management menu by selecting Delivery Confirmation. The Delivery Confirmation page is launched in a new tab.

Figure 1

The Facility list allows you to specify the facility for which you want to enter or see delivery information.

The Shipment ID text box allows you to specify the exact shipment you want to see.

  • The Shipment ID text box uses the ’is’ search (exact match) and will not return a record if there is only a partial match.

The Ship Date: From and To input boxes allow you to specify a starting and/or ending date for the deliveries you want to see reported. Clicking into either input box will cause a small calendar to be displayed from which you may choose the date you want.

You may include records of shipments already marked as Delivered by selecting the check box for Show Confirmed.

Click the Search button.

Click the Create Excel button on the Catalog Add Tracking page to export results in Excel format.

Figure 2

To acknowledge receipt of a delivery, locate the Shipment ID and select the check box in the OK column for that record. You may also click into the Comment column to add remarks. tcmIS will open a multi-line text box to allow you to type whatever you need to include.

When all the receipt information is provided, click on the Confirm link in the upper-left corner of the window to complete the process. You may wait to click the Confirm link until you have entered receipt information for multiple shipments if you wish.

Before you ever click the Confirm link, you may abandon the process and disregard the transfer steps by simply closing the page without ever clicking Confirm.

Figure 3

When you choose to include Show Confirmed records, a Confirmed By column is added to the data grid (even if no Confirmed records are actually found). As you would expect, the ID of the person who added the confirmation is shown on that column for the confirmed record.
