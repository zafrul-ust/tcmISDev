Consigned Inventory Report

The Consigned Inventory Report page is used to produce a usage report of material that is supplier owned or consigned. The Consigned Inventory Report page can be accessed by selecting Inventory from the Reports menu and the selecting Consigned Inventory Report. The Consigned Inventory Report page is launched in a new tab.

Figure 1

Select the Facility from the list and the inventory group from the second list.

Use the Expires Within: and Expires After: text boxes to identify a date range of expirations for the consignment inventory on which you wish to report.

Click the Search button to see the usage of consigned material including PO number, quantity issued, and unit price. Provide this report to the supplier listed for accurate invoicing to HGI.

Figure 2

Click the Create Excel button to export the results of the report in Excel format.
