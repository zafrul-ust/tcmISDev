Work Area Inventory

The tcmIS Work Area Inventory page is used to view information regarding on hand inventory in cabinets. It can be accessed from either of two menu selections.

  • The Work Area Inventory page can be accessed from the Supply Chain menu. Under Work Area Inventory select the Work Area Inventory menu item.
  • The Work Area Inventory page can be accessed from the Reports menu. Under Inventory select the Work Area Inventory menu item.

Using either menu selection, the Work Area Inventory page is launched in a new tab.

Figures 1 and 2

Select the facility where the work areas are located from the Facility list.

  • You may keep the default All in the Work Area Group list or choose to narrow the range of work areas by selecting a specific work area group.
    • The Work Area box will be populated with all active work areas for the facility or the work areas for the chosen work area group.
  • Select the specific work area(s) from the Work Area list. You may keep the default All, or select a single work area, or select multiple work areas by holding down the CTRL key while clicking each wanted work area.
  • You may narrow the range of the search with the three Search by boxes, using one of the listed fields:
    • [Customer] Part Number
    • [HGI] Item Description
    • [HGI] Item ID

You can use the Work Area Inventory page to locate materials in the work areas which are expiring soon. Enter the number of days in the Expires In and To boxes. You may enter a minimum number of days before expiration, or a maximum, or both.

Select the check box labeled Show Only Quantity > 0 to see only work area items that currently have stock.

Click the Search button. All active items in the selected work areas will be shown. Details of each Receipt ID in the work area are displayed including manufacturer lot number, expiration date, last count date, last count quantity, quantity delivered since the last count, total quantity, unit cost, and stocking levels.

Figure 3

By default, the search results (inventory items) are sorted and displayed by Work Area. You may also sort by Part Number or Item ID. Select the desired sort option from the Sort By list.

You may also sort by any of the columns in the results grid by clicking on the column header. Click the header again to sort in descending order.

Figure 4

Right-click on any line and select the Show Inventory Detail menu item to display details on inventory of the item at the warehouse that serves the work area. The Inventory Details page will be displayed in a new window.

Figure 5

The Inventory Details page shows details on the inventory at the warehouse including quantity, status, manufacturer lot number, and expiration date.

In many cases when you hover the mouse pointer on data cells a tool tip will pop up and display the contents of the cell. This is particularly useful when the data in the cell does not fit the default size of the cell: it allows you to see the entire contents of the cell without having to adjust the width of the column.
