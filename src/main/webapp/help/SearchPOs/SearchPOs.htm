Purchase History

+ Larger Font | + Smaller Font

The Purchase History page is used to lookup and view Purchase Orders in tcmIS. This page can be accessed from the Supply Chain Menu. Under Purchasing, select Search POs. The Purchase History page is launched in a new tab.

Figure 1

To locate the desired Purchase Order, first select the Hub and/or Inventory group, if known. If not known, select All. Select the buyer's name, if known. If the buyer is not known, select Any Buyer.

Note that only open POs are displayed by default. Uncheck the Open POs Only box to display all POs which meet the search criteria regardless of status. You may also search for unconfirmed POs by checking the Unconfirmed box. You can locate POs that are not properly associated with Buy Orders by checking the box labeled Broken PR Link.

Click Search to display all Purchase Orders which meet the search criteria.

Figure 2

You may also search by Item Description, Item Id, and Supplier. Select the desired search criteria from the Search drop-down box. Enter the corresponding data in the the text box labeled for.

Figure 3

It is possible to view the entire purchase history of a material or supplier by setting the search criteria on the left to All and enter an Item ID, Item Description, or supplier.

Figure 4

To export results into Excel, click Create Excel.

The data on the Purchase History page can be sorted by certain columns by clicking the column name. The results will be sorted by the selected column in ascending order. Click the column name again to sort in descending order.

Figure 5

Click the icon in the top right of the Purchase History page to collapse the search header. Click the icon to display the search header.

You may resize any column by clicking and dragging the column dividers.

Figure 6

If you hover above some data cells, a tool tip will pop up and display the contents of the data cell. Typically tool tips will appear on columns where the data in the cell contains over 30 characters and the default column width does not display the entire contents of the cell. Use this feature to display the contents of cells without widening the collumn.

Figure 7

Click on the link in the PO column to view the Purchase Order.
