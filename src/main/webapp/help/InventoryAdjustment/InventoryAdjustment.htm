Inventory Adjustments

+ Larger Font | + Smaller Font

The Inventory Adjustments page is used to view and approve requests for adjustments to inventory in tcmIS. This includes both write off and write on requests. This page can be accessed from the Supply Chain menu by selecting Stock Room Inventory and the choosing Reconciliation. Select Inventory Adjustments to launch the page in a new tab.

Figure 1

To view all Inventory Adjustment Requests across all Operating Entities, Hubs, and Inventory Groups for which you have permission, select My Entities, My Hubs, and My Inventory Groups from the corresponding drop-down lists. Click the Search button.

Details regarding the Inventory Adjustment Requests are displayed. The details include the source Operating Entity and Inventory Group, the Item ID and description. The type of inventory adjustment is displayed as well as the quantity and Receipt ID requested to be adjusted and cost. The requestor, request date, and any comments from the requestor are displayed as well.

Click the Create Excel button to export the data displayed into Microsoft Excel format.

Figure 2

To limit the results, you may select the Operating Entity from the corresponding drop-down list. You may also select the Hub and Inventory Group from the corresponding drop-down lists. You may select My Hubs or My Inventory Groups to see all hubs and/or inventory groups for which you have permission. Click the Search button.

Figure 3

If you are an approver you may add comments by clicking in the Approve Comment column.

To approve a Request check the OK box and click the Approve link.

Figure 4

To reject a Request check the OK box and click the Reject link.
