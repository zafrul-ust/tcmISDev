The tcmIS Java App page is used to launch the tcmIS Java Application for any customer. The tcmIS Java Application allows Haas TCM CMS customers to place orders, perform order tracking, inventory tracking, add new chemicals to their catalog, and run environmental reports. The tcmIS Java Application is the program that Haas TCM CMS customers traditionally think of as tcmIS.

A separatetcmIS Home page and Java Application exist for each CMS customer. For example, the UTC tcmIS Home page is located at https://www.tcmis.com/tcmIS/utc/home.do. UTC employees use this page to launch tcmIS and start the Java Application. Much of the functionality previously found only in the Java Application has been transferred to web pages which are accessed from the new Customer tcmIS Application. All features currently available only the Java Application will be migrated to web pages, which will eventually eliminate the need for the Java Application.

The Java Application tab can be launched from the Haas tcmIS Application by clicking Java App in the Main menu. Select the company for which you would like to launch the tcmIS Java Application. Click Launch GUI.
